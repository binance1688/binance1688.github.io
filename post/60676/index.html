上个月 Token 2049 大会在新加坡顺利召开，作为大会期间唯一参展的多链 NFT 交易平台，AlienSwap 通过其卓越的数据表现毫不掩饰地展现了其在 NFT 交易市场的创新力量。

1、交易量飙升

首先是交易量呈现出爆发式增长。据 NFTScan 数据显示，AlienSwap 在大会展期间平均每 24 小时全链交易量 为 910 万美元，超过 OpenSea 和 Blur，排名全网第一。AlienSwap 在加密市场交易领域的影响力和用户基础不断扩大，用户对其交易平台的信任度也在不断提升。

2、用户增长迅猛

用户数量也呈现出迅猛增长的趋势。根据官方数据显示，AlienSwap 在大会期间新增用户数量较平时增长了超过 400% 。注册钱包地址数从 70 w+突破到了 300 w+，AlienSwap 成功吸引了大量新用户的加入，进一步扩大了其用户基础。

3、项目方合作伙伴增多

据悉，AlienSwap 在参会期间与 OKX、DeBox、VIP3 等多个知名项目方达成合作伙伴关系，进一步提升了其在加密货币行业的影响力。各种蓝筹项目方的加入无疑给 NFT 这个细分赛道注入了更多的想象空间和利好预期。

4、社区活跃度提升

AlienSwap 作为第一个基于会员制的 NFT 市场和聚合器，其社区氛围本身就很浓厚。新用户的大量增加对 AlienSwap 的社区活跃度有很大提升，社区成员积极参与讨论、分享经验并参与活动，为 AlienSwap 的发展提供了宝贵的意见和支持。AlienSwap 的社区正在不断壮大，成为一个充满活力和创新思维的社区。

AlienSwap 如何在这么短的时间内产生如此成就？

一、多链星球部署计划

AlienSwap 的多链大航海计划（Space Voyage）于 9 月初刚刚推出，作为其多链星球计划的起航活动，其目的更多的是拓展更多的公链及 Layer 2 链，从而满足更多用户的交易需求。Space Voyage 活动的核心玩法是多链交易限时 0 手续费，对比其他行业头部 NFT 交易市场，OpenSea 的手续费是 2.5% ，Element 是 0.5% - 2% ，X2Y2 和 LooksRare 都是 0.5% ，这对于有很大 NFT 交易需求的用户来讲，尤其是高价值的 NFT 的套利者或者喜欢批量扫地板的用户，一定程度上限制了其交易欲望。限时 0 手续费可以降低用户的交易成本，极大的增强 NFT 行业的信心。

关于多链星球部署计划，现阶段 AlienSwap 已经支持 Ethereum、Linea、Base、Arbitrium 和 Scroll，并且还在持续拓展包括 Solana 和 Ton 等主流公链和 Layer 2 链。这里要特别提一下 Linea，在 9 月份新加坡的 Token 2049 大会期间，AlienSwap 在 Linea 产生的 NFT 每日交易总量达到 500 w 美金，直接霸榜成为 Linea TOP 1 的 NFT 交易市场，并且领先第二名 2 个数量级。目前 AlienSwap 已经是 Linea 链上 NFT 交易量最大的交易市场，并且和 Linea 的官方已经有深度合作，相信接下来会有更多的项目方和链上资产会入驻到 AlienSwap。

二、积分奖励系统

积分奖励系统（AlienScore）也是 AlienSwap 本月初刚推出的功能，与 Space Voyage 同时上线并作为联合空投活动。任何用户在 AlienSwap 上 Mint、Trade、List 或创作 NFT 合集均可以获得一定的积分数量，这些积分未来可以按照相应的比例兑换$ALIEN 空投代币。

一方面，AlienScore 的推出可以增加平台资产的流动性和链上数据交互，因为任何链上的交互均需要一定的 Gas 费成本，积分相当于把这一部分 Gas 费转化为未来的空投预期，从而极大的提高了用户链上交互的驱动力；另一方面，积分奖励系统属于 AlienSwap 基础设施搭建的一部分，一个成熟的 NFT 交易市场需要有完整的交易链路，包括多链的支持、创作者工具、完备的 NFT 合集和适当的活动激励等，积分系统的推出可以作为营销工具帮助平台做更多的活动和推广。

Blur 在去年同期也推出了自己的积分系统，但是不同的是 AlienScore 的推出是联合了多链交互，即参与一次活动，可以获得包括 Ethereum、Linea 和 Base 等多条链上的空投奖励，这种联合多个项目的一鱼多吃的操作可以提高用户的空投预期，更高的空投预期使得更多的用户参与链上交互，从而极大的提高市场信心，这无疑给处于深熊的 NFT 市场注入了新鲜的活力。

三、 0 Gas Mirror Market

我们知道现阶段 NFT 市场面临一个很大的难点就是流动性匮乏，其中最重要的原因之一就是链上交易所产生的高昂的 Gas 费用，尤其是在 NFT 这种高价值的数字资产上。AlienSwap 以其独特的想象力推出了 0 Gas Mirror Market，针对非上链的数字资产可以 0 Gas 费用的铸造和交易。

简单来讲， 通过 0 Gas Mirror Market，用户可以将自己的加密资产或 NFT 资产镜像映射到 AlienSwap Apool 池中，从而无需支付任何 Gas 费用即可交易 NFT。这为用户带来极大便利，能够更轻松、快速、安全地交易 NFT，并将自己 NFT 资产与 AlienSwap 的 0 gas 生态结合。0 Gas Mirror Market 上的创新性功能为用户提供了更低成本、高效率的交易体验，同时推动了去中心化金融和 NFT 的发展。如果用户想提取数字资产到链上，只需要选择 NFT 并提交数据上链，产生一次数据上链的 Gas 费即可。一方面，这样的操作直接去除了 Gas 费的概念，减少了交易过程所产生的阻力并增加流动性，尤其是对那些有很强的交易需求的用户和套利者，直接降低了使用成本；另一方面，没有 Gas 费直接降低了用户使用的门槛，用户无需有链上的数字资产也可以参与到 0 Gas 交易市场里的交互，这无疑撬动了更多有较低Web3行业认知的人进入加密世界。

四、CreateX 创作者工具

创作者作为内容供给侧是 NFT 市场最重要的组成部分，AlienSwap 同样针对创作者推出了 CreateX 创作者无代码开发工具。CreateX 的功能和界面非常简单易用，即使是没有代码经验的艺术家或设计师也能轻松上手。用户只需配置 NFT 合集的名称、内容和一些基本参数就能创建自己的 NFT 合集。除此之外，CreateX 还联合积分奖励系统为创作者提供更多激励。当创作者的 NFT 合集被更多人进行 Mint 和 Trade 时，他们将获得更多的积分。这种激励机制能够鼓励创作者不断创作和推广自己的作品，从而增加他们的收益和影响力。

写在后面

NFT 市场在过去几年经历了快速的发展和繁荣，吸引了大量的投资者和创作者。然而近期几个项目方的暴雷导致整个 NFT 市场出现了十分低迷的情绪。这里我想说的是，市场调整是正常的，任何市场都会经历周期性的涨跌，NFT 市场也不例外。过去几年的繁荣使得市场出现了一些泡沫，价格被过度炒作，导致了一些不合理的估值。深熊行情的出现可以被视为市场的自我调整和洗牌，有助于市场回归理性和健康。

目前主流的 NFT 更多的是 PFP 叙事项目，然而这只是 NFT 的很细分的一个利基市场。NFT 作为未来元宇宙的基础构件，其用例不仅仅只有 PFP 这一种应用场景，我们也不应该因为几个项目方的炒作和暴雷而对整个行业失去信心。路漫漫其修远兮，作为业内人士我们应该对自己所处的领域有更深层的认知和更高的展望，尤其是像Web3这种周期性很强的行业，我们更应该沉下心来耐心且努力的做事，这样才能在接下来新一轮的牛市中获得更丰厚的回报。
