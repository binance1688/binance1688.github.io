1、介绍

在加密货币和去中心化金融（DeFi）不断发展的格局中，一种现象最近引起了经验丰富的加密货币爱好者和新手的注意：Telegram 机器人。这些创新的机器人驻留在流行的消息平台 Telegram 中，引入了一种与加密资产世界交互的新颖方法。在本文中，我们深入研究 Telegram 机器人的有趣世界，研究它们的兴起、功能以及它们对加密货币领域的影响。

2、了解基于意图的交互

要理解 Telegram 机器人的重要性，掌握基于意图的交互的概念至关重要。在这种背景下，意图是通用人工智能（AGI）时代人机交互的核心。基于意图的交互背后的驱动力是先进人工智能和大型语言模型 (LLM)（例如 GPT-3）的集成。这些复杂的人工智能实体将重塑我们使用加密货币和区块链技术的方式。

Anoma 基金会和 Unibot 是两个体现基于意图的交互概念的杰出项目。 Anoma 基金会在第三轮融资中获得了 2500 万美元，该基金会引入了围绕用户意图的以意图为中心的架构。用户表达他们的意图，然后由 Anoma 系统处理。 这个“魔法盒”解释用户意图，执行必要的交易，并更新区块链上的状态。 Anoma 的方法彻底改变了用户与区块链技术交互的方式，强调用户意图作为每项行动的催化剂。

相比之下，Unibot 采取了不同的路线，提供自下而上的加密交互方法。 这种基于 Telegram 的交易机器人允许通过 Telegram 进行自动去中心化交易平台 (DEX) 交易，从而简化了用户体验。它提供狙击购买、跟单交易、DEX 限价单、隐私和反最低可提取价值 (MEV) 功能等功能。Unibot 甚至在 Telegram 机器人生态系统中催生了一系列模拟平台，总市值接近 2 亿美元。

Anoma 和 Unibot 都有一个共同的目标： 简化用户与加密市场的交互。Anoma 侧重于自上而下的方法，而 Unibot 采用自下而上的方法。他们都旨在引入一个新的交互层——用户意图层，允许用户根据自己的意图轻松定义交易状态。

3、Web3 中意图的演变

基于意图的交互的概念并不是全新的。它随着时间的推移不断发展，从 Web 2.0 时代过渡到当前的 Web 3.0 时代。在Web 2.0时代，用户可以将自己的意图输入到Google等搜索引擎中，然后搜索引擎会提供相关的搜索结果。随着亚马逊和 eBay 等电子商务平台的兴起，基于意图的架构进一步发展。用户可以表达购买意向，平台负责物流。

从 Web 2.0 到 Web 3.0 的转变引入了基于意图的网络，在这一概念中，用户只需要表达他们想要的内容，平台就会处理其余的事情。 这种转变代表了以用户为中心的交互的重大飞跃。行业专家预测基于意图的网络将成为网络领域的下一个里程碑。

在 Web3 中，意图呈现出新的维度。LianGuairadigm 将意图定义为一组声明性约束，允许用户将交易创建委托给专门的第三方网络，同时保留完全控制权。Anoma 的底层 Dapp 架构围绕意图展开，将意图解释为表达自定义偏好的消息。它定义了用户定义的系统可以做什么，而不是规定特定的执行路径。

这种意图的概念也催生了以用户意图为中心的架构，称为以意图为中心的架构 。在该架构中，加密协议和基础设施嵌入在用户和加密生态系统之间的交互层中，利用AI/LLM提供更简化的链上交互体验。这种以意图为中心的方法可以显着降低 Web3 时代用户交互的复杂性。

4、Telegram 机器人：让加密货币变得触手可及

Telegram 机器人代表了加密货币领域基于意图的交互的实际表现。 它们是现有 DeFi 应用程序中通常复杂的界面的便捷且用户友好的替代方案。这些机器人使用户只需从移动设备发送消息即可无缝执行各种加密货币操作。使用支持加密的 Telegram 机器人的过程非常简单：在机器人的 Telegram 频道中启动命令，访问主菜单以执行钱包创建或连接等任务，然后将资金发送到指定地址，将资金存入您的钱包。然后，这种交互将导致另一个菜单，为不同类型的交易提供一系列选项。

Telegram 机器人擅长简化用户交互，其功能涵盖广泛的加密活动。用户可以进行交易、狙击（上市后自动购买Token）、复制交易（模仿成功的交易者）和空投农业（增加空投机会的自动化操作）。

Telegram 机器人的一项显着特征是其提供钱包跟踪和分析的能力。此外，许多 Telegram 机器人项目引入了其原生加密货币，导致加密市场中出现了一类新的Token。

5、Telegram 加密机器人的采用和活动

Telegram 机器人在加密社区中迅速获得关注，这主要是由于其用户友好的方法和获得的媒体报道。据 币安研究院称，Telegram 每月拥有超过 8 亿活跃用户，为加密货币扩张提供了巨大的潜在市场。

关键指标反映了人们对 Telegram 加密机器人日益增长的兴趣。Telegram 机器人Token的市值已超过 3 亿美元，各种Token均出现显着增长，尤其是 Unibot，占该细分市场市值的 77% 以上。

Telegram 机器人的用户参与度指标在 7 月份创下历史新高，每日独立用户数量超过 6,000 名。Maestro 是这一领域的领先者，每天约有 2,000 至 3,000 名活跃用户。尽管如此，Unibot 一直在稳步缩小差距。

Telegram 机器人的每日交易量于 2023 年 7 月 23 日创下历史新高，超过 1000 万美元。累计生命周期交易量也超过1.9亿美元。值得注意的是，这些数字可能低估了实际数量，因为并非所有数据都是现成的。

6、与 Telegram 机器人相关的风险

虽然 Telegram 机器人提供了一种与加密货币交互的革命性方式，但它们并非没有风险。用户必须谨慎行事并意识到潜在的陷阱：

1）安全风险：Telegram 机器人要求用户链接他们的钱包，这可能会将他们的私钥暴露给第三方服务。用户必须对这些机器人实施的安全措施充满信心。

2）智能合约风险：Telegram 机器人通常依赖智能合约来执行交易。用户应该彻底审查这些智能合约的代码，以了解风险和潜在的漏洞。

3）诈骗机器人：加密货币领域充斥着诈骗，Telegram 也不例外。用户应保持警惕，避免与承诺不切实际收益的可疑或未经验证的机器人进行交互。

4）监管合规性：根据用户所在位置，使用 Telegram 机器人进行加密活动可能会产生监管影响。用户应了解其管辖范围内的法律要求。

7、结论

总之，Telegram 机器人正在彻底改变我们与加密货币和区块链技术交互的方式。 它们体现了基于意图的交互的概念，简化了复杂的交易，并使加密货币更容易被更广泛的受众所接受。Anoma Foundation 和 Unibot 等项目就是这种方法的例证，每个项目都采取独特的路线来简化用户交互。

意图从 Web 2.0 到 Web 3.0 的演变以及以意图为中心的架构的兴起预示着未来用户可以表达他们的意图，而复杂的 AI/LLM 会处理其余的事情。虽然 Telegram 机器人具有显着的优势，但用户应对安全风险保持警惕并进行尽职调查。

随着 Telegram 机器人的受欢迎程度和功能不断增长，它们为新手和经验丰富的加密货币爱好者提供了一条探索和参与加密货币世界的引人注目的途径。在这个不断发展的加密领域，Telegram 机器人的兴起凸显了我们与区块链技术和数字资产交互方式的持续转变，开创了加密交互智能的新时代。
