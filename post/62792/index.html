Atomicals的概念

Atomicals 协议是一种简单而灵活的协议，一种组织数字对象的创建、传输和更新的方式——它本质上是根据一些简单规则定义的数字所有权链。

核心是铸币、传输和更新操作需要遵循的一些关键的简单规则，

宏伟的愿景是建立并保护比特币区块链作为真理和数字主权的来源

它旨在与其他已出现的协议（例如Nostr、Ordinals等）协调工作。每个协议都有其不同的优势。

五大主要功能和特点

1. 可创建特殊类型的原子数字对象，称为领域

2. ARC20代币，去中心化铸币厂

3. 推出了 GPU 可开采代币，展示了工作量证明和能量消耗，Bitwork挖矿

4. 使用递归和引用创建层次结构并包含原子数字对象

5. 永久文件存储

特色项目—Realm.name（领域）

此段可参考相关博主：

bro.tree@dchatplatypus

当前场内外行情价：2字母地板约280U，3字母地板约15U，3杂（字母+数字）还剩一部分可以打。（主领域不能以数字开头，故没有2D、3D的说法）

Atomicals提出的新兴概念，旨在颠覆传统域名。简单的来说，与普通域名相比较的话，普通域名重点是后缀，比如“.eth”，Realm每个注册的都是前缀的文本，最后拥有所有后缀的定价权。

领域更像是一扇门和钥匙。可以连接 社区 / 组织 / 团体，也成为了整个BTC生态拓展的一扇门，大可被应用到 游戏、社交、土地、勋章 等多个方面。具体而言，在基础功能上，领域可以用作转账收款地址（支付名）；在拓展用法上，有构建社区 / DAO、身份验证（Pass卡 / OG卡 / 参与凭证）、社交档案、特权标识、资产分发 等多种使用场景，也完全契合我们对DID的发展构想。

当前的交易量不大，地板价也相对比较低，因为领域的应用还没有真正铺开，只有把领域的实际功能用起来价值才能得到体现。

其余生态项目

AtomPunks ：稀有 PUNK / Sub1k PUNK 此前成交价已经差不多1000U，普通 PUNK 10-20U。没什么创新。

AtomicalsRekt ：XCSS 是第一个有项目方的原创合集 并且创建了生态首个合集容器。第一批合集和挖矿难度（bitwork commit）挂钩，玩法有趣，目前首批合集只有81个，普通类别成交价150u，稀有类别成交价300u。

AtomMap ：热度不错，机制很有意思，每个 xxxx.atommap 的挖矿难度必须设置为 abxxxx，这就意味着每递增1位数，难度就会成倍上升，目前4D还剩不少能打，用个人电脑，打一个 4D 预估 11 小时。场内外都有成交，参考价 3D 90U、4D 20U。

BlockMap :项目方希望对标Bitmap，热度不足。

对于ARC20的总结

Atomicals 的出现代表着比特铭文的生态大爆发刚刚揭开帷幕,目相比BRC20 铭文赛道来说，这次算是ARC20第一波暴涨，往后还需要经过沉淀才会有更好的未来。

因为ARC20可应用的场景比较多，希望项目方不要因为一波暴涨就躺平停止开发，目前来看也只是处于炒作阶段。其他项目的关注度较低，只有当领域项目Realm真正开始应用，开发新玩法，那么这个项目可以走的更远。所以我们建议可以关注Realm的开发进展，同时Realm和$Realm处于相对价值低洼。

BTC生态：闪电网络概念

闪电网络（Lightning Network）是比特币的Layer2解决方案，在比特币的支付场景中，闪电网络能够帮助用户节省成本，提升效率。一句话就是闪电网络之于比特币，将如同 Visa 之于货币。

2022年，闪电网络领域也出现了多笔大额融资，包括a16z、Paradigm在内的顶级机构也参与到了相关项目的投资中：

2022年4月，专注于比特币业务的公司Lightning Labs完成7000万美元B轮融资，Valor Equity Partners领投。

2022年5月，由Meta加密货币主管David Marcus创立的比特币闪电网络公司Lightspark完成了一轮融资，本轮融资由a16z和Paradigm领投。

2022年9月，基于比特币闪电网络建立的加密支付应用Strike宣布完成8000万美元B轮融资。
