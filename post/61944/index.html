作者：Climber，金色财经

近期，世界经济论坛发布《NFT 演变》报告，其中提出NFT 具有重大的价值创造属性，有潜力创造新颖和创新的方式来与消费者建立联系，并培养更有意义的客户关系。

但NFT 领域长久以来的波动凸显了由炒作和不可持续价值驱动的市场所带来的风险，计划发布 NFT 的组织和机构应该将其整合到现有运营中，并确保在发布之前制定全面的战略。而The Block数据也显示，9月NFT交易量达2.24亿美元，环比下降 37%，连续第七个月交易量萎缩，月交易量已跌至两年低点。

不过，进入10月以来多个主流蓝筹NFT地板价略有回升，NFT交易平台各项数据以及各平台代币如LOOKS、BLUR、X2Y2也有所回暖。此外，9 月 NFT 市场的总交易笔数略有增加，从 483000 笔增至 504000 笔。可见尽管今年NFT赛道已跌到底中底，但加密市场并未完全放弃这一曾经的投资明星。

下面金色财经将全面梳理当前整个NFT市场的生态现状，帮助读者厘清这一板块的发展态势。

一、NFT市场整体表现

首先从多个NFT链上数据分析平台显示的市场情绪来看，投资者的关注度依然处于冰冷状态，近30天内因交易NFT而导致亏损情况的投资者占比超90%。

NFT市场市值和交易额方面，据NFTGo数据显示，当前NFT总市值约为292亿美元，交易总额约为376亿美元。近一年总市值高点出现在今年4月份约为96亿美元，但距今相较于高点下跌41.67%；交易额高点则在今年2月份出现，但距今已下降63.15%。

在持有者与交易者数量上，尽管NFT市场表现不佳，但近一年中NFT的持有者呈45°角上涨态势，其数量增长为40.46%。但受市场行情影响，交易者数量有所下降，跌幅为40.77%。

 

但另据数据显示，自今年6月份因Azuki新系列Elemental发售事件导致NFT市场进一步下跌以来，蓝筹NFT项目地板价在9月份到达历史性低点，不过从10月初开始，蓝筹项目指数有所回升，由4488点上涨至阶段性高位4808点。可见NFT市场略有企稳态势。

而在NFT交易平台市场交易量与市场份额分布方面，据NFTScan数据显示，Blur在今年8月底前一直处于领先地位，但进入9月后即被OKX NFT所超越。不过在市场份额上，OpenSea占据42.72%位列第一，第二LooksRare的30.46%，第三则是Blur。

二、主流NFT交易平台发展态势

今年以来，NFT交易平台Blur表现强势，成功搅局原有NFT市场生态。据上述NFT交易平台市场份额分布来看，目前主流NFT市场分别为OpenSea、LooksRare、Blur、X2Y2以及新势力OKX NFT。

其中已发平台币的NFT市场币价表现亮眼，如LOOKS由9月27日低点的0.046一路上涨至10月26日的高点0.078，涨幅约为69%；BLUR由10月12日地点的0.152上涨至10月25日的高点0.264，涨幅约为73.3%。X2Y2由9月28日的0.00908上涨至10月8日的阶段性高点0.02115，涨幅约为133%。

这三个币种均在10月18日有小幅币价高峰，而此时BTC价格尚未突破3万美元关口，仍在2.9万美元附近徘徊。

1、 OpenSea

OpenSea作为首个也是最大的NFT市场不用过多介绍，但在加密熊市行情下依然面临着业务数据锐减的打击。

从DappRader给出的数据可以看出，当前OpenSea各项数据均已跌至2021年9—10月低值水平。不过今年以来其项目方也在积极做事，以图改变现有NFT冰河期。

今年10月，OpenSea 宣布创作者工具 OpenSea Studio 上线，同时计划在未来几个月内添加新功能，包括新集合页面和快速删除 NFT 版本等；8月，平台公布可赎回NFT标准，并计划向所有创作者推出链上可赎回支持。并且从当月底开始，平台实行创作者版税机制，对用户收取 0.5% 的平台交易费用。

7月份，OpenSea 推出 NFT 互换功能“Deals”，用户可组合 NFT 和 WETH 直接向其他人发起 NFT 交换；5月，OpenSea Pro增加“Farmer Wallet”标记功能，而OpenSea Pro（前 Gem v2）则是在前一个月中由OpenSea 与旗下 NFT 聚合器 Gem 联合推出；而在今年的前三个月中，平台先后为创作者引入自助Drops代理合约、新增NFT保护措施、完成Seaport 1.4部署等。

2、LooksRare

LooksRare是以社区为优先级的NFT平台，这意味着平台会优先奖励、授权并回馈平台的用户和创作者。

该项目于去年1月上线，其中UAW和交易数量在去年12月份达到峰值，此后迅速暴跌并持续至今。而其交易额却有多次到达峰顶的情况，分别是在今年的1月和4、5月份，均超过900万美元。但进入今年10月后却突然出现暴跌情形，从150万美元下跌至10万美元附近。

而交易量的下跌可能与LooksRare更新代币经济模型有正相关，项目官方在9月30日宣布未来费用的 50% 将用于从市场上回购 LOOKS 代币，10% 将作为 LooksRare 协议奖励，40% 将上缴协议财库。

此外，今年8月份，LooksRare 推出平台激励计划第三季，但效果不太明显；5月，项目宣布推出移动应用测试版；4月，项目上线 V2 版本，新版本包括将协议费从 2% 降为 0.5%，Gas 费用优化升级，批量购买、批量上架、自定义发送资产等。3月，项目聚合器上线，用户可以将 NFT 添加到购物车，并从 LooksRare 与 OpenSea 批量购买。

3、Blur

 Blur是一个集交易及聚合功能的NFT市场，该平台提供如实时价格更新、投资组合管理与多市场 NFT 比较等高级功能。此外，该平台还宣称可提供更快的 NFT 扫货功能，且用户界面比其他平台更为直观。

2022年10月 Blur正式推出，问世后迅速受到NFT专业投资者的欢迎。作为基于以太坊的平台，目前已许多知名投资人与交易者处共筹集了超过1400万美元，其中包括：Paradigm、6529、Cozomo Medici、dhof、Bharat Krymo、Zeneca、OSF、MoonOverlord、icebergy、Deeze、Andy8052、Keyboard Monkey 等。

由于Blur 加曾在 NFT 市场版税大战中让用户自由设定版税比率，同时还会以空投 Token 激励等方式来鼓励用户支付版税，导致其流量一度超越OpenSea。然而尽管Blur开创了诸多NFT交易市场的玩法，但依旧未能阻挡住NFT退潮的势头。

今年2月，Blur各项数据来到历史峰值，UAW达到57000，交易数量为78000笔，交易量高达9800万美元 。但此后这三项数据锐减，并在9月到达最低点。不过从近30天表现来看，已略有回暖的迹象。

综合来看，Blur毁誉参半，但就今年来说，其团队仍在继续建设项目。

10月，Blur发文暗示Season 2将于11月20日迎来重要节点；7 月项目新增 NFT 项目表现对比功能，同时上线 V2 版本， 对 Gas 进行优化，交易成本降低 50%；项目支持用户在移动设备上使用出价、上架、借贷、先买后付以及投资组合管理等功能。

4、X2Y2

X2Y2上线于2022年2月，在功能上总体与OpenSea类似，但x2y2有批量上架和批量购买功能，同时也支持点对点的转账，但不收版税和手续费。

与其它NFT市场不同，X2Y2链上数据上涨下跌均相对较为和缓，并有多次高光表现。其中首次高点出现在上线当月，但彼时交易额不高，直到三个月后才连续取得交易额新高，最高约为3300万美元。而其UAW与交易数则在当年的8月份出现，随后三项数据在今年2月有所回升后就又跟随NFT衰退大潮跌落。

不过值得注意的是，以上三项数据自9月20日左右开始小幅爬升。

但与上述三个NFT市场相比，X2Y2今年里的动作要少得多。今年8月，X2Y2宣布平台原“Terms”功能被移除，由“Listing Offer”功能取代，后者允许借款人根据借款人期望的条款提出要约；5月，平台推出X2Y2 Fi和V3版本，其中X2Y2 Fi专门为NFT 提供金融化服务，V3版本核心是为借贷双方提供更好的用户体验；3月，平台对NFT贷款和再融资收取管理费。

5、OKX NFT

欧易 NFT 市场是一家多网络运行的去中心化 NFT 交易平台，当前已累计集成18条主流公链。

平台除提供一级发行市场发售NFT项目外，还建有二级市场交易功能，支持用户多维度筛选查询及批量购买。同时该平台也实时能够获取 OpenSea、Blur、LooksRare、Magic Eden、IMX Official 等主流平台 NFT 订单，支持一键挂单至 OKX、OpenSea、LooksRare。

与其他几家NFT市场相比，OKX NFT在今年表现堪称优异。其链上主要三项数据自今年5月以来屡次取得前高，并且交易量也在持续增长中。而UAW和交易数最高点出现在8月初，交易量最高点在九月中旬，约为1821万美元。值得注意的是，该数据能够稳步发展。

由于去年OKX NFT基建基本完成，因此今年其主要以扩大合作伙伴与举报活动打造业务范围和市场影响力为主。

总结

当前NFT市场依旧一片凄风楚雨，尽管部分蓝筹NFT项目地板价略有回升，但相距高点依然相差甚远。尤其是今年6月Azuki  Elementals发售事件导致NFT社区成员信心受损，NFT市场更能像其他区块链赛道一般崛起。

但是就近期主流NFT市场的表现来看，交互钱包地址数、交易数量以及交易量等均有所上升。尤其是各NFT平台代币价格的上涨在一定程度上缓解了平台的流动性不足压力，同时也为投资者带来了些许信心。

因此，正如世界经济论坛《NFT 演变》报告所说，NFT依然蕴藏着重大的价值创造属性，能够为企业与用户带来新的深厚联系。
