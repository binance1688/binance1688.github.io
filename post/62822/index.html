游戏引擎：他们是游戏世界的无名英雄。这些基础框架已经占据了中心舞台，他们在确保开发团队能够专注于实际制作游戏，而无需每次都重新发明轮子。

图片来源：www.gamesradar.com

从历史上看，Unity 和 Unreal Engine 等名字一直主导着对话。他们在传统游戏中的影响力是不可否认的。随着数字景观的扩展，开发人员发现自己来到了十字路口：是坚持使用现成的第三方游戏引擎，还是冒险定制自己的游戏引擎。CD Projekt Red 等巨头选择了后者，因为他们意识到自己的游戏引擎意味着无与伦比的控制力和灵活性，可以将他们的特定愿景变为现实。

当我们来到游戏的下一个前沿：全链游戏领域时，我们发现自己正处于技术淘金热之中。Lattice 的 MUD 和 Starknet 的 Dojo 等引擎不断涌现，暗示着全链游戏有着巨大的、尚未开发的潜力。虽然基础设施的增长速度似乎超过了优质游戏的生产速度，但这正是大家对一个处于转型边缘的行业的预期 。本文将探讨全链游戏引擎的最新进展，探讨它们的独特功能、挑战以及对行业的潜在影响。

MUD

MUD 是由 Lattice 在 0xParc 驻场期间创建的。它是自主世界领域的灯塔，这是一个显示出巨大前景的新领域。从多用户地下城的丰富遗产中汲取灵感，MUD 提供了链上逻辑和客户端渲染的混合 。

开发人员可以在 MUD 中设计被称为 Worlds 的扩展链上数据结构。每个 World 都包含保存数据包或组件的“实体” 。这些组件赋予游戏机制生命力,并控制交互。此外，MUD 中嵌入的“系统”决定了这些组件如何相互作用，从而使开发人员能够融入独特的动态和游戏玩法。

MUD 的另一个突出功能是它使用本地索引器。这些索引器加快了状态检索，弥合了链上逻辑和客户端可视化之间的巨大鸿沟 。MUD 还展示了有效的事务管理。它具有在链上交易最终进入区块链之前，在本地模拟链上交易的能力 。Skystrife 和 Primodium 等游戏证明了其灵活性和潜力。

Dojo

Dojo 是由 Cartridge、Realms 和 BibliothecaDAO 牵头开发的游戏引擎。它专为 Starknet 设计，专注于为游戏提供增强的所有权、互操作性和可扩展性，同时确保对游戏至关重要的、更快且更具成本效益的交易。Starknet 是采用 Validity Rollups 的以太坊 L2 区块链，使用称为 STARK 的密码学技术来大幅降低计算和存储成本。它以最低的成本，实现高每秒交易率（TPS），确保以太坊的安全 。这使得区块链上的游戏更快、更便宜。

Dojo 是 Starknet 上的明星全链游戏引擎，使开发人员能够在链上环境中无需重新开发基本元素即可制作游戏。

Realms 和 Cafe Cosmos 等游戏已经在 Starknet 上使用 Dojo 进行开发。Dojo 的众多工具和功能包括：

1. 实体组件系统（ECS） ：一种增强代码可维护性的设计模式 。

2. Sozu 和 Torii ：用于游戏部署和高效游戏世界事件管理的专用工具。

3. Katana ：本地 Starknet 快速节点，提升开发体验。

Dojo 将推出多项突破性创新，以塑造区块链游戏的未来。其中包括在 Starknet 上探索 L3 游戏、引入乐观更新和客户端证明 ，这将增强可扩展性、实时交易，并使直接从客户端进行部分证明成为可能。

Paima Engine

Paima Engine 由 Paima Studios 开发，使开发人员能够利用他们的 Web2 编程专业知识来制作无需信任的 Web3 游戏。该游戏引擎允许开发人员使用熟悉的工具，如 JavaScript、关系型数据库和标准 Web2 开发堆栈，同时集成先进的以太坊 L2 区块链功能。

其关键组件之一是 Paima Whirlpool，它使跨不同区块链平台的互操作性和可访问性成为可能 。有了这个功能，就可以消除对单个区块链网络的依赖。Paima 还提供游戏模板，通过提供预设的游戏结构来加快游戏开发速度。

该引擎的设计支持 Unity 和 Unreal 等流行的游戏引擎，并包括连接游戏前端、区块链和游戏节点的中间件。该中间件还与各种 Web3 钱包进行通信，并使用智能合约来捕获和存储游戏操作。Paima 还强调数据存储和安全性，与 AIL 和 Celestia 等项目合作，确保信息广泛可用且安全。

创新是 Paima Studios 的核心。他们还引入了 Stateful NFT，将游戏玩家的行为与其区块链身份联系起来 。该工作室旨在简化开发者和玩家的游戏交互，提供用户友好的界面和工具。在 Cardano 社区资助的支持下，Paima Studios 将引领链上游戏的进步。

Keystone

Keystone 是 Curio 的一个 Rollup 框架，构建在 Optimism 的 OP Stack 上 。它认识到复杂的链上游戏面临的与执行速度瓶颈相关的挑战，特别是当链上状态扩展和并发交易增加时。此外，传统区块链缺乏实现最佳游戏性能的基本功能，例如游戏 tick。

与 Dojo 一样，Keystone 采用了实体组件系统（ECS）开发模式，将执行速度提高了100倍，并支持并行执行。基于 EVM 的 Optimism 支持 Solidity 智能合约与底层 ECS 状态交互。这是通过可以检索和更新数据的预编译函数来实现的。Keystone 还将游戏 tick 集成到区块生产过程中。前一个块的用户输入影响新状态的产生。

World Engine

Argus Labs 的 World Engine 使开发人员能够使用其独特的 L2 分片架构打造开放、可互操作的游戏世界。World Engine 的使命是创建游戏互联网，它使每个游戏拥有自己的可扩展区块链成为可能，该区块链不断发展并由其社区拥有。

World Engine 为游戏提供了先进的功能。通过将游戏的工作分散到不同的部分，它可以轻松应对不断增长的需求。该引擎确保游戏平稳运行，同时仍与区块链技术连接。它的关键部分之一是 Cardinal，它旨在与 Unity 和 Unreal 等流行游戏引擎良好配合。它还允许开发人员使用 Go 来创建游戏规则。

此外，该引擎支持流畅的交互，允许玩家创建独特的内容并与游戏碎分片轻松连接。Polaris 是引擎的另一个元素，有助于定制从简单到复杂任务的游戏功能。

结束语

在不断发展的全链游戏格局中，MUD、Dojo、Paima Engine、Keystone 和 World Engine 证明了传统游戏实践与区块链技术的融合。这些引擎都有一个共同的愿景：无缝衔接 Web2 和 Web3 游戏领域。

MUD 专注于将常规游戏视觉效果与区块链原理相结合，Dojo 提高游戏速度，Paima 的 Whirlpool 确保不同的游戏协同工作，Keystone 专注于同时运行多个任务，而 World Engine 让开发人员创建开放的游戏世界 。但最终，所有这些游戏引擎的目标是为游戏创作者提供最好的工具和自由，以实现他们的想法和愿景。

从这些游戏引擎的发展方向，我们可以自信地说，未来是光明的，尤其是在全链游戏领域。快节奏的游戏机制与区块链不变的信任相结合，正在为游戏革命奠定基础。

随着这些工具和框架的进一步发展，我们可以期待更全面的链上游戏，任何人都更容易参与，有无尽的创造力，并且每个玩家都是主要焦点。在全链游戏和一般区块链游戏的下一阶段，游戏引擎不仅仅是工具；它们是令人兴奋的在线世界的基础 ，将改变我们思考和玩游戏的方式 。
