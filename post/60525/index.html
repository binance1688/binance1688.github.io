译者导读：

随着互联网的快速发展，中心化的平台逐渐主导了整个网络,形成了被少数巨头所控制的格局。尽管这种模式确实带来了一系列便利，但也导致了数据隐私、决策不透明和算法操控等问题的出现。

在这一背景下，区块链与加密货币代表了回归开放、透明的协议运动。作为最初的区块链,比特币协议不仅在技术上实现了去中心化，也在经济和治理设计方面进行了创新探索，内置的代币激励使其拥有近乎不可阻挡的生命力。正如Ralph Merkle所说的，Bitcoin之所以存活下来是因为它付费让人们维持它的生命；是因为它提供了一项有用的服务，人们愿意为之买单。

基于开放且透明的，特别是去中心的代币化协议,构建一个去中心化的、权力下放的网络(Web3),被视为重塑互联网格局的关键方向。这种模式有望减少对中心化平台的依赖,激发 创新和竞争,但同时也需要克服一系列挑战,包括可扩展性、监管合规和用户体验等方面的持续改进。

通过充分利用网络、平台和协议，支持可信赖的品牌，以扩大人们获取知识、资本和福祉的机会。

https://www.usv.com/#thesis-3-0

最后那个术语（即”协议”）虽然具有重要意义，但不幸的是，与句子中的其它术语相比，它的理解程度较低。

协议一直存在，它是人与人之间相互遵守的规范准则。‍‍‍‍‍‍‍

在计算机科学中，协议也是类似的，是一种行为准则。

我喜欢 Cloudflare 网站上关于计算机协议的这个定义：

标准化协议就像计算机可以使用的共同语言，类似于来自世界不同地区的两个人可能无法理解彼此的母语，但他们可以使用共同的第三种语言进行交流。如果一台计算机使用 Internet Protocol（IP），而第二台计算机也使用相同的协议，它们将能够通信 - 就像联合国依赖其 6 种官方语言与来自世界各地的代表进行交流一样。但如果一台计算机使用 IP，而另一台不了解这个协议，它们将无法进行通信。

https://www.cloudflare.com/learning/network-layer/what-is-a-protocol/

协议已经存在了与计算机同样长的时间，但我们正处在一个我称之为 “Web3” 的协议黄金时代的开端。

直到最近，协议大多是免费的，没有货币化。我们每天都使用 HTTP 协议来访问网页。我们每天都使用 SMTP 协议来发送电子邮件。但这些协议都是免费使用的，不会为任何公司、项目或个人带来收益。

随着比特币在2009年1月的出现，我们得到了一个内置了货币化机制的协议。

从那时起，计算机科学家们一直在为各种事物创建 Web3 协议，其中大多数都包含了用于货币化协议的代币。

货币化不仅仅是为了让协议的创建者获益，它还用于激励协议的使用和运营。 在比特币协议中，这包括运行挖矿计算机来维护比特币网络的安全。在 Helium 协议中，这包括运营热点设备以连接网络。在以太坊协议中，这包括质押代币以确保网络的安全。在 Filecoin 协议中，这包括运营存储系统以提供去中心化存储。这些措施鼓励了协议的积极使用和顺畅运行。

正是通过货币化，这些协议得以维持并为其他人提供可靠的使用和构建基础。

而有了可持续和可信赖的协议，我们将迎来一种全新的服务架构，这将更加具备互操作性。‍‍‍

让我们以移动消息传递为例。我的大多数朋友和家人都使用 iPhone，而我使用 Android。然而，iPhone 和 Android 手机之间的消息传递效果并不理想。全球许多人使用 WhatsApp 进行移动消息传递，但 WhatsApp 无法与 iPhone 的消息应用相互通信，Signal 也不行，而 Signal 与 WhatsApp 也不互通，如此循环下去。

如果每个移动消息应用都建立在可实现货币化的协议基础上，这些协议因为运营它们而能够让个人和公司获益，并且开发者也有激励来构建它们以赚取收益，那么情况将完全不同。所有我们的移动消息应用将能够互相通信。

在未来的几十年里，随着 Web3 协议的设计、构建、发布和广泛应用，每个计算领域都将经历类似的情景。尽管我们正处于协议黄金时代的早期阶段，但在去中心化基础设施和去中心化金融领域，已经有许多协议表现出色。

令我感到不安的是，协议具备如此巨大的潜力，但在媒体、主流社会以及政府中，对它们的了解却如此有限。

最近，白宫发布了一份文件，声称“加密资产目前并没有为广泛的经济利益做出贡献”，似乎表明 Web3 对美国的经济影响并不重要。尽管如今 Web3 的经济利益尚未广泛传播，但它们的潜力相当巨大，最终将对每个经济体和市场产生影响。然而，对其发展持消极态度并不会让它们消失，这是一个值得庆幸的事实。
