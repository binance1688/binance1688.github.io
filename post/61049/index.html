作者：David Rotfleisch，编译：TaxDAO

2023 年 3 月，以色列税务机关宣布逮捕两名涉嫌逃税的以色列公民，他们涉嫌从 NFT 交易中获得超过 200 万美元的未报告收入。NFT 本质上代表了独特的数字资产，是不可分割的数字代币。每个 NFT 的代币所有权都记录在数字分类账簿上，通常使用以太坊区块链和智能合约将代币链接到特定数据，这些数据可以是艺术品或任何其他收藏品。通常这些数据在基于云的服务上托管，并允许 NFT 持有者访问。同时，NFT 内容创作者使用公钥来作为其真实性和出处的证明。

据《耶路撒冷邮报》和以色列税务局报道，这两个人是 Holy Rock NFT 项目的创建者，他们试图创建代表耶路撒冷西墙每块石头的独特 NFT，并打算将它们出售给数字收藏家。他们随后出售了该项目生成的 NFT 以换取 ETH。他们进一步将这些 ETH 转移到不同的加密货币钱包中，使得这些交易在公共账本上的可追溯性变得更加复杂。据称，被指控的这两个人没有报告其出售 NFT 换取 ETH 所产生的任何利润。

包括加密货币和 NFT 在内的区块链技术的迅速普及，引起了全球执法机构和税务机关的担忧。他们担心这些数字资产可能会通过掩盖应税交易的性质从而达到逃税目的。这种担忧源于区块链技术为数字资产交易者提供的固有匿名性。虽然区块链交易本质上是公开的，但与数字分类账簿上关联的唯一钱包地址并不能揭示钱包持有者的真实身份。为了应对这些担忧，以色列税务局和加拿大税务局等监管机构已采取严格措施，打击将加密资产作为隐瞒未报告收入手段的行为。

尽管加拿大税务局尚未针对全球 NFT 市场发布任何合规或执行通知，但它已在虚拟货币和加密货币的背景下采取了实际行动。尽管 Holy Rock NFT 项目主要与以色列相关，但它可以作为一个有趣的案例研究，用于探讨加拿大税务局在逃税问题方面的调查权以及逃税者潜在的民事和刑事后果。本文将首先深入探讨加拿大加密货币税收体系中对 NFT 征税的基本原则。随后，将阐明加拿大税务局在调查涉嫌逃税事件中所采用的方法，以及可能对有意或无意参与加密逃税的纳税人施加的一系列处罚。最后，我们将提供一些加密货币税务提示。

为什么 NFT 处置在加拿大要承担纳税义务？

根据加拿大所得税法第 3 条，加拿大纳税人应税收入包括来自境内和境外产生的所有收入。这包括业务或资产产生的收入，本质上涵盖了 NFT 的销售。在加拿大税收框架中，每当处置资产时必须计算、报告该交易产生的收益或损失，并在处置当年纳税。无论资产是否以法定货币或加密货币出售或交换为其他资产，均适用此税项。因此，每当 NFT 被出售为法定货币或交易为另一种资产（例如加密货币或其他 NFT）时，加拿大税务系统都会将这种销售或交易视为应税事件，因此需要计算和报告当年因处置而产生的加密收益或损失。

NFT 处置对加拿大纳税人的税务影响是显而易见的。关键点或技术难点是这种处置是否应该被归类为资本利得或来自业务的收入（这可以包括一次性处置）。这对于加密货币税务规划目的非常重要，因为与业务收入不同，纳税人的资本利得中只有一半会被视为处置当年的应税收入。相反，如果处置导致资本损失，则只有该损失的一半可以扣除，并且只能从资本收益中扣除。鉴于全球 NFT 市场的波动性，区分资本利得和业务收入可能会对加拿大纳税人确定纳税义务方面产生重大影响。

确定财产处置产生的利润或损失是否应被视为资本或收入一直是加拿大税务法院广泛关注的主题。尽管加拿大税务法院尚未审理有关 NFT 出售的案件，但有理由预期这些原则将继续适用于 NFT 处置收益的定性。重要的是要认识到，没有任何一个因素可以单独决定对财产处置所得收益的适当分类。应根据每个案例的独特案情进行评估。然而，加拿大法院一贯概述了在确定财产处置是作为资本还是收入进行区分时应考虑的具体客观标准。在 NFT 的背景下，这些因素可能包括：

  • NFT 交易频率 ：例如如果纳税人频繁从事 NFT 买卖或 NFT 周转率较高，则可能表明其属于商业活动，而不是资本投资；

  • 所有权期限 ：如果纳税人倾向于短期持有 NFT，这可能表明其是商业活动，而不是长期资本投资；

  • 擅长 NFT 市场 ：纳税人在 NFT 市场丰富的专业知识可能表明其是商业活动；

  • 与纳税人其他职业的关系 ：如果 NFT 交易或类似活动与纳税人的主要职业或其他业务相关，则倾向于商业活动分类；

  • 投资于 NFT 的时间 ：如果纳税人的很大一部分时间致力于研究潜在的 NFT 收购、分析 NFT 市场或积极管理 NFT 投资组合，则表明存在商业运营；

  • 财务支持 ：杠杆化的 NFT 购买和交易表明存在商业运营；

  • 广告 ：如果纳税人将其 NFT 活动宣传为一项业务或公开宣传其参与 NFT，则会增加被归类为商业活动的可能性。

一般而言，在确定出售财产是否会产生资本收益或营业收入时，最重要的因素是纳税人在购买财产时的意图。加拿大法院和税务局将结合前面提到的因素，彻底审查有关财产收购和出售的客观情况，以确定纳税人获得财产的根本目的。值得注意的是，《加拿大所得税法》第 248(1)款中“业务”的定义包含“贸易性质的冒险或关注”。这种解释意味着如果证实了这种特征，即使是单一的收购和出售财产的情况也可以被归类为营业收入。

不充分申报 NFT 处置税务的后果

纳税人可能因逃税而受到严厉的民事甚至刑事处罚，具体取决于其不当行为的程度。《加拿大所得税法》第 239(1)款概述了加拿大纳税人可能因故意在纳税申报表上作出虚假陈述、参与此类行动或故意操纵或销毁财务记录以隐瞒纳税义务而面临巨额罚款和潜在监禁的各种情况。根据第239(1)款定罪，可处以逃税额 50% 至 200% 的处罚或最高两年监禁。

逃税案件中潜在刑事责任的决定因素是纳税人在实施相关行为时的心理。另外，第 163(2)款提供了一个民事框架，用于惩罚故意或在严重过失的情况下在纳税申报表上作出虚假陈述或遗漏的纳税人（这些通常称为“重大过失处罚”）。根据第 163(2)款，对于虚假陈述或遗漏，纳税人可能被追究责任，以以下两者中较高者为准：(1)100 美元；（二）纳税人应缴纳税款总额的 50%。

数字账本为 NFT 交易者提供的匿名性给税务局对涉嫌逃税的加拿大 NFT 交易者启动加密货币税务审计带来了挑战。税务局在税务审计期间收集加拿大纳税人信息的现有权力在专门调查参与钱包对钱包交易的纳税人时作用有限。然而，数字资产交易所作为 NFT 交易促进者的重要性日益增强，为税务局针对加拿大 NFT 交易者的税务审计提供了关键途径。《加拿大所得税法》第 231.2 条授予税务审计员和调查员强制个人提供税务调查所需文件或信息的权力。这一权力特别广泛，历来允许税务局要求银行和其他第三方金融机构合作，从而生成报表和记录。税务局毫不犹豫地利用这一权力来迫使数字资产市场提供有关加拿大加密货币和 NFT 交易者的信息。

2020 年，税务局成功获得联邦法院的命令，迫使总部位于多伦多的加密货币平台 Coinsquare Ltd. 披露与加拿大用户的交易活动和加密货币持有量有关的机密税务信息。因此，尽管数字账本提供了匿名性，但税务局保留了对使用交易平台和市场进行交易活动的 NFT 交易者进行审计的手段。

如果税务局在税务审计期间发现可能对纳税人提出刑事指控的证据，则该案件将根据加拿大刑法移交给司法部起诉。虽然税务局与司法部共享民事税务审计期间收集的信息的能力没有受到限制，但税务局对民事税务审计的调查权不能用于协助刑事调查。当税务局税务审计的主要目的转向调查犯罪或准犯罪活动时，就会触发《加拿大权利与自由宪章》第 7 条规定纳税人受宪法保护的防止自证其罪的权利。此外，《加拿大权利与自由宪章》第 8 条规定的纳税人权利可保障纳税人免受国家无理搜查和扣押。这使得税务局根据第 231.2 条规定强制为此类调查提供文件的权力无效。虽然税务局和司法部都有责任在税务审计转变为刑事或准刑事调查时通知纳税人，但如果他们未能这样做，则纳税人有责任对税务局收集刑事案件证据权力的不当使用提出质疑。这就要求纳税人证明税务局在税务审计期间收集信息的主要目的是用于刑事调查而不是民事税务审计，这对任何纳税人来说都是一个巨大的挑战。

税务专家提示——通过利用自愿披露计划最大限度地减少刑事责任风险

税务局的自愿披露计划 (Voluntary Disclosures Program，VDP) 提供了一个重要途径，以避免与加密货币逃税相关的潜在刑事起诉，包括涉及未报告的 NFT 处置的案件。通过 VDP，加拿大纳税人可以在税务局发起调查之前主动向税务局披露任何税务违规事项。如果纳税人符合税务局的减免标准，就可能有资格获得多项福利，包括免除违规处罚，例如重大过失处罚、未缴税款的部分利息减免，以及特别是免受刑事起诉。

及时有效的自愿披露申请非常重要。要获得通过 VDP 获得减免的资格，提交的披露必须符合特定标准。首先，它必须是真正自愿的，这意味着它必须先于税务局就所考虑的不合规问题采取任何执法行动之前。

其次披露必须全面。这不仅需要包括与特定税务违规事项相关的所有相关信息，还需要解决承担责任的任何其他违规问题。因此，如果发现自己面临因 NFT 处置而产生的未报告税款以及其他来源的未披露收入，那么确保 VDP 资格取决于披露和纠正这两种形式的违规行为。

此外，除了提高税务局授予税收特赦的可能性之外，精心设计的披露申请还可作为在加拿大税务局不公正地拒绝自愿披露提交的情况下向联邦法院提出司法审查申请的基础。
