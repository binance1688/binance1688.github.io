在数字资产领域，私钥的安全管理是用户最为关心的问题之一。传统的钱包解决方案往往要求用户记忆或保存复杂的助记词或私钥；一旦丢失或泄露，会造成不可挽回的资产损失。

为了解决这个安全痛点，同时降低用户使用钱包的门槛， 10 月 23 日，Bitget Wallet（原 BitKeep）宣布正式推出了 MPC 无私钥钱包，采用企业级 MPC（多方计算）解决方案，为用户提供更加安全、便捷的 Web3 钱包服务。

“无私钥”体验，安全又易用

MPC 钱包对于用户来说，其使用体验相较于目前大部分用户正在使用的 EOA 钱包差异非常明显；因为无需记忆或抄写复杂的助记词或私钥，实现“无私钥”的使用体验。

Bitget Wallet 的 MPC 钱包，通过邮箱登录认证即可创建，即使是小白用户也能轻松无障碍使用。此外，简单 3 步便可完成密钥分片的云备份，有效规避私钥遗忘、丢失、被盗的各种风险。

可以说，“无私钥”，更易用也更安全。

安全可靠的企业级 MPC 技术方案

Bitget Wallet 的 MPC 钱包采用了 MPC 中特定的门限签名（TSS）技术，且严格遵循最高的安全标准，主要应用了以下技术：安全大质数、分布式密钥生成（DKG）、分布式的协同签名、同态加密、最常用的 2/3 门限阈值，在各个阶段都能达到更高安全防护级别。

在创建钱包时，Bitget Wallet 会生成抗量子攻击的安全大质数。Bitget Wallet 研发团队介绍称，“安全大质数的生成是为确保后续以最安全的标准运行协议的各个环节，从而提高整体的抗攻击能力”，生成大质数的过程只在首次创建钱包时进行，大约需要等待几十秒，“虽然用户的体感会慢一点，但安全性大大提高”。

在生成 MPC 钱包的私钥时，Bitget Wallet 通过分布式的密钥生成技术，在多方协同计算下，分散地生成“隐式映射完整私钥”的私钥分片，且每一个分片永远有一个碎片不会被传输，从而保证私钥分片不会被泄露，完整私钥从未出现也不会被泄露。

在签名场景中，Bitget Wallet 通过分布式的门限签名技术，使签名过程中的私钥始终处于分散状态，用户在不暴露私钥的前提下完成交易签名，即整个过程私钥从未出现，大大提高私钥和资产的安全性。

Bitget Wallet 研发团队表示，“目前市场上大多数面向 C 端用户的 MPC 钱包，在安全性与便利性之间更倾向后者。比如在登录验证方式上，与中心化交易所共用一套账户系统，虽然创建更快，但风险无法分散；比如在分布式密钥生成过程中仅使用普通大质数，同样牺牲了一定的安全性。”

Reshare 机制、独立密码，提供多重保障

Bitget Wallet 还为用户引入了 Reshare 机制。当用户更换新设备登录钱包时，旧设备的密钥片段将自动失效，无法再完成交易签名，大大降低了资产丢失的风险。同时，Bitget Wallet 支持为 MPC 钱包设置独立的交易密码，保障服务端的密钥分片只有在用户的许可下才会参与签名计算。

用户拥有钱包的绝对掌控权

Bitget Wallet 的 MPC 钱包采用 2/3 签名机制，为 C 端用户提供了一套完整的企业级 MPC 技术解决方案。

2/3 签名机制是指最低签名阈值/私钥分片总数量 = 2/3 ，一片在用户当前设备上（用于参与签名）、一片在平台的服务器（用于参与签名）、另一片作为备份可保存至 iCloud、Google Drive 等云端（不参与签名，作为备份用于 Reshare），这比混合托管（用户和平台共同掌控私钥，各自单方面无法动用资金）、第三方托管（平台掌控私钥，可以动用用户资金）都更加去中心化和安全。

此外，通过各方独立生成安全大质数，而不是中心化服务器快速生成，保证了密钥分片生成过程的去中心化；用户可通过 Reshare 来使旧设备的密钥分片失效等。

总之，保证只有用户才拥有对钱包的绝对掌控权。

一直以来 Bitget Wallet 都在产品和服务上追求创新，致力于为用户提供更优质的产品和更便捷的服务。

“Bitget Wallet 内部有个说法，我们要面向 10 亿用户做产品、提供服务。Crypto 和 Web3 被大规模采用，需要钱包这样的基础设施为用户降低门槛、提供便利。MPC 钱包是一个积极的尝试，未来我们会继续在 MPC 钱包以及 AA 钱包等诸多领域做更多的探索。”Bitget Wallet CEO 表示。

关于 Bitget Wallet

Bitget Wallet（原 BitKeep 钱包）是一个一站式 Web3 多链钱包，为用户提供包括钱包、Swap 交易、NFT 交易、DApp 浏览器等全方位的链上产品和 DeFi 服务。

成立 5 年以来，因安全便捷的产品和服务，获得全球 1200 万用户以及 Ethereum、BNB Chain、Arbitrum、Polygon 等数百个行业头部项目的认可与支持。
