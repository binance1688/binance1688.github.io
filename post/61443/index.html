10月 18 日，以太坊联合创始人 Vitalik Buterin 在基于 Farcaster 的社交媒体平台 Warpcast 上进行了一场开放式 AMA，吴说整理如下，部分参考深潮、星球日报：

1、量子计算可能比预期来得更快，以太坊在面对量子霸权方面，有何紧迫感以及缓解思路呢？

Vitalik：账户抽象（ERC 4337）架构下的用户账户已经具备对抗量子计算的能力。对于现有账户，如果量子计算突然出现，我们可能需要进行硬分叉，通过 ZK-STARK 证明用户的 12 个助记词（助记词已经是抗量子计算的，应该是目前大多数用户的账户模式）来恢复其的 EOA。

在共识层，基于 STARK 的替代方案正在积极探讨中，包括（i）BLS 聚合，（ii）Verkle 树，（iii）基于 KZG 的 DAS。

2、你对基于格（Lattice-based）的SNARK替代方案感兴趣吗？毕竟，基于格的加密系统已经成为NIST PQC标准的主要候选项之一，并且与全同态加密系统（FHE）兼容。

ps：基于格的密码学因其抗量子计算机攻击的特性而备受关注

Vitalik：对于全同态加密（FHE），基于格的方法是合理的，而且似乎是唯一的选择。对于大多数链上操作，ZK-STARKs 结合哈希可能就足够了。

3、你最喜欢在哪个城市（或地方）散步？成为公众人物后，喜欢有所变化吗？

Vitalik：最近我明显变得不那么喜欢城市了。即使我在新加坡，我也更倾向于待在离市中心更远的地方，靠近大型水库公园的地区。城市方面，我比较喜欢新加坡、台北、维也纳、多伦多、纽约州伊萨卡市。

4、今天早上你吃了什么？

Vitalik：大约16克的Lindt 85%纯黑巧克力。我用完了我的常吃的90%纯黑巧克力，那是我的最爱。

4.1 你不认同寿命科学界许多人提倡的一日一餐原则，以延长寿命吗？

Vitalik：我通常一天吃两顿饭，分别是早午餐（brunch）和午晚餐（linner），它们之间相隔大约6个小时。

5、你如何看待a16z联创 Marc Andreessen 近期发表的文章《技术乐观主义宣言》？

Vitalik：我同意其中的很多观点——尤其是，衰退是不好的，很多伪装成谨慎和谦卑的停滞主义实际上是一个非常反人类的力量。但我也认为它遗漏了很多内容；具体来说，技术发展的方向和顺序很重要。

例如，我认为大脑计算机接口非常棒，将来会非常重要，但它们也存在风险：我们实际上在谈论读取+写入我们的思想。对我来说，至关重要的是，这些东西需要尽可能开源，有利于于本地计算。a16z 正在做什么来实现这一结果？

以及，我认为加密货币在一定程度上是一个技术领域，它不仅要考虑消费者的效用，还要考虑这些重大的技术政治问题，维护自由、开放以及世界各地普通人参与而不仅仅是消费的能力等。

我认为有另一个非常有价值的领域可以有意加速，那就是国防技术。例如，在生物领域，创建一个流程，一旦出现新的大流行（无论是自然的还是人工的），我们可以：（i）迅速检测到它，（ii）迅速研发疫苗和/或抗体，（iii）进行本地生产，以便全球各地的人们都能迅速开始接种。

新大流行->反制措施流程，最好在不到一个月内完成。

在各个领域，只要进攻/防守平衡更有利于防守，全球就能更少地面临停滞、集中控制和灾难之间的黑暗抉择。

5.1 为此，我们如何设计系统，以帮助 Web3 以服务共同利益为导向进行扩展，而不仅仅关注市场份额和股东利润？Web3 提供了什么激励结构，使“Regen”的理念在社会/文明层面迅速传播？

ps：Regen 指再生性加密经济中的工作建设，更关注对 Web3 如何为世界带来好处这类长远想法，而不仅仅是在金融意义上。Regen 认为，金融系统可以被设计成更大的人类繁荣的渠道，为全人类的需求服务。

Vitalik：我认为需要更多的原始资金投入 Regen。Optimism 的 RPGF（Retroactive Public Goods Funding）很好，但我们还需要更多努力。

6、 你对气候变化有什么看法？

Vitalik：任何可能对世界造成此类影响的事情都值得担心。仅是 GDP 减半这件事，对于世界上最贫穷的国家已经很可怕的，更不用说对人类的直接影响了。不过虽说如此，但诸如“这将使世界变得如此糟糕，我不应该生孩子”之类的想法也是错误且有害的。

我投资了一家太阳能电池公司，并对更多的事情持开放态度。我们需要有意识地投入资源，同时沿着许多技术道路前进，以妥善解决这个问题。

7、你这些天都在听什么音乐？

Vitalik：一些高能量的华语歌曲非常适合在长跑的第 15-20公里听，且能显著提高跑步速度。比如张韶涵的《欧若拉》和《破茧》，几乎都能让我的速度提高 10%。最近我也会听一些日语歌。

8、Scroll 最近启动了其 zkEVM 链的主网；然而，由于它需要模仿以太坊上的经典 EVM，我们不太可能在执行层上看到创新的设计概念和实验，例如隐私功能。你对这些限制有什么看法？

Vitalik：我认为我们不需要执行层的隐私，我们完全可以在其他层建立它。

9、有以太坊的存在，你认为比特币还有价值吗？

Vitalik：Starknet 开发者 abdel.stark 的这篇文章“Ethereum Alignment”很好的解释了这个问题。比特币采用了各种不同的技术权衡，因此涵盖了与以太坊不同的假设空间。

https://hackmd.io/@abdelhamid/ethereum-alignment

10、是什么阻止了轻客户端（例如portal network客户端）尚未达到生产就绪状态？

ps：Portal Network（门户网络）旨在让资源较为有限的设备通过轻量级的方式访问协议。Portal Network 将由一个或多个去中心化的点对点网络组成，这些网络共同提供标准 JSON-RPC API 所需的数据和功能。这些网络经过专门设计，确保客户端只需使用最少的带宽、CPU、RAM 和硬盘资源即可参与进来。

Vitalik：马上就会实现了。我认为下一步是（i）默认集成到钱包中，并且（ii）将其扩展到覆盖 L2s，而不仅仅是 L1。不过，（ii）只有在 L2 起飞或至少减少对辅助功能的依赖，以使底层系统首先具有去中心化性质，才显得有意义。

11、在你看来，为什么基于加密货币的预测市场还没有获得太多关注？你仍然认为他们有潜力吗？他们需要再做些什么才能“起飞”？

Vitalik：他们正在慢慢接近目标。Polymarket 最近取得了相当大的进展，它似乎比之前的任何尝试都更好地糅合了 Crypto 属性与商业发展。

12、你那件用 8.5 BTC 买的 T 恤还在吗？

Vitalik：不在了。很遗憾，多年前我在英国火车上忘带包时把它搞丢了。

13、在你看来，传统金融以及晚期资本主义所面临的最大威胁分别是什么？

Vitalik：军队和晚期权威主义。

14、Zuzalu 项目的最大收获和教训是什么？它是否可以成为可复制的现实世界网络国家模型，还是更像是在寻找这个模型的过程中的一个实验？

ps: Zuzalu 是 Vitalik 于 2023 年 3~5 月在黑山发起的 pop-up city（快闪城市）实验。

Vitalik：我认为即使在今天，它仍然是一个可复制的模型！从政治自治的角度来看，你不会获得一个“网络国家”，但“200人在任何地方聚集数月”这种形式本身就非常有用。我们已经看到了类似的衍生项目。

15、 最近你读过的最好的文章是什么？

Vitalik：https://www.gleech.org/culture

16、鉴于近期关于协议底层是否封装更多功能的辩论，你是否赞成为每个人提供一种最小的链上治理工具，以表达他们的偏好？比如一个可靠的中立的信号功能工具，发给以太坊基金会和客户端团队（尽管存在 Sybil 攻击）？

Vitalik：是的，我绝对认为更多非约束性的情感投票工具将是很棒的。希望不仅仅基于代币；像 carbonvote 之类的东西在过去确实很酷，但最大的缺点显然是它们非常受大户的驱动。

17、作为一个博学、雄辩、并代表新互联网思维的知识分子，你会如何向普通大众传达以太坊意识形态系统所基于的未来？当前世界普遍将加密货币视为骗局，这是一个严重的沟通问题。

Vitalik：在我看来，我们已经进行了太多的传达，我们需要更多的示范行动！

18、不知道你对“票务”有什么看法。票务通常被认为是整个加密货币的一个有益的用例，对于数百万人来说，票务将是他们与分布式账本技术的第一次互动。

Vitalik：是的，我认为它是有价值的。尤其是如果我们能够（i）将其整合到像 zupass 这样的 ZK 系统中，并且（ii）使其实际上可以直接在链上购买门票，使整个过程完全不依赖于任何中心化的接口。

19、你有哪些与技术无关的兴趣爱好？

Vitalik：上周我正在研究道本语 toki pona，下周再看看它的发展。

ps：toki pona 是一门用极其精简、优雅的人造语言，只有 123 个单词。

20、你说过以太坊协议设计的目标是最小化 L1 协议复杂性并最大化稳健性和弹性，我们对比一下分片与 Rollup的情况，显然分片对于 L1 协议要复杂得多。但对于开发人员和用户来说，它更简单，因为它抽象了故障证明/有效性证明，消除了外生跨链桥，保证了 ETH 可互换性（arbETH 与 opETH）等等。为什么对于开发者和用户来说，最小化 L1 复杂性而不是将复杂性推向开发人员和用户是更好的选择？

Vitalik：我认为，如果我们选择了 L1 分片，那么我们拥有的那个非常强大的生态系统，负责实际开发各种 L2s 和扩展的基础设施，可能就不会在同样的程度上存在。L1 团队的资源受到限制，并专注于 PoS。

也就是说，现在的“L1 核心团队”更强大了，我觉得 L2 之间存在着朝更加标准化的方向发展的逆向运动。这包括：

●账户抽象（ERC4337）

●L2EVM努力

●标准化的跨链钱包体验，包括恢复

●未来可能采用的 L1-enshrined ZK-EVM。

当然，目前正在进行的关于DAS（数据可用性抽样）的工作，首先是 4844，然后是扩展到更大和更大的数据块大小。所以我认为以太坊生态系统非常有能力进化和纠正错误，我们最终会得到一个能够很好地平衡标准化和独立创新的东西。

20.1 因此，ETH 是否会放弃市值留给各种 L2 基础设施？如何看待 ETH 的 TAM （整体潜在市场）机会？

Vitalik：肯定放弃了一些，但不是全部。我认为这是健康的。原因是我们需要有一些代币，能够使用发行来支付昂贵的生态系统公共物品，ETH 本身很难发挥这个角色，所以为 L2 代币腾出空间是一个好的解决方法。你可以成为以太坊的一部分，并拥有一个代币，这个代币与你的项目更直接地关联，我认为这是一个巨大的激励。

21、从Farcaster产品角度出发，你会优先考虑做一些我们目前没有做的事情是什么？

Vitalik：无论你对抗 Sybil 攻击的策略是什么，确保它采用 ZK 封装、保护隐私，为不想提供政府身份证明或代理身份证明（例如电话号码）的人提供替代路径等。

22、你经常讨论移动设备上的链上安全性，但在像 Zuzalu 这样的实验之后，你发现了哪些未解决的离线移动安全挑战？你是否计划在网络状态方面采取进一步的行动？是否有办法让其他人协调共享安全的谢林点？

Vitalik：我们确实需要更多的投资来提升设备安全性、开源操作系统等。自我在今年初接触后，我已经向至少几个人介绍了 GrapheneOS。

我们的设备现在已经是我们思维的延伸，因此它们的安全性至关重要。

23、什么让你保持动力？

Vitalik：我知道我们的社区和它正在建设的技术处于一个非常独特的位置，可以在一个日益复杂的世界中做出真正有价值的事情。而积极的结果并不会主动发生，我们有责任仔细思考并实现它。

24、哪本书帮助你最好地理解西方文化是如何发展到这一点的（广泛的自由主义、民主和资本主义）？

Vitalik：我觉得书在这方面有些被高估，而对于理解文化的实际经验则被低估了。我不仅指的是去不同地方，生活在不同文化中，我还指的是加密社区。比特币、以太坊等也有价值观，也面临着塑造这些价值观但有时可能会妥协这些价值观的压力。

通过观察现有社区对各种规模的压力和威胁的反应，我们可以更好地了解在何种条件下自由主义、资本主义等最有利于繁荣。

25、你大部分时间都在思考什么？

Vitalik：在过去的一两周里，我一直在思考我对各种人工智能相关问题的看法，以及以太坊社区如何/是否能够有效地参与其中。

26、在充满内置钱包的未来，你认为人们会：

1. 为每个新应用程序提供一个新地址，

2. 在许多应用中主要使用一个地址，还是

3. 两者之间？

Vitalik：现在我们处于（2）。我希望我们能更接近（1），因为这更有利于隐私。有时不同的应用共享钱包是有意义的，但这不应该是默认设置。我们还需要继续改进 ZK 转账，以便不同活动所带来的隐私带来的隐私收益真正成为现实。

26.1 关于这个问题的后续问题——你的意思是每个应用中都该有这些不同的钱包，还是像 Metamask 这样的单一品牌钱包管理？也就是说，你使用 Metamask 登录并为每个网站生成一个地址，还是使用电子邮件/社交登录并为每个网站生成一个钱包？

Vitalik：我确实希望它是一个钱包，并为每个网站生成一个地址。如果每个应用都有一个自定义的钱包，在我看来将是一场灾难。

27、随着我对 SNARKs 在具有私有和公共数据混合的机制中的限制有了更多了解，我有一个问题：你对全同态加密（FHE）持什么看法？你认为过去两年的进展是否会导致实际代码方面的摩尔定律，就像我们在 Groth16 之后看到的那样？

Vitalik：FHE 仍然存在实际的限制，使其不能成为一个“万能黑盒”，具体来说，你必须在某种程度上对最终的解密步骤进行保护，而且在实践中它往往与 MPC 有相似的限制。所以我确实期待它，但它并不是万能药。当然，更好的 FHE 几乎肯定会转化为更好的见证加密和/或混淆，如果我们可以在实践中使其变得可行的话。

28、于以太坊的未来，你最大的担忧或关注点是什么？

Vitalik：加密技术停滞不前的风险、隐私和开放互联网基础设施等都将成为失败的原因，而 21 世纪的重大技术政治问题将由未来十年人工智能领域发生的任何事情决定。

29、你观察到的使加密行业创始人/领导者成功的特质是什么？

Vitalik：这取决于你如何定义"成功"！

我个人最尊敬的是那些有明确的个人价值观，以及对他们参与加密货币领域的目标有深刻理解的人。这种理解明显深入到超出了他们可能从加密货币会议中继承的陈词滥调。
