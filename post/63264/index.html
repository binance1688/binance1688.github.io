关键洞察

  • Neutron 和 Stride 成为第一批由 Cosmos Hub 的复制安全机制保护的区块链，共享其验证者集合。这标志着 Atom 经济区的开始。

  • IBC 转入和转出分别下降了 27.3% 和 22.0% 。Cosmos Hub 的角色正从互操作性枢纽转变为安全提供者。

  • 尽管 ATOM 价格和 Cosmos Hub 活动量下降，但其收入环比增长了 2.4% 。

  • 每日活跃地址和每日交易量分别环比下降了 19.8% 和 6.3% 。

  • Atom Accelerator DAO 选定了 Blockworks Research、Binary Builders 和 RMIT University 来提出对 ATOM 代币经济的全面改革 - Tokenomics RFP。这些团队提交了五篇论坛帖子，详细说明了提案。

Cosmos Hub 概述

Cosmos Hub 是一个专注于互操作性和安全性的独立应用链。它是一个采用权益证明（PoS）机制的主权区块链，使用基于账户的会计模型，没有内置的智能合约功能。ATOM 是 Cosmos Hub 的原生代币。Cosmos Hub 引领了 Cosmos SDK、CometBFT（Tendermint）、ABCI 和 IBC 等技术的发展，这些技术后来被许多链间网络所采用。链间网络由互操作协议 IBC 连接的主权网络组成，也被称为 Cosmos 生态系统。

Atom 经济区（AEZ）是一个与 ATOM 代币对齐的生态系统集合，包括与 ATOM 代币有各种关联和整合的网络。租用 Cosmos Hub 验证者安全服务的网络，即消费者链，是 AEZ 的一部分。由于缺乏可编程性，Cosmos Hub 的安全不能像 EigenLayer 的再质押模型那样随意共享，但可以通过治理进行单独启用。复制安全（以前称为链间安全）指的是通过治理投票授权，与另一条链共享完整的 Cosmos Hub 验证者集。

Cosmos Hub 于 2019 年推出，利用了由链间基金会和 Ignite（也称为 Tendermint）开发的技术。包括链间基金会、Binary Builders、Atom Accelerator DAO、Informal Systems、Strangelove 等在内的多个团队继续支持 Cosmos Hub 和 Cosmos 技术栈的发展。链间网络拥有 100 多个独立网络，每个网络都有其独特的支持实体。

关键指标

财务分析

ATOM 价格连续第二个季度下跌，环比下降 21.0% 至 7.11 美元。这落后于整个加密货币市场，后者同期下跌了 9.2% 。ATOM 的总市值排名从第 23 位下降到第 24 位。截至第三季度末，在网络代币中，ATOM 的市值排名第 10 。

ATOM 是 Cosmos Hub 的原生代币，兼容 ICS-20 标准，允许用户通过 IBC 协议在链间转移 ATOM。ATOM 的总供应量约为 3.56 亿，流通供应量约为 2.93 亿。这种资产具有以下功能：

  • Cosmos 上的交易需要用 ATOM 支付交易费。

  • 持有者可以质押 ATOM 以运行验证者，保护网络并获得奖励。

  • 持有者可以将 ATOM 委托给现有验证者，以帮助保护网络并赚取验证者的部分奖励。

  • 所有质押和委托的 ATOM 可以用来参与网络的治理过程。

收入以总交易费用计量。Cosmos Hub 的收入环比增长了 2.4% 。7 月 31 日收入出现了飙升，当天的平均每日交易费上升到 0.22 美元，远高于季度平均水平的 0.04 美元。

ATOM 代币经济

第三季度，Cosmos Hub 的实际收益率平均约为 4.5% ，季度末为 4.65% 。ATOM 的高通胀率约为 14% ，几乎抵消了给予验证者和委托者的约 19% 的奖励率。流通供应主要受验证者和委托者的质押奖励影响。由于奖励率和通胀率都很高，这对不参与质押的 ATOM 持有者造成了沉重的负担。

截至第三季度，ATOM 代币存在两种通缩压力：

  1. 治理销毁 - Cosmos Hub 具有链上治理机制，ATOM 代币持有者可以投票和提出提案，如改变共识参数和社区池资金分配。提交提案需要任何代币持有者最少存入 250 个 ATOM。如果提案被否决，存入的 ATOM 将被销毁。

  2. 削减 - 如果交易被重复签名或验证者长时间离线，验证者节点的 ATOM 奖励可能被削减。被削减的奖励随后被销毁。

被拒绝的 ATOM 2.0 提案重新构想了 ATOM 的代币经济。该提案的目标是在几年内大幅减少 ATOM 的发行量，然后完全取消通胀。此外，它旨在通过流动性质押提高资本效率，并将 ATOM 定位为链间储备资产。最终，这些行动将推动 ATOM 代币的价值积累。

尽管最初的提案因涉及范围广泛和快速引入而被拒绝，但各个组成部分仍在作为单独的提案进行探索。值得注意的是，两项提案（增加国库规模和改善 ATOM 资本效率的流动性质押功能）在 ATOM 2.0 中首次引入后，在后续提案中均获通过。社区仍在讨论其他 ATOM 2.0 功能，包括无需许可的共享安全模型和链间技术（如 IBC、CosmWasm 等）的社会协调（即治理）中心。

此后，社区未能通过额外的与代币经济相关的提案，如根据交易费用量销毁 ATOM 或激励持有者质押所持代币。开发者还在讨论改进 Cosmos Hub 的费用机制以减少垃圾交易的方案。一种方案是使用类似于 EIP-1559 的模型，其中支付给验证者的部分费用被销毁。

第三季度，Atom Accelerator DAO（AADAO）选定了 Blockworks Research、Binary Builders 和 RMIT University，提出了一项新的 ATOM 代币经济改革——Tokenomics RFP。这五个提案团队探讨了治理、流动性质押、国库、公共资金、Atom 经济区内的关系以及 ATOM 的发行。

Blockworks Research 提出了一项财政政策变更。该提案旨在从当前的静态 10% 社区池税过渡到更多样化的税收方式。这种新方法包括以下几个部分：

  • 投票权力税：引入一项税收，以阻止向投票权力较高的验证者委托，以解决权力集中问题。

  • 动态社区池税：提议将社区池税从固定的通胀百分比更改为随时间递减的时间表，直到达到 5% 的基本税率。

网络分析

每日活跃地址和每日交易量分别环比下降了 19.8% 和 6.3% 。Cosmos Hub 继续向一个新类型的多链生态系统的中心转变，其中 Cosmos Hub 作为安全提供者而不仅仅是互操作性枢纽。随着转型的进行，Cosmos Hub 主网上的活动可能会减少，因为使用量转移到其「子链」。这在 Atom 经济区分析部分中有进一步探讨。

历史上，活跃地址和交易量紧密相关，第三季度依然如此。持续的耦合表明， 8 月份交易活动的激增是由于新增用户上线并与网络互动，而不仅仅是现有用户更频繁地进行交易。

质押和去中心化

Cosmos Hub 的 67.6% 的质押率虽然环比下降了 3.6% ，但仍是其他 L1 网络中较高的质押率之一。质押者包括验证者和委托者。符合系统要求的用户可以通过质押 ATOM 运行验证者，以保护网络并获得奖励。代币持有者可以将至少一个 ATOM 委托给现有验证者，以获得相关验证者奖励的一部分。委托者根据质押的代币数量获得奖励，尽管他们需要支付由个别验证者设定的佣金。第三季度，治理提案 826 将验证者的最低佣金设定为 5% 。

只有前 180 名验证者（根据自行质押和委托的 ATOM 代币总量排名）能获得奖励。在这些验证者中，前八名控制了总质押量的 33% ，使得 Cosmos Hub 的中本聪系数为 8 。

治理和 Grants

Cosmos Hub 采用线下和线上治理流程的结合。网络改进提案和参数更改通过线下方式设计和讨论，通常在 Cosmos 治理论坛进行。任何人都可以提出网络或财政管理提案，并参与是否追求该提案的讨论。用户还可以将任何网络变更请求作为官方提案提交给 Cosmos Hub 的线上治理系统。一旦进入线上治理系统，ATOM 持有者就会投票决定是否批准（并最终执行）提议的更改。

第三季度有 24 个治理提案，其中 10 个获得通过。值得注意的通过提案包括 V1 1、V1 2 和 V1 3 软件升级，以及扩大 AADAO 对 ATOM 协议拥有的流动性（POL）的管理。AADAO 透明度报告 4 详细介绍了 DAO 分配的财务和赠款。第三季度，AADAO 的第 4 批和第 5 批赠款共计 109.7 万美元分发给了获奖者。这些赠款资助的项目包括赞助 Cosmoverse、DoraHacks 的黑客马拉松等。之后，宣布了第 6 批，并在 10 月分发了赠款。这些轮次使 AADAO 发放的赠款总数达到 27 个，总价值 250 万美元。

Binary Builders 的「ATOM 对齐金库」旨在增加 Cosmos Hub 治理和投票的去中心化。随着验证者运营成本的增加和对网络去中心化的担忧，Binary Builders 提议根据他们的投票权向验证者分配更多的股份。该提案旨在支持低端验证者并进一步去中心化网络。它还探讨了如何在其他 AEZ 网络上使用 ATOM 执行投票权（在 Atom 经济区分析部分有更多细节）。

Blockworks Research 的提案对治理有影响，除了财政政策变化外，在 ATOM 代币经济部分的财务分析中也有概述。

投票权税旨在通过对高投票权的验证者进行委托来解决股权集中问题。

随着流动性抵押需求的增长，将对流动性抵押者征收动态流动性抵押税。其目的是在为 Cosmos Hub 产生税收的同时，找到流动性抵押市场渗透的自然平衡。

取消 25% 的流动性抵押上限将允许更多参与者从流动性抵押中受益。Cosmos Hub 本身不支持流动性抵押，但它可以通过 AEZ 中的 Stride 网络获得。这在 Atom 经济区分析中有进一步探讨。

立方委托（Cubic Delegation ）有可能解决有关线上治理的中心化问题，同时确保仍能满足法定人数。

IBC

Cosmos Hub 拥有 56 个 IBC 对等体和 269 个 IBC 通道。「对等体」指的是与 Cosmos Hub 建立了 IBC 连接的不同交易对手方；「通道」指的是可用于连接对等体的总路由数量。独特发送者、独特接收者和总转移量（包括进出站）的活动都出现了下降。进站独特接收者环比下降了 40.6% ，为最大的变化。所有这些 IBC 指标都达到了年度低点。

大多数情况下，进出站转移是紧密相关的。有一些例外，比如 2023 年 5 月的大量进站高峰。在第三季度，有几次每日出站转移量超过进站转移量 2-3 倍的高峰。

Atom Economic Zone

Atom 经济区（AEZ）是一个与 ATOM 代币对齐的生态系统集合，包括与 ATOM 代币有各种关联和整合的网络。AEZ 中存在多个网络，具有不同的能力。Stride 和 Neutron 获得复制安全，Osmosis 允许使用 ATOM 支付费用。其他网络（如 Kujira 或 Agoric）依赖 ATOM 提供大部分流动性。

复制安全（以前称为链间安全），是一种共享安全形式，是 Cosmos Hub 社区和其他链间网络最关注的 AEZ 策略。共享安全指的是允许一个网络的验证者使用该链上的股份参与另一个网络的共识。这种设置将允许市值较小的网络从市值较大的网络「租用」安全。截至第三季度，Neutron 和 Stride 是唯一使用复制安全的链。

Binary Builders 的「ATOM 对齐财库」（AAT）旨在使 ATOM 成为 AEZ 内的默认货币，并在 Cosmos Hub 和消费者链之间对齐质押和治理。该团队打算使用 AAT 资金进行开发工作，并在 AEZ 内提供流动性。

Neutron

提案 792 的通过使 Neutron 成为第一个通过复制安全利用 Cosmos Hub 验证者集合和安全的链。Neutron 作为 Cosmos Hub 的 CosmWasm 扩展，有效地成为了 Cosmos Hub 的 DeFi 中心。

Neutron 仍然是一个相对较新的网络，仅在今年推出。提案 72 为 Neutron 的开发分配了 Cosmos Hub 资金，提案 819 将这些资金直接重新分配给了 Neutron 基金会。

Stride

Stride 在 Neutron 之后不久实施了复制安全。这个网络为各种链间网络和资产提供流动性抵押。由于 ATOM 是 Stride 活动的核心部分，它成为早期采用 Cosmos Hub 安全的明显选择。流动性抵押是一个大市场，在各种其他生态系统中都有叙述。通过复制安全，Cosmos Hub 在 AEZ 中拥有流动性抵押解决方案，因为其效用是通过治理对齐的。

在 AEZ 中的其他网络也同样重视通过 Stride 抵押的 ATOM，即 stATOM。Cosmos Hub 提案 805 （对提案 800 的修正 —— 作为服务的流动性）将让 AADAO 在 Astroport Neutron 上部署 450, 000 ATOM 到一个 ATOM/stATOM 恒定产品池中。在 Osmosis 上，ATOM/stATOM 的 TVL 超过 2000 万美元。

治理提案 817 寻求 Cosmos Hub 社区批准选择一个 5 人咨询委员会，负责通过 Stride 代表团流程评估和选择宿主链验证者。

Osmosis

Osmosis AMM DEX 自诞生之初就与 Cosmos Hub 和 ATOM 深度整合。截至 2023 年第三季度，Osmosis 上的第一个也是 TVL 最大的池是 ATOM/OSMO 池。大约 75% 的 Osmosis TVL 来自 ATOM 池（主要是 ATOM/OSMO 和 ATOM/stATOM）。

Osmosis 在 AEZ 中的「ATOM 对齐」和角色远超流动性对。ATOM 是 DEX 上的另一种燃料代币。虽然 ATOM 在某些情况下被用于燃料，但由于活动模式，许多情况被怀疑是由机器人产生的。

Cosmos Hub 上的提案 810 将确定 Osmosis DEX 用于灵活的燃料费支付。该提案将允许在 Cosmos Hub 上的交易费用用任何代币支付，利用 Osmosis 将替代代币换成 ATOM.

其他网络

更多的链可能很快以不同的能力加入 AEZ。Composable Finance 提交了一个提案加入 AEZ 并从 Cosmos Hub 获得复制安全，就像 Stride 和 Neutron 一样。Neutron 作为可编程性和 DeFi 中心，Stride 作为流动性抵押解决方案，Composable 提议成为一个 rollup 结算中心，因为它连接到其他多链生态系统。

虽然复制安全的定义很容易界定，但「ATOM 对齐」更主观。一些社区成员认为，由于其生态系统与 ATOM 交织在一起，如 Agoric 和 Kujira 等网络已经是 AEZ 的一部分。换句话说，ATOM 代币可以为 Cosmos Hub 提供大量流动性，促进大量链间活动。

总结

自成立以来，Cosmos Hub 的角色是互操作性中心和更广泛的 Cosmos 生态系统的焦点。在第三季度，Cosmos Hub 向成为安全提供者的角色迈出了重要一步。通过复制安全，Neutron 和 Stride 成为了第一批由 Cosmos Hub 保护的链。Cosmos Hub 作为模块化安全提供者的新定位增加了它作为更广泛 Cosmos 生态系统文化中心的角色，随着链加入 Atom 经济区。

Cosmos 社区已经提交了提案，包括更多的链，如 Composable Finance。此外，其他链间网络正在探索其他形式的「ATOM 对齐」，如使用 ATOM 支付费用，成为 Atom 经济区的一部分。

在被拒绝的 ATOM 2.0 提案之后，该提案以各种方式重新构想了 ATOM 代币经济，大多数概念继续作为单独的小提案进行探索。治理、去中心化、流动性抵押、财政管理和代币经济是第三季度探讨的主题。AADAO 一直在向各团队提供 Grant 以解决这些主题，包括 Tokenomics RFP。
