▌彭博分析师：贝莱德在特拉华州申请注册iShares XRP Trust为假消息

彭博分析师Eric Balchunas在X平台发文称，已向贝莱德确认，贝莱德在特拉华州申请注册iShares XRP Trust为假消息，一些不怀好意的人使用 BlackRock高管名称添加了内容。

▌ 美财政部拟议的加密税收提案仍有修订空间

据Coindesk消息，美国国税局和财政部官员周一在听证会上听取了业界关于加密税收提案可能造成的混乱信息及影响，并提出了一些问题，暗示其提案仍有修订空间。联邦官员在听证会上的提问显示他们对提议规则下去中心化平台的潜在负担、是否应报告稳定币交易以及如何识别非金融资产表现出了兴趣。 

据悉，加密货币税收规则于8月提出，公众意见征询期于周一结束，该机构收到了超过124,000条关于税收提案的评论，提案的最终版本可能还需要几个月的时间，并且还需回应加密行业的一些质疑。

▌ APEC财长将在旧金山会议上分享对加密货币的看法

为期一周的亚太经济合作组织（APEC）峰会于11月11日在旧金山开幕，美国财政部长耶伦 (Janet Yellen)在11月13日财长会议的开幕致辞中表示，他们将讨论“面向长期的优先领域”，重点强调可持续性。他们还将举行一场专门讨论供应方经济学的会议和另一场专门讨论数字资产的会议。耶伦特别提到了无担保的加密资产、稳定币和央行数字货币。 

耶伦表示：“分享见解并与私营部门合作，使我们能够加深对政策制定者可以用来促进负责任地开发和使用数字资产的工具的集体理解”。 

她补充称：我期待听到您对数字资产和区块链技术在我们各自的金融体系中可以发挥的长期作用的看法，以及当局计划如何对其开发和使用进行监管。” 

据悉，APEC由亚洲、北美和南美的21个太平洋地区“经济体”组成，成员资格对经济体而不是国家开放，Ripple是本次峰会的主要（钻石级）赞助商。

▌ 加密行业反对美财政部拟议的经纪商报税规定，逾12万封意见信已提交

加密货币行业正在坚决反对美国财政部拟议的税务报告规定，财政部与国税局在8月份发布了一系列拟议法规，要求加密货币经纪商受到与股票和债券等更传统投资经纪人类似的待遇。自那时起，已有超过124,000封意见信提交至该机构，其中仅过去两天就收到了2,000封信件。 

区块链协会高级顾问Marisa Copel表示，该提案过于宽泛，特别是针对去中心化金融，应仅限于中心化实体。 

一些人士以宪法问题为由称其为去中心化金融的障碍。DeFi教育基金在本月早些时候的信中表示，该提案将经纪商的定义延伸到了宪法限制之外，财政部实质上是在创建一种名为“数字资产中间商”的新型经纪商，中间商术语可以“将区块链技术堆栈中的每个参与者视为经纪商”。

▌Aave社区提议在v3以太坊/Polygon上重新启用CRV借贷

Aave社区Gauntlet提案建议在v3以太坊/Polygon上重新启用CRV借贷，并建议对CRV参数进行一些其他更改。 其中包括在以太坊v3上，建议供应上限从5100万降低至750万，借款上限从770万降低至500万，债务上限从500万美元降低到100万美元；重新激活CRV借贷。 

Polygon v3上，借款上限从90万降低到30万，重新激活CRV借贷。Gauntlet认为，鉴于目前的使用情况，降低上限可能有助于平衡未来需求的空间，同时减轻与尾部流动性下降相关的风险。

▌SSV.Network将和P2P.org合作开发DVT Stake API，以实现去中心化以太坊质押

SSV.Network和P2P.org宣布计划合作开发DVT Stake API，以实现去中心化以太坊质押，简化托管商流程，并通过SSV协议增强机构质押安全性。 

该团队表示，DVT（即分布式验证器技术）通过在节点之间分配以太坊验证器职责来提高安全性。随着SSV.Network即将推出主网，此次合作将加速 DVT Stake API 的开发，促进以太坊质押应用程序的部署和去中心化。

加密货币

▌Bianco Research总裁：现货比特币ETF市场反应将与Facebook IPO轨迹类似

Bianco Research, LLC总裁兼宏观策略师Jim Bianco表示，现货比特币ETF的市场反应将与Facebook在2021年IPO时类似。在Real Vision最近的一次采访中，Bianco回顾了Facebook的IPO，这是近代历史上最受期待的IPO之一，并将其与比特币的潜在市场行为进行了比较。 

Bianco表示，Facebook IPO期间，该公司股价最初定价为28美元，但随后大幅下跌至11美元，与市场预期相反。Bianco表示，在美国批准现货比特币ETF后，比特币可能会经历类似的模式。他担心比特币最初可能会在批准后几天内出现反弹，但随后价格可能会停滞甚至下跌一段时间。

▌TeraWulf第三季度产出994枚自挖比特币，较Q2增长9.5%

TeraWulf Inc.（纳斯达克股票代码：WULF）公布了2023财年第三季度未经审计的中期财务业绩，2023 年第三季度该公司产出994枚自挖比特币，较第二季度增长9.5%。2023 年第三季度收入为1900万美元，比第二季度增长22.6%，总安装算力为5.5EH/s。2023 年第三季度非GAAP调整后EBITDA为900万美元，比第二季度增长18.5%。 

该公司表示，预计将利用2023年第三季度产生的现金偿还约700万美元的债务，其中550万美元已于2023年10月偿还，并以19美元/TH 的价格订购了约 2.8 EH/s 的BITMAIN S19j XP矿机，将于2023年12月交付。

▌Pantera Capital：受FTX倒闭重大影响的投资组合不到5%

Pantera Capital在X平台就FTX倒闭一周年发布反思和教训总结，内容如下： 

在FTX和Alameda财务状况不佳的消息传出后，我们组建了一个作战室来评估对投资组合的影响程度，目标是：1）识别所有潜在风险；2）为高风险团队提供帮助。值得注意的是，我们已经在这个行业工作了十年，这项工作并不新鲜，我们已经启动了应对类似压力的作战室，例如2018年加密寒冬、三箭破产、LUNA崩溃以及最近的银行危机。在确定所有潜在风险（托管、交易对手、投资等）后，我们与受影响的团队合作，以减轻进一步的风险并制定前进计划。幸运的是，我们投资组合团队中只有不到5%受到重大影响，部分原因是我们定期向创始人强调的主动风险管理实践。 

以下是在加密领域进行团队建设的一些想法：1.在组织内建立一个流程，以最大限度地减少任何资产在任何交易所或第三方的停留时间。 

2.通过对所有资产转移实施多签流程，避免任何资产转移的单点控制。通过确保与尽可能多的托管商、交易所和银行打交道，使链上和链下交易对手风险多元化。 

3.通过尽可能保持财务资产的流动性和现金，避免公司财务资产出现任何不必要的本金风险。采取措施确保银行账户符合FDIC保险限额。 

4.将资产从混合综合钱包（例如交易所或项目资金与其他资金混合的任何媒介）转移到链上隔离的钱包（例如托管、自托管或资金与其他第三方资金隔离的任何地方）。

▌ 中金：出口改善的趋势在未来几个月仍将持续

中金研报表示，2023年1-10月以美元计价出口同比-5.6%，但是2023年10月的三个月移动平均同比为-7.2%，仍然较9月的-9.8%改善。我们认为，这种改善的趋势在未来几个月仍将持续。结构性来看，“去中心化”对出口不利，但这个是个逐步过程，未必立即完全显现，短期内贸易再配置效应缓解了“去中心化”对出口的影响；周期性来看，美国补库存将对我国出口带来支撑。(金十数据APP)
