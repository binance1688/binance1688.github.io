以太坊凭借生态的繁荣度在过去公链之争的阶段取得了领先地位。随后L2的发展为市场注入新的活力，但同时不同L2之间的生态之争也进入白热化阶段。

以“四大天王“为首的L2，在吸引不同应用丰富自身生态的同时，也推出不同的 CDK 来吸引项目方构建专属的L2或应用链。

CDK（链开发套件，对照了 SDK 的命名法） 是一个开源框架，旨在辅助开发者快速部署基于以太坊的 L2。它提供一种模块化的环境，使开发人员能够根据特定需求创建特定应用链，或者将现有的L1平滑过渡为 L2 。详细了解可看之前文章《L2之战正酣，Polygon 祭出 CDK 意味着什么？》

就在一个半月前，Polygon CDK 横空出世，截止到现在，已经吸引 13 个项目方来构建 zk L2。其中不乏知名的项目方，比如 Immutable、Manta、Dogechain。更有Web2爆款游戏冒险岛和地下城与勇士的开发公司 Nexon。

Odaily星球日报现将 13 个项目进行盘点介绍。

Polygon CDK 项目盘点

Immutable zkEVM

Immutable zkEVM是为游戏开发者提供 EVM 兼容性、低成本、大规模和以太坊安全性。相较于 Immutable X，开发者可以构建更多的游戏类型，为玩家提供更好的体验。

Immutable 和 Polygon 的生态系统为游戏开发者提供了一个较大的用户基础和资金支持，共计拥有超过一百万玩家和 20 亿美元的投资。这为游戏的发展和收益增长提供了良好的基础。

Gnosis Pay

Gnosis Pay是一个去中心化支付网络和自助储蓄卡。Gnosis Pay 提供了全套银行和支付许可证，并符合欧盟法规。Gnosis Pay 通过 Visa 认证，与全球超过 8000 万家商户进行合作。

Gnosis Pay 提供的基础设施支持众多区块链、代币、DeFi 协议以及入口和出口提供商 Gnosis Pay 建立在去中心化托管协议和集体资产管理平台 Safe 之上，存储超过 100 亿美元。

Gnosis Pay 的目标是为应用程序开发人员提供稳定可靠的支付和存储解决方案，在 Polygon CDK 支持的 zk Layer 2 上构建应用程序。

Wirex

Wirex创建于 2014 年，是一个全球性的数字支付平台和受监管的机构，其核心目标是让所有人都能使用数字资产。2015 年，该公司开发了第一张支持加密的支付卡，让用户可以选择在现实生活中使用加密货币和传统货币。

Aavegotchi

Aavegotchis 是存在于 Polygon 区块链上的数字收藏品(NFT)。每 Aavegotchi 都可以定制各种可穿戴设备，例如帽子、眼镜和其他配件，这些配件可以作为 NFT 进行购买、出售和交易。可穿戴设备具有独特的属性和特征，这些属性和特征会影响 Aavegotchi 的外观、稀有性和游戏能力。

Aavegotchis 即将迁移到由 Polygon CDK 搭建的 Gotchichain。

CapX

Capx是一种专注于通证分发和交易的区块链解决方案。它为项目社区提供通证分发、投资者分配和交易功能。使用 Capx，项目可以激励早期用户和采用者参与平台，并提供无缝的通证兑换机制。Capx 还允许项目自定义通证参数和分发策略。通过 Capx 的通证分发基础设施，项目可以提高可见性，吸引更多的投资者和用户，为长期成功奠定基础。

Palm Network

Palm Network是一个与以太坊兼容的侧链，旨在作为建立可扩展和可持续的 NFT 生态系统的基础。之后 Palm Network 将通过 Polygon CDK 构建 zk Layer 2 ，使用户更容易铸造、交易和收集 NFT。

IDEX

IDEX是一家高性去中心化交易所，通过离链交易引擎和链上结算与托管相结合，提供了以太坊基础层的安全性和低成本的交易。用户可以享受无手续费的交易、低延迟和即时执行，并拥有自主控制和透明的资金托管。

IDEX 通过 Polygon CDK 构建 Xchain，据官网介绍，Xchain 可以辅助 IDEX 提供无 Gas 交易，并提高产品性能，从而实现与其他通过 Polygon CDK 的L2深度交互。

Astar

Astar Network（原 Plasm）是 Polkadot 上的多链多虚拟机 DApp 中心，原生支持以太坊虚拟机（EVM）、WebAssembly、Layer 2 解决方案和 DApp Staking。

Astar 的新L2采用了 Gelato 的易用性、Polygon zkEVM 的可扩展性和以太坊的安全性。它具备本地账户抽象、智能合约自动化与离链数据访问、高吞吐量和低手续费等特点。Astar 的 zkEVM 将提供弹性可靠的基础设施和本地账户抽象，为日本企业进入Web3并推动大规模采用提供门户。

Dogechain

Dogechain 是从 Polygon 分叉而来、由 Dogecoin 爱好者和社区 OG 成员创建。他们的愿景是将 NFT、游戏和 DeFi 等加密应用程序带入狗狗币社区。Dogechain 允许用户将 DOGE 代币封装到智能合约中，然后与 Dogechain 上的项目进行交互，用户可以使用 DOGE 代币来支付 gas 费用。

目前 Dogechain 通过 Polygon CDK 建立 zk Layer 2 。

Canto

Canto是一个兼容 EVM 的公链。让传统公司的系统变得可访问、透明、去中心化和自由。

Canto 将使用 Polygon CDK 建立二层网络，通过以太坊的安全性和 Polygon 生态繁荣度，为传统公司的数字化带来更丰富的体验。

Manta Network

Manta Network是利用零知识证明构建的公链，Manta Pacific 是L2生态系统，提供较低成本和良好的体验。并借助 Celestia 的数据可用性实现模块化以提供低 Gas 费用。

Manta Pacific 后将通过 Polygon CDK 构建L2，并与 Polygon 生态深度结合。

Nubank

Nubank成立于 2013 年，旨在为人们的日常生活提供简化的金融服务，重塑金融行业。Nubank 为巴西、墨西哥和哥伦比亚的 8500 万客户提供服务。借助专有技术和创新实践，Nubank 致力于为个人和中小企业提供简单、直观、方便、低成本、赋权和人性化的金融解决方案和体验。

Nexon

Nexon成立于 1994 年，是互动娱乐软件领域的先驱，在全球 190 个国家/地区运营 80 多款在线游戏。通过专注于质量，我们开发了备受欢迎的知名 IP 游戏系列，包括冒险岛和地下城与勇士等，并在东京证券交易所上市。
