原文作者：Zen，PANews

自 11 月以来，RPG 链游 Illuvium 的 ILV 代币价格一度实现翻倍，从 0.77 美元最高上涨至 11 月 12 日的 1.58 美元（现价约 1.44 美元）。在连续近两周的上涨过程中，Illuvium 官推于 11 月 8 日发布公告表示计划于 11 月 28 日在 Epic Games Store 上架后，其代币价格从 1.17 美元左右继续大幅上涨。

根据 Epic 的 2022 年报数据，Epic 游戏商城现有超过 2.3 亿 PC 用户。同样是传统游戏平台，比起龙头大哥 Steam 对Web3游戏的拒之千里，想要后来居上的 Epic Games 就显得开明得多。今年 3 月，Epic Games 的总经理 Steve Allison 表示，其游戏商城将在 2024 年初之前上架近 20 款加密游戏，但官方不会参与任何此类游戏的开发或发行。

在欢迎Web3游戏入驻的同时，Epic Games 商城也对这些发布使用区块链技术的产品做出多项要求和规定，包括禁止出现任何区块链、NFT 或加密货币商城的链接，不得促进赌博行为，不能向中国和韩国的用户提供产品服务，以及 Epic Games 有权基于市场行为（如为了应对可能对玩家产生负面影响的问题）随时从商城中移除使用区块链技术的产品等。

Epic 能够提供的曝光度与玩家数量，远非当前任何Web3游戏平台可比，这为迫切想要进入主流玩家视野的Web3游戏们提供了机遇。除了 Illuvium，还有 NFT 卡牌游戏 Parallel 等Web3游戏也计划或已经入驻 Epic Games Store。据 PANews 统计，目前 Epic Games 商城内已列出 77 款游戏，其中有 41 款游戏即将推出，已上架的游戏数量早已超过 20 个的“小目标”。按照游戏人气和社区热度，PANews 从即将在 Epic 游戏商城上架的游戏中，挑选出了 15 款Web3游戏进行简要介绍。

Avalon

Avalon 是一款使用虚幻引擎 5 构建的 MMORPG，基于 Polygon、WAX、BNB 区块链，将于 2024 年上线。Avalon 只支持 Windows 系统，其开发商与发行商均为 Danu Games。该游戏以奇幻的中世纪王国为背景，玩家利用自己的角色探索充满危险和神秘的世界，在此过程中，玩家通过购买、租赁或获取资源以及与土地所有者合作获取和建造游戏内房地产来获得奖励。

Forgotten Runiverse

Forgotten Runiverse 是一款奇幻题材像素风的 MMORPG，基于 Polygon 网络，将于 2024 年第二季度上线。该游戏支持 Windows 和 MacOS 系统，开发商与发行商均为 Bisonic。游戏目前处于预测试状态，允许玩家尝试核心游戏机制，包括战斗、采集、制作、任务和升级。游戏中的任何物品都可以铸造到区块链上，并用来展示和交易。

Champions Ascension

Champions Ascension 是一款角色扮演游戏，基于 Polygon 网络，预计将于今年 11 月推出。该游戏分为 PvE、自由对战和 1 v1、 3 v3等 PvP 模式，仅支持 Windows 系统，开发商与发行商均为 Plai Labs。游戏无需钱包即可体验，开发商计划将 Grand Bazar（市场）等第三方市场购买的 NFT 集成至游戏中。

SHRAPNEL

SHRAPNEL 是一款第一人称射击类游戏(FPS)，基于 Avalanche 网络，抢先体验将于 2023 年 12 月上线。该游戏仅支持 Windows 系统，开发商为 SHRAPNEL，发行商是 Neon Machine。玩家无需拥有 NFT 或加密货币，即可体验完整的 Shrapnel 游戏。皮肤、徽章、地图创建内容等游戏内内容可在其游戏市场购买和出售。

Illuvium

Illuvium 是一款使用虚幻引擎 5 构建的开放世界角色扮演游戏，基于 Immutable X 网络，预计将于 11 月 28 日上线。该游戏支持 Windows 和 MacOS 系统，开发商与发行商均为 Illuvium Labs。在游戏中，玩家的目标是探索 Illuvium 星球，尽可能多地收集强大神兽（Illuvial），游戏分为竞技场模式、主世界模式和 Zero 模式。

MetalCore

MetalCore 是一款开放世界的机甲类战斗游戏，采用 Immutable zkEVM，将于 2023 年 11 月推出。该游戏仅支持 Windows 系统，开发商为 Studio 369 ，发行商是 MetalCore United。游戏中，玩家将在矿产丰富的外星球上作战，参与领土争夺战。制造车辆、制作装备、招募飞行员、开发土地、销售和交易装备等等。

Mars 4 

MARS 4 是一款P2E生存游戏，基于以太坊、BNB 及 Polygon 网络，将于 2024 年第一季度上线。该游戏支持 Windows 和 MacOS 系统，开发商与发行商均为 Mars 4 。MARS 4 引入了由区块链和原生加密货币 $MARS 4 支持的 Land Plot NFT，玩家在恶劣的火星环境中挣扎，建造建筑物，收集资源并赚取奖励作为回报。

RIO-X

RIO-X 是一款大逃杀游戏，将于 2024 年第二季度上线。该游戏仅支持 Windows 系统，开发商与发行商均为 First Phoenix Studio。RIO-X 允许玩家无需连接钱包即可玩游戏，玩家还可以通过与区块链的集成，直接在游戏中使用他们的 NFT 资产，并通过将 RIOCOIN 存入位于安全区域的 ATM 来赚取奖励。

Citizen Conflict

Citizen Conflict 是一款射击类游戏，基于 BNB 网络，将于今年上线。该游戏仅支持 Windows 系统，开发商与发行商均为 QORPO Game Studio。游戏中的每一个虚拟资产（VA），例如英雄、装备、皮肤等，都可以被铸造为 NFT，游戏原生 NFT 市场与 QORPO WORLD 平台集成。不喜欢Web3的玩家可以自由地与常规虚拟资产 (VA) 进行交互，就像在任何其他 Web2 游戏中一样。

Defimons

Defimons 是一款收集和冒险类游戏，基于 Avalanche 网络，游戏上线的具体时间待定。该游戏支持 Windows 和 MacOS 系统，开发商与发行商均为 Mon Studios。玩家在游戏中可以捕捉恶魔，与其他驯兽师对战，以及种植农作物、制作工艺物品等。游戏大部分可在没有 NFT 的情况下进行，只有在交易物品时必须使用 NFT 或区块链进行交互。

Battle Leet

Battle Leet 是一款3D PvP 格斗游戏，基于 BNB 区块链，未来将支持 Base 链，游戏上线的具体时间待定。Avalon 只支持 Windows 系统，支持手柄操作，其开发商与发行商均为 GMR Center 玩家可以使用“Battle Leet”或“Non-Fungible Leet”进行战斗，通过赢得回合胜利来赚取 $GMR 游戏内货币和其他奖励。

Angelic: Dark Symphony

Angelic 是一款黑暗科幻风格的叙事策略角色扮演游戏，其 Angelverse 代币基于 Solana 网络运行，游戏上线的具体时间待定。该游戏仅支持 Windows 系统，开发商与发行商均为 Metaverse Game Studios。玩家玩游戏时无需对钱包、NFT 等进行交互，但可以根据需要将游戏内资产转换为链上物品。

Moonray

Moonray 是一款快节奏的多人竞技对战游戏，改编自《Moonray》图画小说，基于以太坊网络，游戏上线的具体时间待定。该游戏仅支持 Windows 系统，开发商与发行商均为 Moonray PBC。玩家无需以任何方式与区块链交互即可玩或享受应用内购买。游戏内的数字物品将在幕后表示为 NFT，只有玩家在外部交易物品时才需要与区块链进行交互。

HELIX

HELIX 是一款开放世界游戏，基于以太坊网络，游戏上线的具体时间待定。该游戏仅支持 Windows 系统，开发商与发行商均为 Hypersonic Laboratories。HELIX 将为玩家提供多种游戏模式。我们最突出的一款是《Parallel City Online》，这是一种开放世界的角色扮演体验。游戏免费提供创建游戏、角色扮演服务器、社交休息室、社区聚会、艺术项目、技术演示等功能和服务。

Superior

Superior 是一款轻度 Rogue 类型的第三人称射击游戏，游戏上线的具体时间待定。Superior 仅支持 Windows 系统，开发商为 Drifter Entertainment，发行商是 Gala Games。在游戏中，玩家每次开始游戏时，都会以力量不足、枪械落后的情况开局，追捕邪恶的超级英雄并夺取他们的力量。
