自从 ChatGPT 推出以来，我们一直对人工智能及其潜在用途有些着迷。自该版本发布以来不到一年，我们已经看到了几个用例的出现，还有许多其他用例即将出现。

我们感兴趣的是人工智能将如何影响金融。随着训练和计算成本的降低，预计人工智能代理的在线人数将超过人类（如果还没有的话）。这些代理将消耗、生成和交换难以想象的大量信息。这给身份管理至关重要的金融等领域带来了很多机会和焦虑。

那么，让我们更进一步。人工智能（特别是类人人工智能代理等问题）将如何影响区块链网络？区块链是否为金融科技中人工智能驱动的代理提供了新的可能性？

人工智能驱动的金融机构的现状

金融和金融科技领域的人工智能信徒认为，必须有一种方法使人工智能代理能够在没有直接人类交互的情况下自由地进行交易和完成任务。当然，我们希望识别、控制和审计他们的行为。

因此，我们不能允许人工智能代理使用传统的金融系统，因为它们不符合这些系统中使用的身份模型。也就是说，我们不能

  • 审核他们

  • 相信他们

  • 保护它们

当前的现实极大地限制了人工智能与金融系统交互的能力……这还是在我们回到资源和成本问题之前。尽管成本下降了，但成本仍然很高。目前的情况是，像 OpenAI 这样的公司通过向其创建的人工智能模型的 API 访问收费来支付巨额运营成本。

那么，问题就是人工智能的协调，或者找到一种方法来支持以可扩展且值得信赖的方式训练和组织人工智能代理。

任何解决方案都需要三件事：

  1. 一种新的软件模型，其中可信代码执行得到保证且可审计；

  2. 一个新的数字金融系统，不仅可以服务于人类，还可以服务于人工智能代理；

  3. 一种加密身份模型，包括去中心化通信和信誉协议。

目前，实现这三件事的最佳方式是通过区块链协议和智能合约。

区块链与人工智能技术结合的潜力

以太坊、Solana、Avalanche 等主要区块链平台允许安全、可靠、去中心化的应用程序以及可审计的交易跟踪。其他网络，例如 Filecoin 和 Arweave，可以提供廉价、安全的数据存储，这将是人工智能模型的关键。这些网络已经锁定了数百亿美元的价值以及数十万笔日常交易。

由于这些平台依赖于智能合约，而智能合约是自动执行的代码，因此使用这些网络来训练、部署和操作可以通过智能合约相互交互的去中心化人工智能代理似乎只是一小步。

那么，区块链似乎可以成为金融科技和人工智能的完美结合……对吧？未来的人工智能代理可以通过智能合约进行操作，在执行任务时支付少量费用（数字代币）。这不仅提高了人工智能代理的效率，而且还增加了智能合约所在区块链的使用：双赢。

创建可以控制数字钱包的人工智能代理意味着他们可以轻松使用任何基于智能合约的平台或服务，包括 DeFi 协议和基础设施服务。

这样的代理甚至可以通过质押或流动性挖矿协议或在去中心化交易所进行交易来投资其代币。这将有助于优化代理商的财务运营，同时增加市场的流动性。根据代理本身的功能，它可以向其他代理收取其服务费用。

结果将是一个由经济激励的人工智能代理组成的庞大网络，所有代理都通过去中心化协议、交易数据和服务一起工作，同时承担其成本。这是使用传统金融体系无法考虑的。

区块链作为人工智能代理的新工具

即使人工智能代理在链外提供利益，他们也在网络上运行……这意味着他们的活动有一个公开且透明的不可变账本。这种透明度带来了许多好处，包括：

  • 将人工智能活动和内容与人类活动区分开来

  • 为人工智能代理构建身份系统

  • 实时审计代理的行为，以及为人工智能代理创建声誉系统

通过这种控制系统，我们可以奖励系统中的好参与者，惩罚或消除坏参与者，并确定哪些智能体在给定任务中表现最好。人们不仅会知道要信任哪些代理，而且其他代理也会根据任何给定代理的公开可用的链上历史记录快速确定要信任和依赖的代理。

当然，要达到这一点还需要做很多工作。虽然区块链基础设施变得更快、更便宜，但我们需要扩展智能合约区块链来处理人工智能代理创建的所有交易。

智能合约可编程钱包正在开发中，它将帮助代理与智能合约和其他代理进行交易。这些较新钱包类型的关键功能之一是能够集成帐户抽象，允许人类参与者授权人工智能代理从给定的钱包中进行支出。

我们还需要建立跟踪人工智能代理所需的注册表和信誉系统，很可能使用它们的公钥作为每个特定代理的标识符。一旦开始，我预计会看到一些新的业务应用程序和模型的出现。

投资者要点

人工智能领域仍有工作要做，但所有这些发展虽然现在看起来遥不可及，但很可能即将到来。一旦闸门打开，更多的区块链/人工智能用例可能会变得显而易见，从而引入一个更开放、更透明、更高效并最终更适合人类需求的新金融体系。

如果您像我一样相信区块链和人工智能的结合是不可避免的，那么您很可能也可以看到这将如何影响各种区块链平台和协议的价值。

我们一直期待区块链的大规模采用，但我们没有人预料到这种大规模采用可能更多是基于人工智能而不是基于人类。无论哪种方式，它将推动底层区块链的价值，为那些有想象力的人创造机会。
