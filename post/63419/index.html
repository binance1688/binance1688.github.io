而在 NFT 这个概念被越来越多的人熟知后，围绕其独特属性，衍生出了诸如用于证明音乐、视频、图片等数字资产以及艺术品、汽车或活动门票等非数字资产的所有权的用途。大量的机构发觉了NFT与众不同的特点，并选择早早布局。

在2022年上半年，我们看到了NFT的繁荣和围绕数字资产的炒作，并诞生出了 NFTfi 的概念，完成了从闲置资产到流动性的金融创新，通过碎片化、借贷等金融化产品，将NFT的流动性提高，促进生态发展。由此也可以看出，与NFT相关的创新会使NFT变得更加不同。

在本篇Bing Ventures深度研究中，我们将分析当下NFT市场概况，挖掘生态中的最新的发展，并探讨可能面临的挑战和未来前景。

NFT市场概况

从2021 年的夏季开始，NFT 市场经历了一个快速的增长和普及期。在这个时间段内，NFT 交易量和 NFT 价格都创下了历史新高。2021 年的 NFT 交易总额达到了约 200 亿美元，创下历史新高。这个数字比 2020 年增长了约 10 倍。

然而，2022 的下半年却成为对 NFT 领域来说艰难时刻。从Luna 暴雷，到 FTX 倒闭事件，对于整个加密生态都有着非常大的打击，虽然 NFT 依旧伴随着新项目的诞生和探索维持着热度，但奈何大环境的低迷也将 NFT 市场拽入了熊市。NFT 总市值已经从21年的最高点21B下降到如今（2023年10月8日）的4.2B，交易量也大幅度萎缩。

值得关注的创新

相比于 2021 年来看，今年的交易量整体大幅度下滑。NFT 缺乏创新性、可玩性导致的流动性下降问题也越来越严重。在近期，有 ERC-6551、Blur 的 Blend 借贷以及 Opensea 的 Deals 等功能出现，让大家有了一丝期待。因为这些熊市中的小迭代，可能都会是将来NFT市场回暖的关键因素。

Deals

Deals 是由Opensea 发布的一项 NFT 组合交换功能，允许用户使用一系列 NFT 组合与其他用户 NFT 组合进行交换。功能来源于 Seaport 协议，该协议为去中心化的智能合约协议，用于创建和履行 ERC 721 和 ERC 1155 代币的订单。每个订单包含供应方提供的代币组合和接收方需要的代币组合。

价值主张

  • 简化交换过程 -  OpenSea的“Deals”功能旨在通过简化交换过程来吸引更多用户参与NFT社区。采用一种互动组合交易的模式，通过以物易物的模式，降低交易流程，来解决 NFT 流动性问题。改变现有 NFT 的交易流程，省略交易步骤。

  • 减少风险 -  通过利用Seaport，收藏家可以在OpenSea上交换NFT，而不会受到可能在平台外发生的风险。

Blend

Blend是由Blur推出的点对点永续借贷协议，可以支持包括NFT在内的任意抵押品。Blend的白皮书介绍说，“Blend 是一个灵活的、无需许可的浮动利率借贷协议，可以支持任意抵押品同时没有预言机的依赖性，允许市场决定的任何利率和贷款价值比率。”

价值主张

  • 灵活性 -  Blend提供无到期的贷款，允许借款人按自己的方便还款。重新融资拍卖机制进一步增强了这一灵活性。

  • 简单性 -  通过避免oracles，Blend简化了NFT贷款过程。贷方承担风险并设定首选的贷款价值（LTV）比率和利率，市场然后确定条款。

  • 资本效率 -  “Buy Now, Pay Later”（先买后付）的选择降低了小交易者进入的门槛，使他们能够交易如BAYC和Wrapped CryptoPunks等大票收藏品。

  • 风险管理 -  重新融资拍卖降低了违约风险并提高了效率。但是，它要求借款人和贷方都监控NFT底价，尤其是在市场波动性大的时候。

ERC-6551

ERC-6551 也叫做 “Token Bound Accounts/代币绑定账户，是由 @futureprimitive 团队推出的新以太坊标准。ERC-6551 将使每张NFT都拥有一个专属于自己的钱包。

同时由于其使用的是 CA(合约账户，没有私钥，通过代码来控制智能合约)，意味着所有支持 ERC-6551 的NFT 都将自动获得一个绑定地址，而无需进行任何额外操作。 ERC-6551 将为所有 ERC-721 Token 提供一个智能合约账户，这些账户不仅能够使得 ERC-721 Token 拥有 ERC-20、ERC-721、ERC-1155 等各类资产，还能使 ERC-721 Token 能够与各种应用程序交互。 自 ERC-6551 概念被提出，到真正落地的项目，其独特的体验迅速引起了加密社区的密集关注。而到如今，已经有了许多基于ERC-6551的应用出现。

价值主张

  • 更多的互动性和功能 -  ERC-6551不仅仅是一个新的NFT标准，它还为NFTs提供了更多的互动性和功能。这意味着NFTs现在可以与其他智能合约互动，拥有代币，甚至代表真实世界的实体。例如，游戏玩家可以用一个NFT账户容纳其所有的游戏资产。

  • 安全性的提高 -  令牌绑定账户的一个主要优势是其安全性。由于ERC 6551的NFT可以储存器所拥有的所有其他资产的所有权历史，这使得这些资产更能抵抗黑客攻击和盗窃。此外，这也意味着NFTs的所有权和交易记录都是透明的，这为用户提供了更高的信任度和确定性。

  • 解决ERC-721的限制 -  尽管ERC-721为NFTs的广泛接受铺平了道路，但它具有一些固有的限制，例如它不能与其他智能合约互动，也不能存储额外的资讯或价值。 ERC-6551解决了这些问题，使NFTs不仅仅是数字资产的所有权证明，还可以作为一个功能强大的智能合约工具。

  • 更多的创新空间 -  ERC-6551解决了ERC-721的多个限制，模糊了NFT、钱包和智能合约之间的界限，为开发人员提供了更多的创新空间。它为整个区块链生态系统带来了巨大的潜力和机会。

1. AquaNet ：社交类产品，ETHGlobal Waterloo 2023 Hackathon决赛⼊围的项⽬

功能与创新：

  • NFT“活化”社交网络：AquaNet是一个基于AI驱动的NFT的社交网络，让NFT可以与其他NFT互动，发展处独有的“个性”，分享信息，表达兴趣等。

  • NFT的活动记录：利用ERC-6551标准，每一个NFT都可以在AquaNet中创建自己的账户，并保留其所有的活动记录。这为NFT的持有者提供了更多的信息和价值。

  • AI个性：通过GPT-3.5技术，每一个NFT都可以根据其元数据生成独特的个性，如用户名、简介等。这为NFT提供了更高的价值和吸引力。

未来展望：

  • NFT的社交化革命：AquaNet结合AI技术和ERC-6551标准，有可能引领NFT的社交化革命，而是成为NFT社交领域的先锋，为NFT提供前所未有的价值和机会。

  • 利用ERC-6551的优势：AquaNet可以利用ERC-6551的特性，为NFT提供更多的互动性和功能，这可能会吸引更多的用户和开发者加入。

  • NFT的新应用领域：AquaNet可能会开创NFT的新应用领域，如NFT的社交媒体、NFT的互动游戏等。

2. Fukuro：盲盒类产品，ETHGlobal Waterloo 2023 Hackathon决赛⼊围的项⽬

功能与创新：

  • 利用EIP-6551的创新交易方式：该项目利用EIP 6551创建了一个拍卖市场，用户可以在此市场上列出和竞标Fukuro包。Fukuro包：灵感来源于日本的福袋（Fukubukuro），也称为“幸运袋”。每个Fukuro都可以包含以太坊地址可以包含的任何东西，如其他NFTs、ERC20代币等。

  • 多样化的资产交易：用户可以交易各种资产，从艺术收藏、交易卡片集到策略投资组合。

  • 完整的市场体验：从列出、竞标到交易，用户可以在一个平台上获得完整的市场体验。

未来展望：

  • 扩大应用范围：随着NFT市场的持续增长和用户对于新型交易方式的接受度提高，Fukuro包的概念可能会吸引更多的艺术家、投资者和收藏家。这不仅仅是因为它提供了一种新颖的交易方式，还因为它允许用户在一个包中交易多种资产，这为复杂的交易策略和组合提供了可能性。

  • 成为NFT拍卖领域的先锋：考虑到Fukuro包的独特性和它在拍卖市场上的应用，这个项目有潜力成为NFT拍卖领域的领导者。它可以为用户提供前所未有的交易体验，从而吸引更多的用户和开发者。

  • 与其他平台和服务集成：Fukuro包的概念可以与其他平台和服务集成，为用户提供更加丰富和完整的体验。例如，它可以与DeFi平台集成，允许用户使用Fukuro包作为抵押品获得贷款，或者与游戏平台集成，允许用户使用Fukuro包中的资产在游戏中交易。

Source: ETHGlobal Waterloo 2023 Hackathon

3. Piggybank ：存钱罐，ETHGlobal Waterloo 2023 Hackathon决赛⼊围的项⽬

功能与创新：

  • NFT储蓄账户：Piggybank 6551是一个基于NFT的储蓄账户，用户可以铸造Piggybank NFT，并通过付款功能或直接发送ETH到NFT的6551账户地址来为其充值。

  • 实时更新的元数据：随着每个Piggybank中ETH的积累，NFT的元数据（颜色、文本标签和属性）会立即更新，这得益于100%的链上元数据渲染。实时更新的元数据为用户提供了直观的反馈，使他们可以实时了解其储蓄的状态。

  • 安全的ETH提取方式：Piggybank 6551的设计确保了ETH的安全性，用户只能通过销毁NFT来提取ETH。铸造NFT（存钱罐）➡往NFT（存钱罐）内发送ETH➡随着ETH数量变化，该NFT颜色发生变化➡取钱方法是燃烧掉该NFT。

未来展望

  • 与其他NFT集成：Piggybank 6551的账户实现与所有现有的NFT兼容，这意味着用户可以为任何现有的NFT创建Piggybank风格的保险库钱包。

  • 技术创新：Piggybank 6551利用了ERC-6551和其他技术，未来可能会引入更多的功能和特性，如更高的安全性、更快的交易速度或更好的用户界面。

  • 成为NFT储蓄领域的先锋：结合NFT和储蓄账户的概念，Piggybank 6551有可能成为NFT储蓄领域的领导者。

  • 拓展到其他媒体：Onchain Albums 可能会考虑拓展其服务，包括其他形式的艺术作品，如电影、电视节目或播客。

  • 社区驱动的特性：考虑到 ERC6551 的特性和去中心化的趋势，Onchain Albums 可能会引入更多的社区驱动的特性，如粉丝投票、众筹或艺术家与粉丝的直接互动。

  • 音乐行业的创新：Onchain Albums 有潜力引领音乐行业的创新，为艺术家和粉丝提供新的机会和体验。

9. Lens V2

功能与创新：

  • NFT 的社交化：通过使用 ERC-6551 标准，Lens V2 允许用户将价值累积到配置文件 NFT，而不是传统的钱包地址，从而增强了用户的社交体验和互动性。通过为 NFT 提供自己的社交关系和货币化机会，Lens V2也为 NFT 市场带来了新的机会和价值。

  • 多社区支持：用户可以访问和互动多个社区，为他们的句柄和 NFT 创建独特的操作和体验。

  • 去中心化的优势：Lens V2 利用去中心化的优势，为用户提供了一个更加安全、透明和可靠的社交平台。

未来展望：

  • 拓展 ERC-6551 的应用：考虑到 ERC-6551 的特性，Lens 可能会进一步拓展其在社交领域的应用，为用户提供更加丰富和个性化的社交体验。

  • NFT 社交网络的领导者：结合其独特的功能和去中心化的优势，Lens 有潜力成为 NFT 社交网络领域的领导者，为整个行业带来新的机会和挑战。

  • 与其他平台的集成：Lens 可能会考虑与其他流行的社交平台和 NFT 市场进行集成，为用户提供一个一站式的社交和 NFT 体验。

面对的挑战及困境

宏观NFT市场的挑战

NFT市场与加密货币市场，尤其是以太坊，有着密切的关联。当加密货币市场表现不佳时，NFT市场往往也会受到冲击。这种高度的关联性意味着，NFT市场的稳定性和健康性很大程度上取决于更广泛的加密货币市场的表现。

Source: Dune, @hildobby

数据更加突显了NFT市场的低迷程度。市场犹如回到上年暑假的低迷情况，交易量大幅下降，尽管Blur的出现曾稍微提升了市场的需求，但在熊市下投资者的资金选用更加谨慎，以NFT作为资产投资的资金少之又少。而这亦不利于nft新技术的发展。 由于市场的低迷，NFT的吸引力和实用性受到了挑战。为了使市场重新焕发活力，需要引入新技术和创新，否则NFT可能会陷入一个纯粹的投机和炒作的循环中，失去其真正的价值和潜力。

创新的瓶颈

  • ERC-6551的潜力与局限性  尽管ERC-6551为NFT提供了更多的互操作性和可玩性，但目前市场上真正利用这一标准的创新产品仍然相对较少。这可能是因为开发者和项目方还在探索如何最大化利用这一新标准的潜力。

  • 欺诈行为的风险  在一个去中心化的环境中，欺诈行为始终是一个需要警惕的问题。对于NFT市场，这意味着可能会有不法分子出售伪造或虚假的NFT。虽然ERC-6551为NFT提供了更多的功能，但它并没有直接解决这一问题。为了确保市场的安全和买家的利益，可能需要在市场层面或通过外部智能合约来解决欺诈问题。

  • 所有权循环的问题  ERC-6551的一个潜在风险是NFT可能被误转移到其自己的代币绑定账户，导致资产被锁定。这种“所有权循环”问题可能会导致资产的永久损失。为了避免这种情况，可能需要进一步的技术和策略来确保资产的安全。

未来展望：NFT的奇异演变

对于NFT生态的长期发展，整体加密市场的复苏还是着扮演关键角色。如果加密货币市场能够迎来更稳定和持续的增长，这将有助于吸引更多投资者进入NFT领域。然而，与此同时，NFT项目本身也需要不断创新和演进，提供更具吸引力的用例和实用性，以推动市场的活跃。

而ERC-6551 的到来则是 NFT 领域的一个历史性里程碑，通过将 NFT 变成自己的智能合约账户，开辟了一个充满新可能性的世界。这为 NFT 带来了更多的多功能性，使它们对用户来说更加实用和相关。缩小范围，我们可以一起关注未来几个月 ERC-6551 的采用和实验的增加情况。

短期来看，我们列举了几样可以期待的关于ERC-6551在未来的实际用例：

  • 扩展商店：开发者可以在上面构建可以安装在 NFT 的扩展插件，想象一下你如果拥有 Azuki ，你只能等待项目方去创建新的合约，为 NFT 添加新的功能，但是在未来如果基于 ERC 6551 产生扩展商店，并且开发者在里面发布很多有趣的功能，你就可以将它添加在自己的 NFT 中，而不是 NFT 的创建者。

  • 社交赛道应用：现在已经有像Lens这类的去中心化社交平台在采用ERC-6551作为主要核心，通过ERC-6551与去中心化社交相得益彰的配合，可以使未来的去中心化社交更加便捷，在增强实用性的同时降低用户使用的门槛。

  • 盲盒类应用：基于ERC-6551的盲盒应用有望为NFT市场带来更多的创新。通过重新定义用户体验、扩展收藏价值，以及创造更多可能性，这一模式或将成为NFT中引人注目的趋势。

  • 游戏赛道：通过ERC-6551的特性，可以让链游提供更佳的用户体验，能够实现基于单角色的升级、以及NFT装备强化、角色换装（NFT时装）等游戏刚需功能。ERC-6551的特性或许将会为游戏赛道注入新的活力，并且能够激发更多开发者的创意。

  • 与DAO结合：通过ERC-6551，DAO社区将集成了更丰富的功能。例如积分与SBT等内容可以被嵌入成员的NFT中，赋予成员更多权益和激励。这种多元赋能设计，不仅提高了成员的参与度，也为DAO的可持续性发展打下了坚实的基础。

长期来看，我们可以期待的关于NFT演变的趋势包括以下几点：

  1. 量子版权时代 在21世纪的数字时代，随着量子计算的逐渐崭露头角，我们不禁思考：当量子计算遇到NFT，会擦出怎样的火花？想象一下，未来可能存在一种量子NFT，其状态可以是“重叠”的。这意味着一个NFT可以同时存在于多个所有权状态，直到被观察或交易。这种前所未有的“量子版权”将彻底颠覆传统的所有权和交易模式，为艺术家和创作者提供更多的创意空间。

  2. 多维宇宙与NFT交互 随着虚拟现实技术和增强现实技术的发展，未来可能存在一个多维的数字宇宙，而NFT将成为连接各个维度的桥梁。例如，一个NFT可能同时存在于2D的数字画作、3D的虚拟雕塑和4D的时间艺术中。这将让创作者的艺术作品在不同的维度中呈现出不同的美感，赋予NFT更多的深度。

  3. DNA级的数字验证 当前，市场上的伪造和欺诈问题是NFT面临的巨大挑战。但想象一下，如果我们能将NFT与生物技术相结合，例如使用DNA序列作为每个NFT的独特标识，那么这种真实性和独特性将几乎不可能被伪造。未来的NFT可能会与每个创作者的DNA序列紧密关联，从而确保其真实性和独特性。

  4. 超级AI的艺术创作 当超级AI逐渐成为现实，这些AI将成为未来的艺术家和创作者。他们可能会使用NFT来出售和交易其创作的艺术品。而这些AI创作的艺术品，由于其深度学习和大数据的背景，可能会具有前所未有的复杂性，为NFT市场带来全新的可能性。

  5. 完全自动化循环的NFT 当前的NFT市场主要关注的是单个资产的买卖。但在未来，可能会出现一种基于循环经济的NFT模式。在这种模式中，NFT可以被回收、重塑和再利用，从而创造出全新的艺术品。这不仅有助于保护环境，还将为艺术家和投资者提供持续的价值。

总之，未来的NFT市场将远超我们当前的想象，其潜力和可能性只受限于我们的创意和技术的发展。但无论未来的NFT如何演变，其核心价值 — — 为创作者提供真实性、独特性和价值 — — 始终不变。我们大可不必对目前的NFT熊市感到过于悲观。
