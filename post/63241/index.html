一、Cosmos Hub 入门

Cosmos Hub是一个专注于互操作性和安全性的独立应用链（appchain）。Cosmos Hub是一个权益证明（PoS）的主权区块链，采用基于账户的会计模型，没有本地的智能合约功能。ATOM是Cosmos Hub的本地通证。Cosmos Hub先驱采用了诸如Cosmos SDK、CometBFT（Tendermint）、ABCI和IBC等技术，后来被许多Interchain网络采用。Interchain由通过通用互操作性协议IBC连接的主权网络组成，也被称为Cosmos生态系统。

Atom Economic Zone（AEZ）是一个与ATOM通证一致的生态系统集，包括与ATOM通证有各种关联和集成的网络。那些从Cosmos Hub验证人那里租用安全性的网络，即Consumer Chains，是AEZ的一部分。由于缺乏可编程性，Cosmos Hub的安全性不能在类似EigenLayer的restaking模型中任意共享，但可以通过治理进行传递，并在个体基础上启用。复制安全性（曾被称为Interchain Security）指的是与另一个链共享完整的Cosmos Hub验证器集，由治理投票许可。

Cosmos Hub于2019年启动，利用了Interchain Foundation和Ignite（也称为Tendermint）构建的技术。包括Interchain Foundation、Binary Builders、Atom Accelerator DAO、Informal Systems、Strangelove等在内的各种团体继续支持Cosmos Hub和Cosmos技术堆栈的发展。Interchain是100多个独立网络的家园，每个网络都有自己独特的支持实体。

三、财务分析

Cosmos Hub 财务指标概述

ATOM的价格连续第二个季度下跌，季度降幅为21.0%，降至7.11美元。这低于整个加密市场的跌幅——9.2%。ATOM的总市值排名季度下降，从第23位降至第24位。在网络代币中，截至第三季度末，ATOM按市值排名第10位。

作为Cosmos Hub的本地通证，ATOM是ICS-20兼容的，允许用户通过IBC协议在相互连接的链之间传输ATOM。ATOM的总发行量约为3.56亿，流通量约为2.93亿。这个通证有以下功能：

  • 在Cosmos上进行的交易需要支付以ATOM结算的交易费。

  • 持有者可以抵押ATOM来运营验证器，确保网络安全并赚取奖励。

  • 持有者可以将ATOM委托给现有的验证器，帮助确保网络安全并获得验证器奖励的一部分。

  • 所有抵押和委托的ATOM都可以用于在网络的治理过程中投票。

ATOM 9月创年内新低，但营收仍环比增长

收入以总交易费用来衡量。 Cosmos Hub 收入环比增长 2.4%。 7 月 31 日，收入出现飙升，日均交易费用升至 0.22 美元，远高于季度平均 0.04 美元。

1、ATOM代币经济学

第三季度实际收益率平均为 4.6%

在Q3，Cosmos Hub的实际收益平均约为4.5%，季末达到4.65%。ATOM的通货膨胀率高达约14%，几乎抵消了约19%的奖励率，该奖励率提供给验证器和委托者。流通供应主要受到验证器和委托者的抵押奖励的影响。由于具有如此高的奖励和通货膨胀率，这对于未参与抵押的ATOM持有者来说是一种较重的负担。

截至第三季度，ATOM 代币面临两种形式的通货紧缩压力，从而导致通货紧缩。

  • 治理燃烧 - Cosmos Hub拥有一个链上治理机制，ATOM代币持有者可以进行投票并提出提案，例如修改共识参数和社区池基金分配。为了提交提案，必须从任何代币持有者那里存入至少250个ATOM。如果提案被否决，存款将被销毁。

  • 削减 - 如果验证器节点出现双签名交易或验证器离线时间过长，ATOM中的验证器节点奖励可能会被削减。削减的奖励随后将被销毁。

被拒绝的 ATOM 2.0 提案重新构想了 ATOM 代币经济。 此次改革的目标是在几年内大幅减少 ATOM 的发行量，然后完全消除通货膨胀。 此外，它还旨在提高资本效率，并通过流动质押将 ATOM 定位为跨链储备资产。 最终，这些行动将推动 ATOM 代币的价值增值。

虽然最初的提案被拒绝（可能是因为其广度较大且快速引入），但其中的各个组件继续作为个别提案进行探讨。值得注意的是，在ATOM 2.0首次引入后，两个提案（增加财政储备规模和改进ATOM资本效率的流动权益特性）在随后的提案中都获得了通过。社区中仍在讨论的其他ATOM 2.0功能包括无许可的共享安全模型和用于Interchain技术的社会协调（即治理）中心（例如IBC、CosmWasm等）。

社区此后未能通过其他与代币经济相关的提案，例如基于交易费用量销毁ATOM或激励持有者质押代币。开发者还在讨论改进Cosmos Hub的费用机制的选项，以减少垃圾交易。其中一个选项是使用类似于EIP-1559的模型，其中支付给验证者的费用的一部分将被销毁。

在第三季度，Atom Accelerator DAO（AADAO）选择了Blockworks Research、Binary Builders和RMIT大学，提出了一个新的ATOM代币经济改革——Tokenomics RFP。这些团队提出的五个提案探讨了治理、流动权益、财政储备、公共资金、Atom Economic Zone内的关系以及ATOM发行等方面。

Blockworks Research提出了一项财政政策变更的提案。该提案旨在从当前的静态10%社区池税转变为更多样化的税收方法。这种新方法包括以下组成部分：

  • 投票权力税： 引入一项税收，以阻止将代币委托给具有较高投票权力的验证者，以解决权益集中化问题。

  • 动态社区池税： 提议将社区池税从固定的通货膨胀百分比更改为随时间减少的计划，直到达到基本税率为5%。

四、网络分析

Cosmos Hub 网络概述

每日活跃地址和每日交易分别环比下降了19.8%和6.3%。Cosmos Hub继续转变为一个新型多链生态系统的中心，其中Cosmos Hub不仅仅是一个互操作性中心，还充当了安全性提供者。随着这一过渡的进行，Cosmos Hub主网上的活动可能会减少，因为使用情况转移到其“子链”。这在Atom Economic Zone Analysis部分进一步探讨。

活动地址和交易保持紧密耦合

历史上，活跃地址和交易一直紧密相连，这一趋势延续到了第三季度。这种一致的关系表明，8月份交易活动的激增来自更多用户上线并与网络进行交互，而不仅仅是现有用户更频繁地进行交易。

1、质押和去中心化

质押比例环比下降 3.6%，但年初至今仍增长 7.5%

尽管环比下降了3.6%，Cosmos Hub的67.6%的质押率仍然是其他Layer 1网络中较高的之一。质押者包括验证者和委托人。质押ATOM并符合系统要求的用户可以运营一个验证者，以确保网络安全并获得奖励。Token持有者可以将最少一枚ATOM委托给现有验证者，以获得相应验证者奖励的一部分。委托人根据质押的代币数量获得奖励，尽管他们要支付由个别验证者设置的佣金。在第三季度，提案826将验证者的最低佣金设置为5%。

只有前180名验证者，按照自身质押和委托的ATOM代币的综合排名，才能获得奖励。其中前八名验证者控制了总质押的33%，使Cosmos Hub的Nakamoto系数为8。

2、治理和拨款

Cosmos Hub 采用链下和链上治理流程的组合。 网络改进提案和参数更改是通过链下方式设计和讨论的，通常是在 Cosmos 治理论坛中。 任何人都可以提出网络或金库管理提案，并参与是否实施该提案的讨论。 用户还可以将任何网络变更请求作为官方提案提交给 Cosmos Hub 的链上治理系统。 一旦进入链上治理系统，ATOM 利益相关者就是否批准（并最终执行）提议的变更进行投票。

第三季度共有 24 项治理提案，其中 10 项获得通过。 值得注意的通过提案包括 V11、V12 和 V13 软件升级，以及扩展 AADAO 对 ATOM 协议拥有的流动性 (POL) 的管理权。 AADAO 透明度报告 4 详细介绍了 DAO 分配的财务和拨款。 第三季度，第 4 批和第 5 批 AADAO 拨款总计 109.7 万美元，已分发给获奖者。 这些拨款资助的项目包括赞助 Cosmoverse、DoraHacks 的黑客马拉松等等。 后来，宣布了第六批，并于 10 月份发放了补助金。 这些轮次使 AADAO 发放的拨款总数达到 27 笔，总价值达 250 万美元。

Binary Builders 的“ATOM Alignment Treasury”旨在提高 Cosmos Hub 治理和投票的去中心化程度。 随着验证者运营成本的不断增加以及对网络去中心化的担忧，Binary Builders 提议根据验证者的投票权向其分配更多权益。 该提案旨在支持低端验证者并进一步使网络分布式发展。 它还探讨了如何使用 ATOM 在其他 AEZ 网络上执行投票权（更多详细信息请参阅 Atom 经济区分析部分）。

除了财务分析的 ATOM 代币经济学部分概述的财政政策变化之外，Blockworks Research 的提案还对治理产生影响。

  • 投票权税不鼓励委托具有更高投票权的验证者来解决权益集中化问题。

  • 动态流动性质押税将针对流动性质押者征收，并且随着流动性质押需求的增长而增加。 它的目标是在为 Cosmos Hub 创造税收收入的同时，找到流动性质押市场渗透的自然平衡。

  • 取消 25% 流动质押上限将使更多参与者从流动质押中受益。 Cosmos Hub 本身不支持流动质押，但可以通过 AEZ 中的网络 Stride 获得。 Atom 经济区分析对此进行了进一步探讨。

  • Cubic 授权有潜力解决链上治理的中心化问题，同时确保仍能满足法定人数。

3.IBC

Cosmos Hub IBC 概述

Cosmos Hub 拥有 56 个 IBC 对等点和 269 个 IBC 通道。 “对等方”是指已建立 IBC 连接器的 Cosmos Hub 的不同交易对手的数量； “通道”是指可用于连接对等点的路由总数。 唯一发送者、唯一接收者和总传输（入站和出站）的活动全面下降。 入站独立接收者环比下降 40.6%，降幅最大。 所有这些 IBC 指标均创年度新低。

第三季度 IBC 出站转账出现数次高峰

总体而言，出站和入站转账是紧密耦合的。有一些例外情况，比如2023年5月出现的大规模入站激增。在Q3中，有几次每日出站转账超过入站转账2-3倍的高峰。

五、Atom经济区分析

Atom 经济区 (AEZ) 是一个与 ATOM 一致的生态系统集，其中包括具有各种隶属关系并与 ATOM 代币集成的网络。 AEZ 中存在多个不同能力的网络。 Stride 和 Neutron 获得复制安全性，而 Osmosis 则可以使用 ATOM 支付费用。 其他网络（例如 Kujira 或 Agoric）的大部分流动性都依赖于 ATOM。

复制安全（以前称为链间安全）是共享安全的一种形式，是 Cosmos Hub 社区和其他链间网络探索最多的 AEZ 策略。 共享安全性是指允许一个网络的验证者使用该链上的权益参与另一个网络的共识。 这种设置将允许较小市值的网络从较大的网络“租用”安全性。 Cosmos Hub 拥有所有 Interchain 网络中最大的市值，是出租安全的主要候选者。 截至第三季度，Neutron 和 Stride 是唯一使用复制安全性的链。

Binary Builders 的“ATOM 对齐金库”（AAT）旨在使 ATOM 成为 AEZ 内的默认代币，并协调 Cosmos Hub 和消费者链之间的质押和治理。 该团队打算将 AAT 资金用于开发工作并引导整个 AEZ 的流动性。

1.Neutron

792 号提案的通过使 Neutron 成为第一个通过复制安全性利用 Cosmos Hub 验证器集和安全性的链。 Neutron 作为 Cosmos Hub 的 CosmWasm 扩展，实际上已经成为 Cosmos Hub 的 DeFi 中心。

Neutron 仍然是一个相对较新的网络，今年才推出。 提案 72 为 Neutron 的开发分配了 Cosmos Hub 资金，提案 819 将该资金直接重新分配给 Neutron 基金会。

2.Stride

Atom占据stride TVL的80%以上

Stride 在 Neutron 之后不久就实施了复制安全性。 该网络可以为各种链间网络和资产提供流动性质押。 由于 ATOM 是 Stride 活动的核心部分，因此采用 Cosmos Hub 安全性显然是早期的选择。 流动质押是一个巨大的市场，也是各种其他生态系统的一个故事。 Cosmos Hub 受益于通过复制安全性在 AEZ 中拥有流动性质押解决方案，因为公用事业通过治理进行协调。

Stride-staked ATOM stATOM 也是其他 AEZ 网络不可或缺的一部分。 Cosmos Hub 提案 805（提案 800 的修正案——流动性即服务）将让 AADAO 将 450,000 ATOM 部署到 Astroport Neutron 上的 ATOM/stATOM 恒定产品池中。 在 Osmosis 上，ATOM/stATOM 的 TVL 超过 2000 万美元。

治理提案 817 寻求 Cosmos Hub 社区的批准，以选择一个由 5 名成员组成的顾问委员会，负责通过 Stride 授权流程评估和选择主机链验证者。

3.Osmosis

Osmosis AMM DEX 自诞生以来就与 Cosmos Hub 和 ATOM 深度集成。 Osmosis 上的第一个池是 ATOM/OSMO 池，截至 2023 年第三季度，按 TVL 计算仍然是最大的池。 总 Osmosis TVL 的大约 75% 来自 ATOM 池（主要是 ATOM/OSMO 和 ATOM/stATOM）。

Osmosis 的“ATOM 联盟”和在 AEZ 中的作用比流动性对更深入。 ATOM 是 DEX 上的替代 Gas 代币。 虽然 ATOM 在某些情况下被用于Gas，但由于活动模式，其中许多被怀疑是机器人。

Cosmos Hub 上的 810 号提案将纳入 Osmosis DEX，以实现灵活的 Gas 费支付。 该提案将允许使用任何代币支付 Cosmos Hub 上的交易费用，利用 Osmosis 将替代代币交换为 ATOM。

4.其他网络

更多链可能很快就会以不同的身份加入 AEZ。 Composable Finance 提交了一份加入 AEZ 的提案，并从 Cosmos Hub 获得复制安全性，就像 Stride 和 Neutron 所做的那样。 以 Neutron 作为可编程性和 DeFi 中心，以 Stride 作为流动质押解决方案，Composable 建议成为汇总结算中心，因为它与其他多链生态系统相连。

虽然复制安全性很容易严格定义，但“ATOM 一致性”则更加主观。 一些社区成员认为，Agoric 和 Kujira 等网络已经成为 AEZ 的一部分，因为它们的生态系统与 ATOM 交织在一起。 换句话说，ATOM 代币可以提供大部分流动性或 Cosmos Hub，从而促进大部分链间活动。

六、总结

自成立以来，Cosmos Hub 的角色就是更广泛的 Cosmos 生态系统的互操作性中心。 在第三季度，Cosmos Hub 采取了重大举措，将其角色转变为安全提供商。 通过复制安全性，Neutron 和 Stride 成为第一批受到 Cosmos Hub 保护的链。 随着区块链加入 Atom 经济区，Cosmos Hub 作为模块化安全提供商的新地位正在增强其作为更广泛的 Cosmos 生态系统文化中心的作用。

此后，Cosmos 社区提交了包含更多区块链的提案，例如 Composable Finance。 此外，其他 Interchain 网络正在探索其他形式的“ATOM 联盟”，例如使用 ATOM 进行收费，以成为 Atom 经济区的一部分。

在 ATOM 2.0 提案被拒绝后，该提案以各种方式重新构想了 ATOM 代币经济学，大多数概念继续作为单独的、较小的提案进行探索。 治理、去中心化、流动性质押、财务管理和代币经济学是第三季度探讨的主题。 AADAO 一直在向各个团队提供资助来解决这些主题，包括 Tokenomics RFP。
