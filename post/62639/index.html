作为老牌的区块链生态系统，Cosmos以其去中心化和开放的特性，在与其他生态系统相比中脱颖而出。然而，由于相对缺乏高调的市场宣传活动，Cosmos在推特上给人的印象总是不温不火。但实际上，作为当今应用链革命的发源地，Cosmos从不缺乏孵化和吸引优质生态项目的能力。尽管Cosmos生态系统在一段时间里相对萧条，但目前来看，它仍是加密行业中最有潜力实现百倍增长的领域之一。最近的发展趋势表明，具备强大基本面和可持续发展叙事的项目将在即将到来的牛市中受益匪浅，而 Cosmos 已做好充分利用这一趋势的准备。

**崛起中的Cosmos生态：重新评估被低估的潜力 **

Cosmos是一个基于区块链技术的开放源代码生态系统，旨在实现不同区块链之间的互操作性。尽管在技术和生态项目方面与一些头部系统相比还有一些差距，但是在去中心化和互操作性方面，Cosmos具备强大的优势。尤其是Cosmos 开发团队推出Cosmos SDK 和 IBC，帮助开发者解决了重复造轮子的难题，在推动整个行业技术进步方面，Cosmos发挥了重要作用。

然而，与实力不相符的是Cosmos的市场估值，相对于其他一些区块链生态系统，Cosmos的生态系统相对较小，尚未吸引足够的开发者和用户，再加上Cosmos的技术和应用场景相对较为复杂，需要更多的时间和资源来推广和普及。因此在更看重数据和流动性的加密市场，Cosmos的市值长期以来被严重低估，截至目前，Cosmos的市值只有31.5亿美元，排名第23位。其代币ATOM也因缺乏实用性而与其他生态系统的代币市值相差巨大。

为了缓解这种局面，去年的Cosmoverse大会上，Cosmos 2.0白皮书被推出，计划改变Cosmos Hub网络和 ATOM 代币利用率低下的困境。围绕这些目标，Cosmos陆续宣布了一些进展，例如 Cosmos Hub将原本Interchain Security（链间安全）升级为“Replication Security”（复制安全），通过了流动性质押模块（LSM）提案。此外，今年5月份Cosmos 生态链Neutron 主网上线，Neutron首次使用了升级后的安全机制“Replication Security”，并宣布将向 Cosmos Hub 质押者空投，关于 Cosmos、 ATOM 价值捕获及其生态的讨论才逐步升温。

正是基于这些技术方面的改进，Cosmos的生态项目在今年迎来了流动性爆发。首先，生态公链Canto的日交易量一度超过Solana，为Cosmos带来了一个成功的开端。接着，流动性质押协议Stride、Layer1公链Injective和Kujira的总锁定价值（TVL）持续上升，相较于一月份增长了2到6倍不等。今年4月份，根据Token Terminal的数据监测，Cosmos SDK上的活跃开发者数量一度超过了以太坊，目前位居第二。此外，最近基于Cosmos SDK构建的模块化区块链网络Celestia也引发了市场关注，Celestia + Cosmos模式也被誉为应用链的最终形态。

Cosmos生态项目出色的表现重新引起了市场对Cosmos的关注。许多用户对于Cosmos2.0今后的计划落地表现出了较高的期待，市场上甚至出现了“Cosmos Summer要来了”的声音。在最近的这波反弹行情中，与Cosmos生态相关的代币，如Celestia（TIA）、Injective（INJ）和Osmosis（OSMO），在过去一周内涨幅都超过了20%。INJ更是在一个月内涨幅超过了110%。这些积极的趋势表明Cosmos正在逐渐赢得市场的认可和关注，并为其未来的发展奠定了坚实的基础。

**Cosmos技术的未来发展规划：为流动性爆发提供技术支撑 **

在我们探索 Cosmos 生态的最新发展进展时，有必要先了解一下 Cosmos 技术的未来发展规划。持续的技术创新对于维持生态系统的增长并为区块链应用程序提供前所未有的流动性至关重要。20223年 9 月，推动 Cosmos生态发展的Interchain 基金会公开发布了Interchain 2024 发展计划，概述了 Cosmos 技术的未来发展规划。

CometBFT：跨链可扩展性的基石

CometBFT 是一种创新的共识算法，是 Interchain 可扩展性的核心。通过确保验证者就区块生成达成共识，CometBFT 使 Interchain 生态系统能够保持安全性、去中心化和完整性。Interchain基金会计划进一步扩大CometBFT的覆盖范围和适用性，赋能主权、可互操作应用程序的开发。目前，CometBFT 已升级至 v0.38，基金会计划于 2023 年底发布 v0.39，具有效率提升、全面的数据配套 API 和增强的索引系统等特点。

Cosmos SDK：模块化和新功能

Cosmos SDK 是一个用于开发区块链或 Rollup 应用程序的多功能工具包。它已被许多新的区块链项目采用，包括基于 Cosmos 的应用程序、Fairblock、Berachain 和即将推出的 dYdX V4。在未来的发展计划中，Cosmos SDK将侧重于模块化和引入新功能。这将使开发人员能够创建更复杂、模块化的应用程序，从而培育更广泛、更强大的生态系统。

CosmJS：提高效率和用户体验

CosmJS 是一个帮助开发人员将 JavaScript 客户端与 Cosmos SDK 区块链集成的库。Interchain 基金会计划通过新的编码器增强 CosmJS，提高其效率，减少代码膨胀，并提供更精简、更专业的开发体验。这将使开发人员能够创建更优雅、用户友好的应用程序，最终增强 Cosmos 生态系统的吸引力。

CosmWasm：解锁去中心化、无许可部署

CosmWasm 是一个基于 Cosmos SDK 构建的智能合约框架。它使开发人员能够在没有许可要求的情况下部署去中心化应用程序，同时还允许区块链开发人员将其产品开发周期与区块链开发分开，从而降低验证器升级的成本。Interchain基金未来会进一步优化CosmWasm，使开发人员更容易创建和部署各种应用程序，最终提高区块链利用率。重点关注领域包括 IBC 采用和基于账户的智能合约。

IBC 协议：增强互操作性和可扩展性

IBC（区块链间通信）是一种能够在区块链之间实现安全、去中心化且无需许可的数据传输的协议。Interchain 基金会计划在 2024 年更新 IBC，重点关注两个主题：可扩展性和可用性。通过提高 IBC 的可扩展性和互操作性，Cosmos 将进一步增强其生态系统的容量，从而实现区块链、应用程序和智能合约之间的无缝交互。

**Cosmos生态最新改变：加速发展与创新 **

随着数字资产市场的不断发展，最近几个月内，Cosmos生态系统再次展现了巨大的潜力。作为一个开源的多分支区块链网络，Cosmos生态系统拥有智能合约、可扩展性、安全性和跨链互操作性等多项优势。同时，它的应用链理论也在不断发展演进。以下是近期Cosmos在生态领域发生的一些值得关注的变化。

流动性质押模块： Liquidity Staking Module (LSM)是一个新的模块，旨在为Cosmos生态系统中的流动性抵押提供模块化解决方案。Cosmos Hub 上的 LSM 实施于 2023 年 6 月 14 日开始，并发布了软件升级计划。LSM的实施对Cosmos生态系统的影响是，用户现在可以直接流动性抵押他们已经抵押的ATOM，而无需等待解除绑定期。

代币经济学升级： Cosmos代币ATOM的经济模型发生了重大改变，从过去追求高回报的时代转向追求真实收益的叙事。5月10日，Neutron主网上线了，它是一个结合了Cosmos SDK和IBC的CosmWasm（Cosmos虚拟机）平台，是第一个使用Cosmos的“Replication Security”消费链。Neutron的目标是解决ATOM1.0因为缺乏明显的实用价值而受到批评的问题，并带来新的叙事和推动力。

互操作性增强： 互操作行一直是Cosmos 相较于其他生态系统最大的优势，今年8月份，Evmos2.0的推出无疑进一步巩固了Cosmos 在这一方面的优势，Evmos 建立在 Cosmos 的区块链间通信（IBC）协议之上，该协议有潜力将强大的多链生态系统提升到新的高度，Evmos2.0 将实现以太坊和 Cosmos 生态系统之间的互操作性，允许开发人员构建可以与两个生态系统交互的应用程序。
永续合约崛起： 永续合约在数字资产市场中扮演着重要角色，并且在近期经历了快速增长。作为以太坊生态领先的永续合约交易所，dYdX v4宣布迁移到Cosmos生态中，无疑会对Cosmos永续合约模块产生积极的影响，因为它将为Cosmos生态系统带来更多的流动性和创新。

原生流动性增加： 原生流动性对于DeFi领域至关重要，它不仅能够实现无缝的链上交互，还能减少桥接风险。最近，Tether的稳定币USDT已经集成到Kava区块链中，巩固了Tether在该领域的领先地位，并为Cosmos引入了新的原生流动性。此外，今年3月份，Cosmos 生态原生资产发行平台Noble正式宣布与Circle合作，致力于将原生 USDC 带入Cosmos。Noble称这种整合将在未来几个月内在 Cosmos 中催生数亿美元的流动性。

**Cosmos生态热门项目盘点 **

Initia

Initia是一个基于Cosmos的热门项目，致力于构建高度互联的模块化Rollup网络。通过抽象传统模块化和多链系统中的复杂性，Initia为开发人员构建可扩展且自主的系统提供了支持。它的架构包括L1（Layer 1）、L2（Layer 2）和通信层，使开发人员能够轻松地启动基于Initia L2的应用程序特定区块链，而无需了解复杂的链级基础设施或运行验证者集合。

Initia采用了Cosmos的软件开发套件（SDK），但使用了一种名为MoveVM的计算框架，该框架基于Move智能合约语言。同时，Initia利用乐观汇总（Optimistic rollups）作为特定于应用程序的L2解决方案。通过整合L1、L2和通信层，Initia实现了不同虚拟机（如EVM、WasmVM和MoveVM）之间的无缝消息传递和桥接，使其成为Cosmos IBC协议兼容的第一个MoveVM计算框架，并实现了与Cosmos生态系统中其他区块链网络的无缝交互。

近期，据The Block报道，Initia已获得币安实验室的种子轮前投资，具体融资金额和进一步细节尚未披露。这笔资金将用于支持Initia基础设施的开发、产品工具的扩展以及关键应用层项目的孵化，以推动Initia生态系统的扩大。目前，Initia仍处于测试网的开发阶段，一旦上线，将极大地提升Cosmos生态系统的流动性。

Celestia

Celestia是一个基于Cosmos生态系统的模块化区块链网络，旨在提供可插拔的共识和数据可用性层。该项目的目标是让任何人能够快速部署去中心化的区块链，而无需额外的共识层成本，并为开发者提供更灵活的构建和部署区块链应用程序的能力。

Celestia的一个重要特性是数据可用性采样。它允许节点仅验证一个区块的一小部分数据，而不需要验证整个区块。这样可以大大降低验证节点的硬件要求，并提高网络的扩展性。在与Cosmos的合作中，Celestia作为数据可用性层与Tendermint和Cosmos Zones合作。这使得Cosmos生态系统中的链可以利用Celestia提供的数据可用性功能，并通过欺诈证明来实现最小化的信任。这种合作促进了Cosmos生态系统的互操作性和扩展性，同时也为Cosmos生态系统提供统一的安全级别。

10月31日，Celestia主网正式上线，其代币TIA也相继在各大交易所上架，目前TIA币价为2.33美元。

dYdXV4

dYdX 是以太坊上领先的 L2 Dapp，其最新版本 dYdX V4 预计将会在今年12月份发布，此次升级是dYdX一个重要里程碑，因为它准备迁移到去中心化网络 Cosmos。此举旨在解决以太坊的可扩展性限制，为 dYdX 快速增长的用户群和交易量提供更加去中心化的架构。

dYdX作为热门的L2 Dapp，吸引了大量的用户和交易，这给以太坊区块链带来了压力。然而，以太坊的可扩展性限制阻碍了 dYdX 的扩展，促使需要更先进的解决方案。Cosmos 及其尖端的 SDK 为 dYdX 的发展提供了完美的解决方案，支持开发去中心化且独立的区块链网络，可以满足 dYdX 不断增长的需求。

通过迁移到 Cosmos，dYdX V4 将不仅提高交易速度，还能够消除中心化的担忧。Cosmos 的去中心化网络使得 dYdX 能够继续遵守其对透明度和公平性的承诺，为用户提供更安全可靠的交易体验。作为一家头部的去中心化交易所，dYdX 的日交易量已超过 30 亿美元，其迁移也将会为 Cosmos 生态系统带来大量新用户和流动性。

Neutron

Neutron是一个基于CosmWasm平台的智能合约基础设施，它让开发人员可以在这个平台上轻松地部署dApp，而不需要经过复杂的治理投票或白名单验证。通过与Cosmos Hub和其他项目的合作，Neutron将安全性和跨链功能结合起来，让DeFi应用程序能够安全地扩展到由51个以上互相连接的区块链组成的庞大网络中。简单来说，Neutron为开发人员提供了一个方便而安全的方式，让他们的应用程序能够在多个区块链之间顺畅运行。

虽然Neutron之前并不太知名，很多人可能对这个项目不太了解，但实际上，Cosmos一直在积极准备。根据Interchain堆栈2024年的路线图，Neutron将成为Cosmos上首个消费者链。它旨在托管围绕NTRN和ATOM构建的DeFi产品生态系统，为代币生成效用和需求提供支持。这将对推动Cosmos生态经济的发展以及ATOM 2.0的实施发挥重要作用。

今年5月份，Neutron成功完成了一轮融资，由Binance Labs领导，共筹集了1,000万美元。其他参与者包括CoinFund、Delphi Ventures、LongHash Ventures、Semantic Ventures和Nomad Capital。Neutron代币NTRN于10月份在币安上线，目前的交易价格为0.290美元。
