前有 Pudgy Penguins 线下卖 NFT 系列玩具，后有 Yuga Labs 裁员，OpenSea 前高管也深陷 NFT 舆论风波。

一波未平一波又起，NFT 的能源消耗以及对环境的影响引起了关注，而相关法律问题和监管也正在浮出水面。

NFT 的监管问题

9 月 13 日，SEC 指控 Stoner Cats 2 LLC（SC2）发售未经注册的证券，SC2 通过销售动画网剧的 NFT 从投资者手中募集了大约 800 万美金。这是 SEC 第二起监管 NFT 案件（8 月底，SEC 指控总部位于洛杉矶的媒体和娱乐公司 Impact Theory 以 NFT 形式进行未经注册的加密资产证券发行，从数百名投资者「包括美国各地的投资者」那里筹集了约 3000 万美元）。

目前，NFT 领域的监管还处于相对初级阶段，以下是一些 NFT 监管的案例和趋势：

  • 金融监管：一些国家的金融监管机构对 NFT 进行监管，主要关注涉及 NFT 交易的数字资产交易平台和市场。他们可能要求这些平台遵守反洗钱（AML）和了解您的客户（KYC）等金融法规，以确保合规性和用户资金的安全；

  • 知识产权保护：由于 NFT 涉及到艺术、音乐、视频等数字内容的交易和所有权转移，相关的知识产权保护也是一个重要问题。一些国家和地区正在探索使用区块链技术来确保数字创意作品的版权归属和保护；

  • 消费者保护：NFT 市场存在着风险，如欺诈、虚假销售和知名度不高的项目。为了保护消费者权益，一些国家的消费者保护机构可能采取措施，要求平台提供透明度、提醒风险，并建立投诉处理机制；

  • 税收要求：NFT 交易可能会带来资本利得税、销售税或增值税等税务问题。一些国家正在研究相应的税收政策，以确保 NFT 交易纳税并符合相关税法规定。

需要注意的是，由于 NFT 领域的快速发展和国际性质，目前，监管措施和法规在各个国家和地区可能存在差异：

  • 美国：美国已经开始监管数字货币市场，并将 NFT 视为数字资产。此外，美国证券交易委员会（SEC）正在审查 NFT 是否应被视为证券；

  • 欧盟：欧洲联盟（EU）正在积极制定针对数字资产的监管法规。草案提议将 NFT 视为“虚拟商品”，但尚未正式获得认可；

  • 中国：中国一直非常谨慎地处理加密货币和数字资产，对其实施了严格的监管。目前，NFT 在中国被认为是虚拟财产；

  • 日本：日本的金融市场监管机构，金融厅(FSA)，正在考虑对 NFT 进行监管。目前，NFT 在日本被视为非货币性质的商品，因此不受监管；

  • 新加坡：新加坡金融管理局（MAS）将数字货币定义为商品，而不是货币或证券。对于 NFT 的监管还没有明确的规定。

需要指出的是，由于 NFT 市场的快速增长和不断变化，各个国家和地区对于 NFT 的监管状况也在不断调整和更新。因此，未来会有更多的国家和地区开始对 NFT 进行监管。

铸造 NFT 及交易 NFT 的环境成本

以全面的视角来看待 NFT 和环境主题至关重要。与之前的许多其他数字技术一样，NFT 的创建也会消耗能源以及产生碳排放。

NFT 的铸造过程涉及通过在区块链上进行交易来证明数字资产的唯一性。每次铸造都会消耗能源，就像数字领域中的任何其他操作一样，尽管在几乎没有用例的情况下铸造 NFT 所消耗的能源量可能会引起警惕。

为了能更真切地感知，dappGambl 的研究确定了 195, 699 个没有明显所有者或市场份额的 NFT 藏品。铸造这些 NFT 所需的能源相当于 27, 789, 258 千瓦时，导致排放约 16, 243 公吨二氧化碳。

这些都是一些很大的数字，所以让我们考虑一下…… 16, 243, 017 千克二氧化碳是 16, 243 公吨，相当于：

  • 根据 EPA 的数据， 2048 个家庭的年排放量为每户 7.93 。

  • 3531 辆汽车的年排放量——根据 EPA 的数据，每辆汽车排放 4.6 排放量。

  • 根据新西兰航空公司的数据，从伦敦（英国）飞往惠灵顿（新西兰）的 4061 名乘客的碳足迹为每人 4 吨二氧化碳。

从更广泛的角度来看：许多日常活动和行业都有能源足迹。银行交易、日常电子产品使用（例如看电影或充电设备）以及整夜灯火通明的城市地区都会影响我们的整体能源消耗和碳足迹。在更大的消费图景中查看 NFT 的能源使用情况至关重要。

与任何行业一样，深思熟虑和负责任的创造和消费对于平衡利益与任何潜在缺点至关重要。

NFT 还有救吗？

根据 dappGambl的报告，大多数 NFT 现在已经毫无价值。根据该平台的数据，在其确定的 73, 257 个 NFT 收藏中，有 69, 795 个收藏的市值接近于零。当 dappGambl 根据区块链分析网站 CoinMarketCap 查看排名前 8, 850 个的 NFT 集合时，发现这些顶级 NFT 中有超过 1600 个已经死亡，蓝筹 NFT 也不例外。

从 dappGambl 统计来看，供大于求，只有 21% 的藏品有所属权，剩下的 79% 等待“有缘人”买单。

据 OpenSea 10 月 12 日数据显示，贾斯汀·比伯花费了当时超过 100 万美元购买了 Bored Ape #3001 ，如今跌至 4 万美元左右。

加密资产的价格会根据现实世界的事件和跨境技术因素而发生巨大变化，因此几乎不可能找出导致 NFT 崩盘的单一原因。

从现实世界的事件来看， 2022 年 2 月俄罗斯入侵乌克兰震惊了传统金融和加密货币市场，导致投资者变得更加恐惧和规避风险。

谈到加密经济，加密货币 Terra (LUNA) 尽管其市值排名前十，但还是在 2022 年 5 月崩盘。这损失了数十亿美元的价值，并引发了加密货币交易所和贷方的流动性紧缩。同年晚些时候，FTX 交易所倒闭，其首席执行官、亿万富翁 Sam Bankman-Fried 被捕。随着比特币在 2023 年触及多年低点，许多投资者因恐慌而抛售资产。

通货膨胀和科技行业的大规模裁员是可能降低人们对实验性投资产品兴趣的其他因素。

加密经济极其不稳定，因此很难预测 NFT 行业是否会复苏，更不用说回到牛市高点了。虽然比特币和以太坊价格上涨可能会给 NFT 带来帮助，但这两种加密货币的价格区间也远低于 2021 年的历史高点。
