声誉：通往卓越的许可证

曾在雷霆效力九年未曾问鼎总冠军的凯文·杜兰特（KD），在 2016 年作出惊人决定，离开雷霆，加盟了对手金州勇士。自此，他时常被称为“软蛋”，忠诚度也遭到质疑，不过，负面的声誉并没有跟随他太久。新到金州勇士的两年内他和队友们一起连夺两次 NBA 总冠军，并在比赛关键时刻脱颖而出，他夺得了 FMVP 的荣誉，从而开启了个人声誉修复之路。在球场上“大杀四方”后，KD 高调“代言”非洲裔美国球员，关注社会平等和种族公正，回馈社区，捐助青少年篮球培训项目。一时间 KD 迅速积累了极良好的声誉。这份声誉价值在商业和社会层面为他带来了巨大的利益：截止今年 8 月，KD 职业生涯一共赚到了 3.58 亿美元，NBA 历史排名第 3 名。在职业收入之外，杜兰特还有代言收入。他的影响力甚至延伸至商业领域，成立风险投资公司，成功投资了众多初创企业，包括加密货币交易平台 Coinbase。

声誉(Reputation）最早起源于经济学中激励代理人发挥积极性的隐性激励机制 （Holmstrom， 1999），代理人的诚信经营行为可以为其提供良好的声誉，有利于在下一轮博弈中获得更多的合作机会和更高的价格溢价。放在个人身上，则体现在人们通常会放弃短期的机会主义行为，维护良好的声誉以期获得更长远的利益。社会成员基于各种收集到的言行举止、社交媒体活动、专业成就等信号对某一个体进行评估，这构成个人声誉的形成过程。在这种情况下，信任、能力和认可是构成良好声誉的重要元素。

如何测度一个人的声誉好坏呢？一种方法是，将一个人的声誉分为认知声誉和情感声誉，认知声誉是利益相关者对个人行为的理解、判断和评价，情感声誉则是利益相关者对个人形象所持有的喜欢、尊重或厌恶等情结体验。在物理世界，一个人就是一个社会单位，但在链上，一个单位更多时候对应的是一个钱包地址。而一个人（一个身份），通常由一个或多个单位（钱包地址）组成。利益相关者对组成一个身份的一个或多个钱包地址的认知和情感声誉，才能组成某个身份的链上声誉。

Coinbase 创始人 Brian Armstrong 和 Base 生态基金都曾撰文公布自己看好的赛道或链上建设方向。他们认为，随着去中心化身份的出现，现在的问题是如何确定谁是值得信赖的，以及如何在链上建立个体的声誉。为此，他们提出了构建一个链上建立声誉协议的想法，认为这将在去中心化身份和链上实体的互联中发挥关键作用，为用户创造更多信任。

是你的敲门砖；

有助于人际关系的建立；

让你的想法更易于为他人接受；

赢得他人的友好和善意;

驱使你获得成功。

这些所有好处的合集提醒我们，良好的声誉虽然不是能够驱动你成功的唯一因素，但绝对是最重要的因素之一，因为它能够从很多方面帮你打造成功的职业生涯和获得良好的生活氛围。

具体到 Cypto 世界来说，个人的声誉分数还不像物理世界那样影响我们生活的方方面面（大到金融信用，小到向同桌借橡皮擦），但已经有许多声誉分数的真实用例发生或正在发生。

  • 例如 Optimism 发布的 Superchain Faucet，允许开发者利用链上声誉数据获取高达 20 倍的测试网 ETH；

  • 例如 Nomis 已经可以为自己平台的声誉分数提供回报。他们的 ScoreFront，是 Nomis 分数持有者利用其链上声誉的一站式平台。；

  • 例如 Degenscore 的 Cafe，有着 3 类根据 Degenscore 链上声誉衍生的权益，比如提前获得内测资格，获得产品的权益加成；在 Phi 和 Interface 获得相应的展示权益；寻求与你技能匹配的工作机会。

  • 例如 CreDA、SpectralFi、Repute_x、RociFi 等项目能给高声誉分数的用户提供提高杠杆率（贷款与抵押品），甚至无需超额抵押即可获得贷款的服务。有效的信用评分系统将提高 DeFi 市场的流动性，因为更多的用户将能够利用更少的抵押品参与借贷活动，这将极大的提高市场流动性，提高整个生态系统的效率。

我们还可以设想在声誉系统的引导阶段（初始阶段），某些声誉平台可以通过量化你为社区提供的贡献和影响来扩大或缩小你在治理决策中的发言权；你的链上声誉还能更快地为你匹配到对应的社交圈子；还有更直接的，某一种声誉证明被预备空投项目采用，作为他们分发奖励的定量标准。

总之，未来声誉系统的关键价值，可以帮助社区或者网络内的搜索与发现。随着引导阶段的成熟，用户可以开始利用声誉分数来寻找他们需要的项目，项目也可以通过用户的声誉分数为他们提供更匹配的价值和激励手段。未来链上声誉系统还可能会影响到传统金融：链下的商业实体或金融实体可以通过消费者的声誉分数来判断他的链上行为从而为其提供相应的服务。

声誉建设现状

如果你认同链上声誉系统建立的重要性，那么我们可以一起来探讨执行层面的一些问题。目前看来，在链上建设声誉的初期，我们很难找到一个既综合又准确的尺度，让所有人都执行一个统一的链上声誉标准，因为不同的用户和应用场景可能需要不同的衡量尺度。但我们可以通过用户、数据、基础设施和项目的相互作用关系来了解短期内比较有效率的声誉建设模型。

基础设施方面，以太坊构建的 EAS 使得任何实体都能够以数字签名的形式对各种事物进行 attestation。这对于在充满真实和虚假信息的在线和链上世界中建立对信息的信任至关重要。EAS 的设计理念强调了去中心化和开放的原则，确保协议层不受到单一利益相关者的操控。尽管 EAS 是一个基础层，但其灵活性使其可以为构建信任、验证和声誉系统提供全面的支持。 Op 的链上证明服务现在也在采用 EAS 了。Linea 构建的 Verax 在互操作性上有了进一步提升，可以帮助开发者发行与其他标准兼容的 attestations，从而促进更加统一的声誉评判标准的诞生。与此同时，Verax 允许多个声誉协议使用相同的 attestations，用户可以根据其需求和偏好选择适合自己的声誉协议，而无需担心“被锁定”到某一个协议。此外，通过 Verax，声誉协议能够更灵活地利用 attestations 来构建对用户、协议或 dapp 声誉的综合评估。这也显示出 Linea 在构建 Verax 之初就在主动吸引潜在的声誉建设项目。

在艺术品领域，了解一个人是谁以及对其真实记录的了解对于判断其品味和对艺术品的欣赏能力至关重要。但传统艺术品市场中的艺术品价值受到少数专家意见主导的情况。$RARE stake 呼吁用链上声誉系统重新构建传统艺术品市场，使之更加公平、透明，并给更多的个体提供展示和发展的机会。他们在自己的产品中引入了社区池和策划列表的概念，推动建立了一个让个人不仅能够在时尚领域建立无可争议的声誉，而且还能在Web3创作者经济中自我策划，展示自己的才华和品味创造能力的艺术平台。

对有致于获得项目对早期贡献者奖励的用户来说，钱包地址拥有良好的声誉从而避免被识别为 Sybil 攻击地址也是十分重要的。Arbitrum 进行空投快照前，联合 Nansan 制作了一个 An on-chain distribution model 用于 token 分配。其中，部分地址早期参与 Arbitrum 被认为不是出于实用目的，而是聚焦于获取空投。最终，Offchain Labs 研究人员通过在来自 Nansen Query 的 from_address / to_address 交易数据使用聚类算法，并在 Arbitrum 和 Ethereum 上整合跟踪和 Token 转移模型，识别可能的女巫钱包。在这个案例里，女巫地址和真实行为地址在链上可能表现了出完全相同的模式，但项目方根据动机论排除了女巫地址参与治理的可能性。这个时候，链上声誉就成为了你证明自己不是女巫地址的一个解决方案。好的链上声誉成为了一种强大的筛选工具，帮助用户向项目方确保自己真实性和可信度，从而间接为整个生态系统的安全性和可靠性提供的能量。

DeFi 市场是声誉建设短期内最有可能大规模被实践的领域。因为 DeFi 行业可以通过引入有效的链上声誉评分系统来改变现有的过度抵押模型。比如说，链上信用评分系统将允许 DeFi 平台更精确地评估用户的信用风险，而不仅仅是依赖于抵押物价值。用户还可以通过维护良好的信用历史可以获得更低的利率和更好的借款条件。

相似之处:

  • 都专注于在区块链生态中利用链上声誉为用户提供信用评级。

  • 提供链上身份和信用评级，为用户提供更多金融和社交机会。

不同之处:

  • CreDAfinance: 以 cNFT 形式创造用户信用档案，主要应用于解锁 DeFi 平台上的优惠和激励，以及降低借款利率。

  • Repute_x: 注重参考多方面的数据，建立全面的信用档案，适用于 Web3 生态系统中的个体，用于信任建设和金融服务。

  • RociFi: 引入 NFCS，着重于实现 DeFi 的信用、声誉和信任代币，用于提供用户可验证的身份和信用评级，适用于借贷、治理、信任体系建设等场景。

以上的项目都有一个共同特点，就是用分数对应用户的声誉价值，这和现实世界（链下）的信用评价体系如 FICO、Experian、Equifax、TransUnion 和芝麻信用等如出一辙。这里我们不禁要问，有没有更 crypto 的方式可以用于协助用户建立声誉呢？

我们可以考虑用 credentials 来代表声誉。怎么来判断一个地址背后的人的声誉如何呢？通常情况下，“声誉”两个字代表着我们在他人眼中的形象。既然如此，最为关键的问题也就是：“别人对我们的看法是基于什么？”而关于这个问题，简要的回答就是：“我们的行动，或者说我们的行为方式。”如果你认同你的行为可以影响你的声誉，那么你控制的钱包地址的链上行为（集合）也将代表这个地址的声誉。链上行为以数据形式体现的时候繁杂且抽象，而可升级的 credentials 则在一定程度上“标签化”了那些数据，让它们以更具象化的形式呈现出你的链上行为，从而帮助你积累链上声誉。

目前在 Base 上有这样一个用 credentials 代表用户声誉的项目。用户可以在 Metopia 上通过申请可升级、人类可读的凭证，快速发现自己在 Base 生态系统上积累的链上交互数据的价值，并附加声誉 XP 来量化你在 Base 链上所有交互行为的整体进展。这些凭证将用户的链上历史封装成人类可读的形式，以展示用户在整个生态系统中的参与和贡献。这种方法更注重协助用户体现自己个性化、动态变化的声誉，使用户能够更好地感知自己在链上的进展并获得社会认可。

积累声誉：贡献、持有和建设

个人声誉管理核心在于建立清晰的自我认知，即你希望你的链上形象是什么风格，人们看到的是你的什么特质。

首先，你需要设想自己在链上希望被他人感知的方式。这可能包括个人特质、职业能力、社交技能等方面。然后，相应的丰富或调整自己的行为和活动，确保它们与期望的形象相一致。比如近几日的热门 Celestia 空投 TIA 的案例。与重点关注为自己生态做贡献与否的 Arbitrum 不同，Celestia 关注了更复杂的数据，包括：

  •  多链交互数据监测： 开以太坊主网、Optimism、Arbitrum One 之间的交互数据。这包括链上的信息流动、交易互动等，可以更全面地了解用户在不同链上的活跃度和参与程度。

  • Gitcoin 捐款： Celestia Airdrop 也关注了参与者在 Gitcoin 上的捐款记录。这一考察点突显了社区参与和支持开发者的态度，也为社交信誉和贡献度提供了有力支持。

  • Ens 域名： 考察了用户在 Ens 域名系统中的活动，这反映了个体在去中心化网络中的身份认同和品牌建设，进一步强调了个人在区块链生态中的角色。

  • 地址年龄和活跃月数： Celestia Airdrop 采用了对地址的年龄和活跃月数进行快照的方式，提前 6-9 个月。这有助于识别长期参与者和活跃用户，为空投权重提供更为准确的数据支持。

这一系列考察内容表明，你在链上行为的成熟度和复杂度帮助你被 Celestia 项目方识别为“高价值用户”。所以要持续的贡献和建设，让自己的高价值行为为自己带来良好的声誉，从而获得认可和奖励。

其次，还需要深刻了解风险在哪里。“____在关心我的链上的____行为”，第一个空白处可以认为它是机构、雇主或是项目方，我们称之为个人的利益相关者。第二个空白处即是我们在链上的具体行为，比如交互、比较社交，还应包括是否有链上不良记录。理解这些利益相关者的期望和关切，可以帮助个人更有针对性地进行链上声誉管理。我们之前举例的项目方排除女巫攻击账号就是个人声誉面临风险的典型——被 HOP 视为女巫攻击的账号在一两年后还在饱尝第一步就被排除的痛苦。

此外，个人声誉的提高也需要规划。个人可以基于当前的声誉状况来规划如何进一步提升地址价值，这就涉及到一些系统性的实际工作的讨论，我们在这里可以提出一些方法论供大家参考。积累好的声誉，可以这几个方向去做：

1. 持有（Hold）： 长期参与并持有项目代币，展现对项目的信心和支持。这不仅有助于个人财务增值，也在社区中树立你对生态系统稳定性的承诺。

2. 贡献（Contribute）： 进一步细化贡献，包括时间、能力和金钱。包括但不限于捐赠，参与社区建设和管理。参与开源项目，提交代码，为社区提供有价值的工作（Build）。

3. 教育和分享： 如果你有专业知识，考虑写博客、制作教育性的内容，分享你对所在生态的见解和经验。参考 Starknet“早期社区成员计划”，它将奖励那些鼓励技术讨论、组织与 Starknet 相关的活动并定期发布 Starknet 品牌内容的个人贡献者。

4. 支持他人： 帮助解决他人的问题，提供有价值的建议，以友善和专业的态度获得社区的认可。

最后，我想说，声誉的力量源自信任。信任是关系的基本元素，但随着全球化和信息化的发展，中心化信任模式的风险变得更加明显，而 Crypto 世界正在试图通过去中心化的方式重新定义社会信任。声誉作为信任和责任相结合的概念仍然值得被高度重视，但也面临着新的具体挑战。希望我们能早日利用声誉在链上“创造” 新的信任，那意味着我们即将通过自主的联合来 “创造“新的社会。
