本文为大家介绍了 Aptos 上的订单簿式 DEX—Econia，其为 DeFi 提供高性能、原子结算和无许可全球访问的交易体验。它将流动性聚合到一个数字市场中，具有类似传统交易所的交易体验，同时实现了去中心化和用户资产保管的优势。Econia 的原子匹配引擎执行快速交易，减少延迟并降低 MEV 攻击风险。对于交易者、开发者和机构，Econia 提供了流动性、效率和创新的优势。

以下由Odaily星球日报编译。

Econia 是什么

Econia 是在 Aptos 区块链上运行的一个链上订单簿式 DEX。经过高性能的工程设计，原子结算，并建立在提供对市场的无许可全球访问的基础上，Econia 利用 Aptos 的乐观并发模型，为用户提供无缝高效的交易体验。

Econia 旨在将来自世界各地的流动性聚合到一个单一的数字市场中，它是 DeFi 的核心后端系统，对用户来说是不可见的，除非他们是 DeFi 开发人员。

为什么选择 Aptos？

Aptos 的乐观并发模型允许并行执行交易，使区块链能够每秒执行数万个的 Move 交易。Aptos 通过利用事务的预设顺序并结合软件事务内存（STM），使其块级 STM 执行模型可以在几分之一秒内处理大量的交易。

Econia Labs 的背景

Econia Labs 是由 2021 年首届 Aptos Hackathon 孵化而出，在该 Hackathon 中赢得了奖励。在 Hackathon 之后，Econia Labs 被选为最初的 Aptos 加速器计划队列，并获得了 650 万美元的种子轮投资，由 Dragonfly 领投，Lightspeed Faction，Wintermute Ventures，Hudson River Trading，Flow Traders，Aptos Labs 等参投。

Econia Labs 团队成员具有斯坦福大学，高盛，NASA，Garmin 和 The Block 等知名机构的多样化背景，专注于开发 Aptos 区块链的顶级链上订单簿。

为什么选择 Econia 而不是中心化交易所或 AMMs

与自动化做市商（AMMs）不同，订单簿提供类似于 Coinbase Pro 或 Binance 等传统交易场所的交易体验，为用户提供了从非保管钱包进行交易的权限。通过将中心化交易所（CEXs）的优势（高效的价格发现和最小滑点）与去中心化交易所（DEXs）的优势（去中心化，无需许可和用户资产保管）相结合，结果是一种将效率、公平性和去中心化结合在一起的交易体验。

Econia 的匹配引擎是原子的，意味着它在一次交易中执行了结算市场订单所需的所有步骤。例如，当用户决定以市场提供的最佳价格购买或出售数字资产时，Econia 将他们的订单与另一个市场参与者的限价订单（询价或出价）进行匹配，更新订单簿上的位置，并在各方之间传输资产，所有这些都在单个交易中完成。这种单步交易在没有 Aptos 区块链的乐观并发性的情况下是不可能的。

Econia 如何使交易者受益

订单簿是一个基础的金融原语，将流动性整合到一个全球性的场所，供所有市场参与者共享，无论他们如何集成。特别是对于 Econia 来说，其原子匹配引擎减少了计算开销，导致更快的交易执行。

Econia 并行处理和执行多个订单，减少系统开销，提供更快速响应的交易环境。Econia 同时处理多个订单的能力也意味着它可以处理更高交易量。由于 Econia 的匹配引擎在一次交易中执行了结算市场订单所需的所有步骤，因此交易速度比传统的去中心化交易所要快得多，因为在交易被认为是最终完成之前无需等待多个交易得到确认。

总体而言，Econia 的原子匹配引擎提供了高效的交易体验。用户可以有效地交易其资产，无需担心延迟或意外结果，同时减少 MEV 攻击的风险。

Econia 如何使开发者受益

构建和测试基础设施可能是一项艰巨的任务，但 Econia 的开源项目使加密领域的创新参与者能够构建、调整和扩展。Econia 的超并行订单簿执行和结算基础设施为关键合作伙伴提供了在 DeFi 领域的竞争优势。

Econia 的 API 和 SDK 提供直接访问 Econia 协议的途径，使几乎任何开发人员都可以创建、访问和构建在这个创新基础设施之上。Econia Labs 也正在最后确定一个开源的参考前端，不久之后，任何人都将能够 fork 它并运行自己的定制交易仪表板，为用户提供更多对 Econia 的访问。

Econia 如何使机构受益

由于 Econia 将流动性聚合到一个全球性的场所，机构可以在各个前端与来自零售领域各个层面的交易对手以极低的价差进行交易。此外，Econia 的基础设施是资产不可知的，允许机构交易像衍生品或期权这样在中心化交易所或传统市场上可能不可用的异类资产类别。

结论

Econia 是一个超并行的在 Aptos 区块链上运行的链上订单。相比中心化交易所和自动化做市商（AMMs）。Econia 为交易者、开发者和机构提供优势，DeFi 领域的关键参与者可以从 Econia 提供的流动性、效率和创新中受益。
