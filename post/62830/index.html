2023 年 11 月 6 日，zCloak Network 最新的全链 KYC 产品Legit ID (legit 3.id)正式上线。zCloak 是一个隐私优先的去中心化数字身份与可信凭证基础设施，旨在Web3时代建立新型的身份数据体系和信任传递范式。Legit ID 的出现将重塑数字资产及数字身份领域的行业格局。

zCloak Network 的创始人张博士表示：“在保持Web3去中心化和开放性内核的同时，必要的监管与合规也至关重要。我们就像是狂野西部的探索者，准备通过技术手段为那片‘土地’带来法律和秩序。” 回顾Web3行业的发展历程，区块链技术一直以其开放性和匿名性而闻名，众多 DeFi 协议及 RWA 资产上链的出现为金融领域带来了深刻的变革。随之而来的是一系列艰巨的挑战，目前行业面临的最棘手和最严峻的问题莫过于监管与合规的问题。

探寻数字领域中的监管与合规问题

在数字世界中，全球监管与合规法则的存在可以有效抑制与打击非法金融活动，保护消费者和投资者。而区块链去中心化、无需许可的特性必然会引发普通用户对于资产安全、数据隐私、信息管辖权等方面的担忧。

鉴于Web3生态对监管需求的不断增长，政府和各大监管机构一直在寻找解决方案，试图找到一种既符合Web3的去中心化特性又可靠的监管方案。随着 RWA 代币化的兴起，KYC（Know Your Customer）和反洗钱（AML）等解决方案逐渐受到更多关注，而行业也普遍认为这是链上应用合规的必经之路。虽然 KYC 和 AML 对于验证合法交易及阻止非法链上活动成效卓著，但传统 KYC 在Web3的推行仍有很大阻力，我们认为其主要原因是传统 KYC 方案侵犯了Web3用户习以为常的全匿名和隐私保密原则。

而这正意味着我们需要一种既能符合全球监管要求，又能保护用户隐私的全新 KYC 解决方案，所以 zCloak Network 带来了 Legit ID。Legit ID 是 zCloak 多年在零知识证明和 DID 领域深耕的产物。“一次 KYC，全链通用”这句话可以完美总结 Legit ID 的核心特点。它只需要用户进行一次 KYC 验证，就可以在全球范围内进行全链 KYC 证明。在分享身份信息的同时，充分践行了Web3倡导的个人数据主权的理念。基于 zCloak 的长期技术积累，Legit ID 利用零知识证明技术和去中心化身份系统，实现了以安全、隐私为核心导向的 KYC 服务。

Legit ID 的核心优势

首先，Legit ID 的一个显著特点是其全链可用性。Legit ID 使用 W 3 C 标准下的 DID 和 VC 协议将 KYC 结果数据存储在链下，使用独特的自验证数据上链技术取代预言机，可以在保证数据真实的情况下轻松与跨多链的智能合约进行数据共享，包括以太坊、Solana、Sui 以及任何 layer 2/3 公链。用户不但在真正意义上获得了自己身份数据的主权，且无需执行传统的消息跨链操作，就可以灵活地使用自己的 KYC 验证结果在协议层驱动智能合约开展业务。

其次，Legit ID 可以在协议层提供用户地址的实时 AML 数据更新。众所周知，身份合规监测并非一劳永逸的操作。今天合规的地址，明天就可能参与非法金融活动。因此只是把 KYC 结果上链是远远不够的，还需要配合动态的 AML/CTF 监测来进一步降低风险。Legit ID 与链上数据安全项目 Chaintool 进行合作，可以为交易各方提供秒级 AML 告警，旨在保护企业、投资者和广大个人用户免受非法地址的威胁。

此外，Legit ID 还支持项目方自定义身份信息验证内容。第三方智能合约拿到的用户信息可以是简单的 KYC 检查通过/未通过状态、反洗钱风险评分，也可以是更复杂的基于零知识证明的身份信息。Legit ID 可以在完全保护用户隐私的情况下，向智能合约传递包括所在地范围、年龄区间、收入水平、信用等级、风险偏好等多种要素在内的用户身份证明。证明结果的丰富和详细程度可以根据每个验证者的具体需要进行快速定制。

最后，Legit ID 为企业提供了一个低成本，易集成的用户真实身份验证解决方案。Legit ID 产品使用订阅制，其使用成本远低于项目方自行联系 KYC 公司进行身份验证工作的成本，在集成时间和商务流程上更有巨大的优势。项目方无需修改任何已有的业务逻辑，只需要访问 Legit ID 部署的合约就可以获得完整的用户身份信息。

综上所述，Legit ID 是一个创新的身份验证平台，提供了跨多个区块链平台的全链可用性，以及实时的 AML 监控和告警系统，以增强企业和用户的合规性和安全性。除了金融领域的监管与合规用例，Legit ID 还可以在链上反女巫、DAO 治理、人格证明等场景发挥巨大的作用。
