最近，Avalanche 代币 AVAX 走势强劲，从 11 月初 11.3 USDT 飙升至最高 24 USDT，涨幅超过 100% 。Odaily星球日报甄选编译了 Avalanche 第三季度报告，帮助读者更好地了解该项目过去一段时间的动向，提升投资决策的可靠性。以下为原文编译：

概述

Avalanche 是一个专为可扩展性而设计的公链平台，具有由可扩展网络的子网组成的多链框架，主网包括 P 链、X 链和 C 链：P 链管理验证器和子网级功能；X 链管理 Avalanche 原生代币（现实世界资产的数字表示）；C 链是实现以太坊虚拟机（EVM）。

Avalanche 上的子网是指一组节点或验证者的子集，他们一起就有关一个或多个区块链的交易达成共识。子网可以是在另一个网络中运行的网络，也可以是构建在较大网络之上的网络。Avalanche 的子网架构为构建者提供了快速终结性（Fast Finality）、自定义虚拟机支持和广泛的配置参数。（Odaily 注：Fast Finality，即所有交易都会被快速打包，并且无法撤销，更改。）

2023 年第三季度，通过与 DeFi、NFT、游戏和企业领域达成各种合作伙伴关系以及相应举措，Avalanche 的生态系统变得更加丰富，展现了传统金融与 DeFi、游戏创新和企业区块链解决方案进行融合的潜力。

主要进展： 2023 年第三季度

  • RedStone与 Avalanche Evergreen 合作，通过 Avalanche Spruce 测试网上的定制数据源促进机构参与去中心化协议。Avalanche 的 Evergreen 子网提供专业的区块链部署，将公共网络的优势与包括 KYC 管理在内的企业功能融合在一起。Spruce 测试网为传统金融机构探索区块链和 DeFi 应用提供了一个安全的环境，弥合了传统金融和 DeFi 之间的差距。

  • Avalanche 基金会推出了“ Avalanche Vista ”，拨出最多 5000 万美元购买其区块链上的代币化资产，以突显资产代币化的价值，并提升其在链上金融中的作用。该倡议将包括各种类型的资产，旨在展示区块链在传统手动金融流程中的效率，并通过使用链上资产发行和管理，来解决私募股权投资中的挑战。

  • Avalanche 基金会与 Sigma Prime合作，增强其 AvalancheGo 代码库的稳定性和安全性。Sigma Prime 以在区块链安全方面的丰富专业知识和开发以太坊共识客户端 Lighthouse 而闻名。它正在创建 fuzzer 来识别和解决 AvalancheGo 中的潜在错误，从而增强网络可靠性，并确保用户和开发人员的稳定性。

  • Movement Labs 筹集了 340 万美元的 Pre-Seed 融资，以扩大 Move（一种智能合约语言）的开发和采用，并推出 Movement SDK 和 M1 ，这是一个与 Avalanche 共识集成的可扩展区块链。该筹资旨在增强 Move 在智能合约开发中的安全性和可扩展性，提供针对常见区块链问题（例如重入攻击）的解决方案，同时通过 M1 提供面向社区的区块链。

  • Ava Labs 推出了新的 C-Chain Explorer，将所有 Avalanche 主网链（P、X 和 C 链）整合到 Avalanche 子网浏览器中。这个平台通过提供全面的实时和历史区块链数据视图，加强了 Avalanche 的链上透明度。C-Chain Explorer 提供各种高级功能和搜索选项，包括与其他链的协调交易信息、质押和委托详细信息、原子交易细节、以及 Glacier API 和 Metrics API 等。

  • Avalanche 推出了 Avalanche Academy，旨在教育开发者创建顶尖的区块链应用程序。课程包括 Avalanche 基础知识、EVM 调整、子网结构和开发工具的，毕业生完成课程后将获得毕业证书。

  • Avalanche 基金会推出了“Ted Yin”资助计划，以支持和资助利用 Avalanche 公共区块链的开源项目。该倡议旨在表彰计算机科学家 Ted Yin 对 Avalanche 社区的重大贡献。

  • Ava Labs 推出了 Firewood，这是一个用 Rust 设计的顶尖数据库，用于优化 Merkle 化区块链状态的存储。Firewood 解决了区块链可扩展性中高效状态管理的主要瓶颈问题，提供了一个专门的存储堆栈，超越了通用的键值存储，如 LevelDB/RocksDB。Firewood 专为高效的 Merkle 化区块链状态存储而设计，承诺在 Avalanche 网络中提供增强的可扩展性，并邀请贡献者参与其进一步发展。

生态系统

Avalanche 在其 C 链和子网上托管着一个由 dApp 和协议组成的大型生态系统，用户可以在 Avalanche生态系统页面上进行探索。在 2023 年第三季度，Avalanche 生态系统项目和协议有许多令人兴奋的更新，特别是：

DeFi

  • 通过 LayerZero 实现以太坊和 Avalanche 之间的互操作性，Uniswap扩展到 Avalanche C 链，增强了其 DEX 生态系统。此举将 Uniswap 的大量交易量和创新的集中流动性模型带到了 Avalanche，为平台内不断增长的 DeFi 空间的流动性提供了新的机会，为流动性提供者和交易者带来了更多的选择。

  • Balancer 在 Avalanche 上部署，旨在通过与四个 Avalanche Liquid Staked Token（LST）协议合作，增强流动性抵押增长，并扩大 DeFi 机会。利用其创新的 Composable Stable Pools 和 Boosted Pools，Balancer 旨在高效管理生息资产，并将流动性池与收益市场相结合，从而为 Avalanche 参与者优化 DeFi 技术堆栈。

  • Struct Finance在 Avalanche C-Chain 上推出，提供利息金库和多样化的收益策略。Struct 旨在实现金融产品民主化，推动 Avalanche 的 DeFi 生态系统发展。流动性提供者可以在不同的波动阶段选择固定收益率和可变收益率。

  • GMX V2 测试版 在 Arbitrum 和 Avalanche 上推出，为交易者提供新功能，如更多可用资产、更多抵押品类型、更快的执行和低费用互换。流动性提供者从隔离池以及增强的平衡和互换激励中获益。

  • Avalanche 基金会向 Avalanche 子网（Dexalot Subnet）上的去中心化交易所Dexalot分配了高达 300 万美元的 AVAX 代币，以增强其去中心化交易能力。该资金是 Avalanche Multiverse 计划的一部分并将在 12 个月内分配，同时这一计划具有里程碑意义。

  • 由 CavalRe 开发的 Multiswap 已在 Avalanche 上推出，使用户可以在一个交易中换取多种代币，并使用任意数量的代币创建所有资产池。它允许流动性提供者接触到所有资产池，并可能从交易费中获得收益，类似于去中心化的 ETF。

  • Cerchia 利用 Avalanche 区块链，开创了直接的点对点保险风险交易，允许保险买家直接从机构投资者那里获得风暴损失保护。该方法使用智能合约进行自动支付，并使用 Backed 的代币化美国国债券 ETF 作为产生收益的抵押品。

NFT 和游戏

  • HUG 和 Avalanche 基金会的 Avaissance 合作，通过每周艺术挑战和在韩国区块链周的曝光，以使新兴艺术家为人所认知。同时 Avaissance 艺术家加入了 HUG 平台，这使得他们与全球收藏家社区能够联系起来。

  • Solert Games 和 Ava Labs 合作创建了一个新的 Avalanche 子网，通过融合 Web2 和 Web3 功能增强游戏性，并推出了手机游戏 Legends at War。通过其燃料代币 LAW，Solert 子网旨在为开发者提供全球平台和创新的游戏生态系统，且玩家可以从中影响治理和战略。

  • Rumble Kong League (RKL)选择 Avalanche 作为其区块链扩展，优先考虑用户友好的Web3游戏体验。RKL 将逐步过渡到 Avalanche 以确保无缝的玩家体验，并使得玩家专注于游戏内的操作而非底层技术方面。尽管未来所有交易和交互将在 Avalanche 上进行，RKL 依然保持对各种区块链的包容性，并将其中长期目标集中在 Avalanche 网络上。

  • Neowiz，一家著名的韩国游戏发行商与 Avalanche Arcad 3 联手，通过其Web3部门 Intella X 探索Web3游戏。这项合作在 Avalanche House Seoul 2023 上宣布，旨在利用 Neowiz 的游戏经验和 Avalanche 的Web3能力，增强Web3游戏行业。此次合作的重点是分享 Web3 游戏技术专业知识，希望为用户提供顶级的 Web3 游戏。

  • Hyperspace 在 Avalanche 上推出了 NFT 市场和发行平台，为 NFT 收藏者引入了实时交易和分析等功能，该平台还计划在未来推出更多功能和 API，并通过其发行平台推出备受瞩目的 Dokyo 项目。这一发展标志着 Hyperspace 在多链旅程中迈出了其重要一步，并同时增强了 Avalanche 的 NFT 生态系统。

  • Dreamus 已将 Avalanche 区块链集成到其流行的票务应用程序 OK Cashbag 中，以提升其 2100 万用户的购票体验。此举旨在解决韩国音乐会门票行业中的票务黄牛和伪造问题。链上门票由 Avalanche 专用子网支持，现已可用于各种活动，同时 Dreamus 计划在未来发布基于区块链的二级市场。

企业

  • 西班牙医疗保健 REIT Manicipi SA 计划通过 Securitize 发行代币化股票筹集 1.5 亿欧元。Securitize 计划于 9 月开始交易，并在西班牙国家证券市场委员会（CNMV）监管的监管沙盒内运营。他们正在积极努力获取西班牙和欧盟的证券许可证，与基于区块链的 RWA 代币化趋势保持一致，正如 Avalanche 为代币化资产发行者设立的 5000 万美元基金举措所体现的那样。

  • Loyalty Marketing 与 PlayThink 合作，通过 Avalanche 的子网基础设施将其“Ponta”奖励计划引入区块链，旨在到 2023 年底为近 1 亿用户提供Web3服务。该计划将利用 Avalanche 的智能合约能力和快速交易的确定性，大规模进行 NFT 发行和分发。该子网将促进用户参与基于 NFT 的奖励和活动，而无需获取数字资产。

C 链日常交易

来源： Avalanche C 链每日交易

2023 年第三季度，Avalanche C 链的每日交易数量波动，范围在 13.6 万至 50.4 万笔交易之间。这意味着日交易量较上一季度的 20 万笔小幅下降至 55 万笔。与此同时，网络内的 Gas 使用量也呈现出类似的趋势，在 8.7 k 至 49 k 之间波动。

C 链每日活跃地址

来源： Avalanche C 链每日活跃地址

2023 年第三季度，Avalanche 的 C 链日活跃地址峰值达到 8.6 万，该数字在 2.7 万至 8.6 万之间波动。该网络整个季度的用户活动呈下降趋势。相比之下，第二季度的日活跃地址在 2.2 万至最高 11.7 万之间。

平均每日 Gas 费（与以太坊相比）

来源： 以太坊与 Avalanche C 链的平均每日 Gas 费用

2023 年第三季度，Avalanche 的平均每日 Gas 费在 0.04 美元到 0.17 美元之间波动，与以太坊形成鲜明对比，以太坊用户遇到的 Gas 费更高，从 1.95 美元到最高 15 美元不等。Gas 费突然飙升可能是由 8 月 18 日的重大清算事件引起的。

为了了解 2023 年第三季度主要参与者的情况，Nansen 的分析聚焦于值得注意的 Avalanche 的交易和用户活动参与者。LayerZero 以 721 万笔交易占据主导地位，其次是 Stargate 和 USDC，分别记录了 214 万笔和 167 万笔交易。Stargate 在用户参与度方面处于领先地位，拥有 69.6 万用户，而 Tether 和 USDC 紧随其后，分别拥有 69.3 万和 53.1 万用户。

总结

总的来说，Avalanche 的 2023 年第三季度展示了一个充满活力的生态系统，在波动的市场行情中游刃有余，并相对于以太坊维持了较低的 Gas 费用。Avalanche 的战略合作和创新跨足各个领域，包括 DeFi、NFT 和游戏，同时为其生态系统增添活力。此外，链上数据揭示了关键的参与者动态和用户参与。

展望未来，这些发展和其背后的见解凸显了 Avalanche 进一步优化其平台、提升用户体验并巩固其在区块链领域地位的潜力——这条公链正引领着一个无缝融合传统金融、DeFi 和技术生态系统的未来。
