在阅读 X 上的众多讨论后，我发现似乎许多人都忽视了以太坊与 Solana 之争背后的核心。

这不仅仅是以太坊与 Solana 的比较，而是我们如何扩展区块链以实现大规模采用。直接跳到结论，我相信一切都还未成定局，而且这些问题并不像看起来那么简单。这太好了，因为我们还处于早期。

这里的困境不仅仅是技术上的，还与文化价值有关。

非以太坊 L1 有什么意义？

为了了解以太坊与 Solana 争论的背景，我想简要回顾一下其他主要 L1 以及它们的价值主张。这将帮助我们根据扩容解决方案构建 L1 图景。

最近，我做了 6 场 AMA，重点关注 L1 和 zkSync，探讨 L1 在 L2 时代的作用。

讨论的主要问题是当以太坊的 L2 可以通过快速交易和更低的费用扩展以太坊时，alt-L1 还有什么意义。

不出意外，分歧最大的地方在于以太坊是否已经赢得了 L1 游戏。Polygon 的 Sanket 和 zkSync 的 Alex 相信以太坊在很大程度上赢得了 L1 游戏，但 Solana、Avalanche 和 BNB Chain 团队并不同意这种看法。

NEAR 的立场很微妙：以太坊将继续占据主导地位，但其他 Layer 1 解决方案将在以太坊不擅长的领域蓬勃发展。NEAR 的价值主张不太清晰，但他们最近与 Polygon 合作在数据可用性层上扩展以太坊，这与上述观点一致。

Bankless 在他们的播客中表示，NEAR 正在迁移到以太坊 Layer 2 解决方案，但这并不准确。NEAR 作为一个单体区块链存在，同时还通过 NEAR 的数据可用性和存储功能来帮助以太坊实现可扩展性。

事实上，NEAR 还可以为 L2 提供快速最终确定性，其中 Arbitrum 在 NEAR 上结算，然后到以太坊，同时为 L2 提供以太坊 + Eigenlayer 级安全保证和去中心化排序器。

Eigenlayer 质押者正在重新执行交易，以确保排序器是诚实的

Avalanche 模块化设计的核心主张是通过子网进行扩展，任何人都可以无缝构建自己的区块链。从架构上来说，Avalanche 介于 Cosmos 和 Polkadot 之间，具有 P-Chain（进行验证）、C-Chain（智能合约）和 X-Asset（用于发送和接收资金）。你可以在我与 Avalanche 的 DeFi/DevRel 负责人 Luigi 的谈话中详细了解它的工作原理。

BNB Chain 的推出是为了解决以太坊高昂 Gas 费用的问题。但即使 Gas 费低于 0.5 美元（并且在去中心化方面做出了妥协），BSC 对于高频交易等许多用例来说仍然不够便宜。为了解决这个问题，它们在 BSC 和 Greenfield 上推出了 opBNB L2 以实现数据可用性。

更多内容请参阅我之前与 BNB 核心开发团队合作撰写的博客文章。

这些扩展解决方案听起来很复杂吗？欢迎来到模块化区块链的世界。这是以太坊与 Solana 辩论的核心，我真正关心正是这一点。

这里的假设是，单个区块链需要在区块链技术的三个关键方面之间进行妥协：安全性、可扩展性和去中心化。区块链只能满足其中两个，而以太坊专注于安全性和去中心化，导致高昂的 Gas 费和缓慢的交易速度。因此，它需要将扩展外包给执行层和数据可用性层。

更多相关内容请参见下面我的 X 线程。

然而，雄心勃勃的 Solana 希望通过单一区块链设计解决三难困境。这可能吗？

保卫 Solana

Solana 的愿景是将廉价且快速的交易、快速最终确定和低延迟等所有优势汇聚到一起。实际上，这消除了对繁琐跨链桥的需要，而成本只是目前最好的 Layer 2 解决方案的一小部分。

Arbitrum 上的单次 Swap 交易 Gas 消耗约 0.69 美元（在撰写本文时），这仍旧不尽人意……我们需要 L3！

L2 Gas 费用

现在，所有这些优势都需要强大的硬件，这使得运行验证器变得昂贵。事实上，由于开发成本高昂，Lido 甚至已经退出了 stSOL 质押。

此外，对于 SOL 代币分配也存在批评，其中 48% 分配给了内部人士和风险投资人。最后，Solana 缺少第二个生产客户端（Solana 中仅存在一个区块链软件客户端，这意味着破坏区块链只需要修改一个程序）。

由于上述功能，以太坊社区（请注意，我的大部分投资组合都在 ETH 中）声称 Solana 并不像以太坊那样去中心化。这是一个敏感话题，但我目前的想法如下：

  • 以太坊 L2 也不是真正去中心化的（排序器中心化）；

  • 硬件价格会不断下降，从长远来看有利于 Solana；

  • Solana 将推出 Firedancer ，这是第二个去中心化网络的验证器；

  • 人们可以说，以太坊扩容的未来掌握在为 L2、DA 协议等提供资金的风投手中。

我在这里扮演魔鬼代言人的角色。

事实上，根据中本聪系数，Solana 的得分高于以太坊。它代表需要多少实体来控制区块链 33% 的质押份额，体现了区块链的去中心化和安全性。值越高意味着网络对抗操纵的弹性越大。Solana 得分为 22 ，而以太坊得分仅为 2 。

然而，去中心化是一个范围，如果你真的关心去中心化，你应该购买并持有比特币。这是针对以太坊、法定货币或其他黑天鹅事件的保险。

实践中，我认为 DeFi 最重要的一个属性是自我托管。我希望 Solana 基金会和政府都不能扣押或冻结我的资金。实际上，我对在 Solana 上持有 SOL 比在 Coinbase 的 Base 上持有 ETH 更有信心（这与价格无关）。不幸的是，以太坊上的高额 Gas 费将人们推向安全性和去中心化受到损害的 L2。

比特币上的 BTC > 以太坊上的 ETH > Solana 上的 SOL > Coinbase 上的 ETH

更重要的是，关于权力下放的讨论由西方国家主导。然而，Tron 和 BSC 等平台在非西方国家的成功，甚至社交网络 Farcaster 在西方的成功表明，区块链的用例需要足够的去中心化。不是每个人都能负担得起 18 美元的 swap 费用，你知道吗？以太坊和 L2 目前是富人的游戏。

无论如何…我希望看到区块链的用例多样化。

尽管我对在以太坊主网上使用 DeFi 进行借贷和质押大笔资金或持有昂贵的 NFT 更有信心，但我更高兴在 Solana 上进行交易并将其用于小额或非金融交易。

Solana 似乎非常适合大多数 GameFi、Metaverse、P2E、衍生品、期权等 DApp。这正是 Solana 推出配备有 DApp 商店的 Solana Phone 的原因。STEPN 就是一个很好的例子，我渴望看到更多面向消费者的应用程序在 Solana 上推出，以验证我关于 Solana 的观点。

这也是为什么 TVL 不是 Solana 生态系统的完美指标。以太坊拥有最多的资产，因为它是价值存储区块链。但这些资产被动地存在于产生收益的智能合约中，而 Solana 则强调资本流通速度——分配到每美元 TVL 的交易量。

凭借其高可扩展性和低交易费用，Solana 能够实现快速的价值流动和资本周转。这种资本流动速度证明了 Solana 基础设施对用户和投资者的吸引力。
- Michelle，Solana 基金会 DeFi BD 

模块化与单体式区块链的未来：为什么不能两者兼得？

公正地说，Solana 并不是唯一的整体式区块链。NEAR、Algorand 是另外两个例子，Fantom 正在转向单体式扩容设计。

但事情变得更有趣了，因为 NEAR 是一个单体式和模块化的区块链，为以太坊 L2 提供数据可用性。Algorand 拥有用于私有区块链的 Co-chains。

上面对两极框架的概述过于简单，我建议关注 Justin，他在这个问题上分享了很好的观点。他赞成执行分片或采用 Rollup 来进行扩展。

Tyler Reynolds 等其他人则更相信单体式区块链是正确答案。

但正如你所看到的，扩展解决方案有很多种，社区对于哪种方案占上风仍存在分歧。所以，我的主要观点是：

扩展区块链的方法有很多种，每种方法都有自己的利弊。然而，我是一个务实的人，希望通过投资可能在不久的将来占据主导地位并具有更高上升空间的解决方案来最大化我的回报。

尤其是在当前的环境下，以太坊在繁忙时段的交易费用令人愤慨，而 L2 的使用成本并不便宜，这给用户体验带来了负面影响。我希望以太坊社区在不久的将来能够抽象用户体验。

因此，我不能忽视 Solana 通过单体式设计进行扩展的愿景的潜在优势。如果他们成功吸引更多的开发者、DApp 和用户，那么增长的机会就不容忽视。

如果你阅读我的博客，你就会知道我关注的生态系统具有 1）技术创新；2）造富机会；3）引人入胜的故事。

Solana 满足所有三个标准。它提供了独特的扩展愿景，成功地吸引了一大批信徒和反对者。仇恨是衡量关注度的有用指标，因为它表明即使是怀疑论者也关心 Solana。

最后，Solana 的生态系统在熊市期间被消灭，甚至像 DeGods 和 yOOts 这样的 NFT 项目也离开了 Solana。这就像一个饱受战争蹂躏的国家，目前正在重建。这提供了机会，因为只有少数 DApp 能够吸引用户 1) 注意力；2）资金流入。你可以在 Jacob 的这篇文章中看到项目列表。

在不断争论 Windows 与 Mac、Android 与 iOS 的世界中，我们很容易陷入两极分化的讨论之中。但作为投资者，问题不应该是选择其中之一。相反，我们可以分配投资组合的一部分，以达到舒适的平衡，其中也包括其他 L1。

就我个人而言，我认为 Solana 的愿景足够有吸引力，值得进行一些配置。采取极端立场违背了我保持开放心态并在新信息出现时不断重新评估自己观点的信念。

我也关心实践方法，我会先尝试一下 Solana，而不是仅仅根据那些不喜欢它的人的意见而放弃它。

我非常看好以太坊，大多数人可能也是如此。这就是为什么我甚至省去了「捍卫以太坊」的部分，因为以太坊的价值主张很强大。但这并不意味着我不会考虑投资其他 Layer 1 解决方案。
