作者：Darren Kleine，Blockworks；编译：松雪，金色财经

苹果与安卓。 Windows 与 UNIX。 Mert Mumtaz 表示，它们是经典技术竞争的例子，在很多方面类似于 Solana 集成生态系统与以太坊模块化世界之间的区块链冲突。

在Mumtaz记忆中的过去的任何自由市场科技竞争中，领导者并没有长期保持总体市场优势。

“我从未见过一项技术占据了 95% 的市场份额，人们押注它会在未来五到十年内保持增长（就相对市场份额而言）。”

“即使是 Windows 也没有保持这种主导地位很长时间，”他说。

然而，他在 Lightspeed 播客（Spotify/Apple）上表示，人们似乎认为 EVM（即以太坊虚拟机）将永远保持其对竞争对手 95% 的主导地位。

Helius 首席执行官“百分百确信”，Solana 虚拟机 (SVM) 和 MoveVM 等 EVM 竞争对手“指数极低，而且市场不对称”。

“在十年的时间里，5% 的市场份额不会是 SVM 和 Move 的结合，而 95% 的市场份额永远是 EVM，”他说。 “那只是从未发生过。”

还没到宣布胜利的时候

Mumtaz 表示，他认为 SVM 特别受欢迎，并且正在进行广泛的开发， 包括来自 Eclipse、Firedancer 甚至 Visa 等实体的努力。

“这些事情会慢慢地向你袭来，直到有一天你会说，‘等一下。 实际上这里发生了很多事情。 发生了什么？’”

Mumtaz 声称，SVM 和 MoveVM“客观上是比 EVM 更好的技术” ，他说，现在“它们也通过区块链的模块化理论进行分发。”

Mumtaz还指出，市场通常忽视了像Solana这样的“集成”链实际上在很多方面仍然是有效的模块化。

“他们有不同的组件，内部有不同的核心团队负责处理。 它只是全部集成到一个消息总线中，”他说。 “它仍然是模块化的。 只是，他们不会外包某些东西。”

他说，正如软件历史上一直存在的情况一样，集成和模块化的方法都必须面临一些权衡。 “历史上的集成技术已经显示出非常高的成功率，”他说。 “所以还有一些模块化堆栈。 也就是说，它们都有效。”

Mumtaz 批评了 Layer-2 已经“赢得”区块链扩展解决方案的说法。 “他们赢得了什么？ 我们有大概五个链上用户。”他笑着说。

“我不确定现在是时候宣布胜利了。 让我们继续努力吧。”

Mumtaz 解释说，Solana 的一个“奇怪的讽刺”是，作为一个全球状态机，它能够支持“不可组合的不同用例”。

“也许其中会形成不同的社区，这实际上可能最终会帮助你实现模块化。” 他提到了状态压缩的例子，这是 Solana 的一项创新，源于对 NFT 铸造成本高的担忧。

他表示状态压缩技术是“基础设施的核心部分，基本上被纳入 RPC 规范本身”。 Mumtaz 表示**，由于系统更加集成，状态压缩等创新可以快速添加到链中。** “通信开销更少。”

Mumtaz 表示，一些模块化支持者批评 SVM 据称无法“与堆栈的不同部分并行创新”。

“我知道情况并非如此，”他说，“Helius 是一家基础设施公司，我们与其他正在做这件事的团队合作。”
