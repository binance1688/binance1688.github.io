作者：Shogun Saski 来源：medium 翻译：善欧巴，金色财经

先前被宣告死亡的非同质化代币（NFT）系列如今已经回归。有趣的是，它们现在能够展现出“再度繁荣”的迹象，这是许多交易者和加密专家认为不可能的事情。以下是本周一些最畅销的NFT系列的列表：

1.Bored Ape Boat Club NFT

11 月第二周最畅销的 NFT 系列是 Bored Ape Yacht Club，这是数字资产公司 Yuga Labs 推出的系列，其中包含固定的 10,000 个 NFT。NFT 集合显示最后一天的交易量为 1,876 ETH。Bored Ape Yacht Club NFT 系列的市值为 6.01 亿美元，底价为 60,134 美元。

2. Captainz

NFT 复兴期间第二畅销的 NFT 系列是 The Captainz，这是一个来自专注于 web3 的初创工作室 Memeland 的 NFT 系列，在以太坊网络上存储了 9999 个 NFT 的固定供应量。Captainz 在最后一天的交易量中提取了 530 ETH。其底价为9,589美元，市值为9500万美元。

3. Mutant Ape Yacht Club NFT

Mutant Ape Yacht Club 是数字资产公司 Yuga Labs 的另一个 NFT 系列，在以太坊网络上托管了限量 20,000 个 NFT，是本周交易量第三大的 NFT 系列。在过去的24小时内，NFT集合的交易量已达到483 ETH。它的市值为 2.2 亿美元，底价为 11,267 美元。

4. Mad Lass NFT 系列

Mad Lads 是数字资产公司 Backpack 的 NFT 系列，在 Solana 网络上托管，限量供应 10,000 个 NFT，是本周交易量第四大的 NFT 系列。在过去的24小时内，Mad Lads NFT集合的交易量已达到12,288 SOL。它的市值为 5,300 万美元，底价为 5,330 美元。

5. CryptoPunks 的 NFT 系列

本周交易量第五大的 NFT 系列是 CryptoPunks，这是一个 NFT 系列，之前由数字资产公司 Larva Labs 所有，并在以太坊网络上托管了一组固定的 10,000 个 NFT。过去 24 小时内，CryptoPunks 上记录了价值 366 ETH 的交易额。它的市场估值为11亿美元，底价为115,188美元。

6. Moonbirds NFT 集合

本周交易量第六大的 NFT 系列是 Moonbirds，这是数字资产公司 Proof Collective 推出的 NFT 系列，其中包括以太坊网络上托管的 10,000 个 NFT 限制系列。Moonbirds NFT 系列在过去一天的交易量为 234 ETH。其底价为 4,107 美元，市值为 4100 万美元。

7.蓝筹Azuki NFT系列

本周第七畅销的 NFT 系列是 Azuki，这是数字资产公司 Chiru Labs 推出的 NFT 系列，其中包括存储在以太坊网络上的 10,000 个 NFT 的有限选择。近24小时Azuki NFT集合的交易量已达到227 ETH，较前一天增长0.2%。其底价为12,178美元，市值为1.22亿美元。

8. Pudgy Penguins

本周交易量第八大的 NFT 系列是 Pudgy Penguins，这是加密货币企业家 Luca Netz 及其团队的 NFT 系列，由存储在以太坊网络上的 8888 个 NFT 组成。Pudgy Penguins NFT 集合在过去 24 小时内产生了 184 ETH 的交易量。其市值为 9200 万美元，交易价格为 10,381 美元。

9. CryptoPunks NFT 系列即将上市

Wrapped CryptoPunks 托管在以太坊网络上，是一个 NFT 集合，其中包括采用 ERC-20 代币标准包装的原始 CryptoPunks NFT。Wrapped CryptoPunks 系列在最后一天的交易量为 175 ETH，是交易量第七大的 NFT 系列。它的市场估值为9800万美元，底价为120,785美元。

10.DeGods 的 NFT 收藏

DeGods 是由数字资产孵化公司 Dust Labs 创建的 NFT 集合，在以太坊网络上托管着固定的 10,000 个 NFT。本周，DeGods 是交易量第六大的 NFT 系列。DeGods 已经关闭了本周最畅销的 10 个 NFT 列表，过去 24 小时内的交易量为 145 ETH。底价为 6,593 美元，其市值为 5,900 万美元。
