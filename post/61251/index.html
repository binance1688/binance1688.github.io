由 Foresight Ventures 旗下 Foresight X 主办的第二期Web3 加速器评选结束。因 Foresight X 在巴黎 EthCC 大会期间举办了主题为「Empowering a Trustless Future」OpenBuild Hacker House 活动，为完整收录活动参与团队， Foresight X 加速器将线上申请延期开放至了 7 月底。此次共收到了全球范围内 400 多个项目的申请。

最终有 4 个优质项目成功通过 Foresight X 筛选进入二期加速营，它们是：Quaere（AI+Web3）、Glacier Network（基础设施）、TonUP（LaunchpadPlatform）、T 2 T 2 （SocialFi)。本期入选项目将获得来自 Foresight Ventures 和 Foresight X 共计 200 万美元的扶持资金，以及包括经济模型、运营、宣发、技术、listing、法务、公司管理、人才建设等方面的相关支持。欢迎更多的有志于发展链上生态的建设者们与 Foresight X 共同推进 Web3 世界的繁荣发展。

以下介绍本期加速营的入选项目：

Quaere 

Quaere 是 AI 和占星理论结合的 Web3 心灵疗愈世界。 提供 7/24 的智能占星服务。Quaere 拥有先进的人工智能系统：从星盘测算，塔罗占卜，到合盘。聆听你的想法，根据具体需求情况提供生动贴心的分析。

Quaere 将使用 Web3 生态系统来奖励早期创世成员，为 AI 提供培训激励，并在链上验证 AI 占星师和应用内资产的所有权。

在 Forisight Ventures 的支持下，Quaere 正在打造一个独特的空间，占星爱好者可以在这里探索并与他们共建这个社区。

项目链接：网站｜推特｜预览

Glacier Network

Glacier 旨在为大规模去中心化应用打造一个可组合、模块化、可拓展的 Layer 2 数据网络，并推动去中心化应用（DApp）基于去中心化数据库（DDB）创造用例。结合多种 L2 Rollups 解决方案，Glacier 打造一个可编程、可组合的多链数据网络，让开发者可以自由地在 Arweave、Filecoin 和 BNB Greenfield 中使用 NoSQL GlacierDB 管理数据。

项目链接：官网｜推特｜电报｜Discord

TonUP

TonUP 是 TON 区块链上领先的 Launchpad 平台，专注于探索和推动具有高潜力的资产发展，为整个社区创造新的机遇。从创立之初就得到了 TONCoin.Fund 及 TON Foundation 的大力支持。TonUP 的目标旨在发掘更多 TON 上具备潜力的资产，并为生态的发展作出贡献。

项目链接：官网｜电报｜推特

T2T2

T2T2 是一个全方位的 Web3 社交服务平台，专注于创新的链上社交玩法。内置的 T 2 Influencer 板块做为一款全新的 SocialFi 应用，基于 Base Chain 构建，在 Friend.tech 和 Galxe 的基础上进行了大胆创新。它的目标是所有用户提供 Web3 领域的一站式社交服务。通过 T2T2 ，用户、项目方和 KOL 都可以更轻松地进行社交互动，以实现最大效益的价值变现。

项目链接： 官网｜推特｜Discord｜Google Play｜IOS

Foresight X 专注于培养行业建设者，建立全球研究者社区，并已经启动了包括加速器、生态基金和研究资助在内的一系列模块，旨在推动 Web3 项目的增长，高质量的 Web3 项目与顶级基金强强结合，同时在 Foresight 强大的资源网络优势下，整个行业的开发者社区、媒体资源和其他附加价值都将被充分调用。

同时，Foresight X 还与 Gitcoin、Celestia、MetaMask、Ton、Sei Network，常青藤区块链团体 Berkeley RDI、Oxford Blockchain Society、Cambridge Blockchain Labs 等，以及知名社区 Zuzalu、 85 2d ev 等建立合作关系，从启动资金、全球开发者社区和全球媒体资源等各方面助力区块链创业者，使他们能够在竞争激烈的市场中脱颖而出，实现创新并推动区块链技术的发展。 Foresight X 致力于支持优秀的项目创始人，并推动区块链世界的发展，尽管市场仍处于熊市，但此时也是区块链项目大规模创新和增长的最好时机。

如今，越来越多的 Web3 初创企业申请加入 Foresight X 加速器，Foresight X 也期待在未来帮助更多的 Web3 企业共同成长。敬请期待 Foresight X 加速器第三期的启动消息。
