作者：刘红林律师

为了进一步激活数字藏品用户交易量，不少平台上线了新的用户交易方式：“藏品求购”功能。曼昆团队和业内人士沟通时也了解到，很多没上线“藏品求购”功能的平台，近期也在跃跃欲试。这篇文章就来聊聊：NFT数字藏品平台上线藏品求购功能等用户交易模块，有风险吗？哪些能做，哪些红线不能碰？

0 1  金融交易所的37号、38号文

国务院与国务院办公厅分别于2011年、2012年，发布了《国务院关于清理整顿各类交易场所切实防范金融风险的决定》（国发〔2011〕38号，以下简称“38号文”）、《国务院办公厅关于清理整顿各类交易场所的实施意见》（国办发〔2012〕37号，以下简称“37号文”）。

37号文、38号文在金交所领域的地位，堪比币圈的94公告和924通知。 我国对非法金融活动向来是坚决遏制、严厉打击的监管政策，而与金融风险界限没有那么明晰的数字藏品行业，也一直是重点关注对象。

0 2  37号文、38号文中的禁止性规定

37号文、38号文的重点清理整顿对象，就是从事权益类交易、大宗中远期交易以及其他标准化合约交易的各类交易场所，包括名称中未使用“交易所”字样的交易场所。

37号文、38号文中，对设立交易所作了很多准入性门槛限制，例如交易品种、交易方式、投资者人数等，没有达到这些门槛，就必须遵守相应的“禁止性规定”。

两份文件中，对于经营NFT数字藏品交易的平台而言，需要格外注意的是如下两点：

  1. 任何交易场所均不得将任何权益拆分为均等份额公开发行，不得采取集中竞价、做市商等集中交易方式进行交易；不得将权益按照标准化交易单位持续挂牌交易，任何投资者买入后卖出或卖出后买入同一交易品种的时间间隔不得少于5个交易日；除法律、行政法规另有规定外，权益持有人累计不得超过200人。

  2. 任何单位一律不得以集中竞价、电子撮合、匿名交易、做市商等集中交易方式进行标准化合约交易。

此外，艺术品类、收藏品类数字藏品平台也需关注到，《艺术品经营管理办法》中明确规定，艺术品经营单位“未经批准，不得将艺术品权益拆分为均等份额公开发行，以集中竞价、做市商等集中交易方式进行交易” 。

0 3 数字藏品用户交易合规建议

数字藏品平台的寄售功能和求购功能，核心都是为用户提供数字藏品流转的通道。当一个数字藏品平台的交易功能既有“买”，又有“卖”时，平台方在“用户间互相交易藏品”这一流程中就有很多可操作的空间，可以通过设定种种规则来提高流通率，或从中赚取手续费。

但数藏平台无论采取什么玩法，都不能触碰 37、38号文 中列明的禁止红线 ，关于交易的设置，曼昆律师给到的合规建议如下 ：

  1. 求购功能应当使用点对点交易、真实身份交易、买卖双方直接交易的方式。

  2. 对于数字藏品的交易价格，由买卖双方参与确定，避免由平台方根据外部市场价格直接在平台上中设定藏品求购价格。同时建议平台方对交易藏品设置一定的最高交易价格。

  3. 对于数字藏品的成交机制，平台方应当赋予用户自主选择交易对象的权利，避免被认定为连续竞价、电子撮合、匿名交易等集中交易模式。

  4. 对于数字藏品的交易份数，建议平台方设定单个用户的最大同时求购份数。

  5. 严禁集中交易模式，例如集合竞价、电子撮合、匿名交易、做市商等。在数字藏品交易场景下，这些术语可以简单翻译为以下内容：集合竞价 是数字藏品平台在开盘的一刻钟内，把所有的买单和卖单汇集后进行撮合交易；连续竞价 是数字藏品平台每隔一分钟、一秒钟，把所有的买单和卖单撮合一次。电子撮合指众多买方和卖方同时通过交易系统进行撮合配对、点选成交或其他方式促成合约成立的交易方式。匿名交易 指参与交易人员完全不知道对手方的身份、年龄、信用状况等非交易信息。做市商机制 简单来说是指，数字藏品平台作为用户买卖数字藏品的中介，平台以A价向卖方用户买入数字藏品，以B价向买方用户卖出数字藏品。

  6. 不得以集中交易方式进行标准化合约交易。 交易对象的标准化合约通常指除价格、交货地点、交货时间等条款外，其他条款相对固定的合约。交易者将此类合约作为交易对象，订立合约时，并非全额付款，而只缴纳商品价值的一定比率作为保证金，即可买入或者卖出：合约订立后，允许交易者不实际履行，而可通过反向操作、对冲平仓方式，了结自己的权利义务。

  7. 严禁将任何权益拆分为均等份额公开发行。 例如，数藏平台把某作品的著作权切成1000份，并铸造发行1000份数字藏品，且持有这1000份藏品的人按持有比例享有其著作权和增值权益。

  8. 不 得将数字藏品按照标准化交易单位持续挂牌交易。 “标准化交易单位”是指将股权以外的其他权益设定最小交易单位，并以最小交易单位或其整数倍进行交易。“持续挂牌交易”是指在买入某数字藏品后5个交易日内挂牌卖出同一数字藏品，或在卖出后5个交易日内挂牌买入同一数字藏品。例如，数字藏品平台方对某发行量1万份的数字藏品，设定用户最少购买10份或10份的倍数；同时平台对藏品的交易时间不进行限定，可以即行买入卖出，易引发较大风险。因此，建议数字藏品平台设定“T+5”的交易时间限定 。

0 4 总结

总的来说，数字藏品平台方要坚持科技向善的理念，合理选择应用场景，规范应用区块链技术，坚决遏制数字藏品金融化证券化倾向，避开高压红线，从而做好和非法金融活动风险的隔离防范。
