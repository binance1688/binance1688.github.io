作者：Joel Monegro，Placeholder；编译：松雪，金色财经

随着人工智能模型的生产成本降低，AI代理的数量将呈指数增长。代理很快将在线上比人类多，创造、消费和交换比我们能想象的更多的信息。但如果我们获得了数字活动的百万倍增长，其中99%的增长来自机器，那么在不采用区块链基础设施和商业模型的情况下，我们将难以应对这一转变，这些模型需要赋予代理充分的潜力，同时允许我们识别、控制和审计他们的行为。

如今，像 OpenAI 这样的公司承担了生产模型的巨额成本，然后通过其专有接口和 API 出售访问权限，而且它们大多仅限于消费和创建内容。 但为了充分利用人工智能的潜力，我们需要大量能够相互交谈和交易的专业代理。 他们必须能够自由地在互联网上漫游，并且能够代表我们（或他们自己）拥有并花钱执行任务。 我们还需要方法来识别、控制和审计他们的行为。

问题在于我们无法强迫代理遵守我们的法律，这妨碍了有效的监管。他们不能使用依赖于辖区身份模型的传统金融系统，限制了他们的交易能力。他们消耗大量在线信息，把所有流量成本强加给服务提供商，但却没有通过订阅或点击广告产生相应的收入。要解决这些问题，我们需要一个原生数字的法律和金融系统，结合新的商业模型，充分利用这一新技术带来的机会。

该解决方案需要 (1) 具有新软件范式的主权数字基础设施，通过不可变的审计跟踪保证可信代码执行； (2) 一个独立的数字金融系统，对人和机器一视同仁； (3) 加密身份模型与去中心化通信和信誉协议相结合。 这只有使用区块链协议和智能合约应用程序才能实现。

区块链协议提供各种去中心化的数字服务，这些服务可以通过智能合约访问并用数字资产支付。 例如，以太坊和 Solana 等智能合约网络可以安全、可靠地执行由区块链交易的可审计跟踪支持的开源软件，而 Filecoin 和 Arweave 等网络则提供廉价且可扩展的链上数据存储服务。 随着在现有平台上构建新协议变得更加容易和便宜，通过去中心化网络提供的服务范围正在扩大。

我们可以使用这些平台以去中心化的方式训练、部署和操作代理，但更重要的是，通过智能合约消费它们的能力促进了代理之间的互动。 AI代理要想消费典型的Web2 REST API要困难得多，而读取智能合约并使用代币支付其服务则更容易，无需账户或信用卡。

支持钱包的代理可以使用任何智能合约服务或平台，从基础设施服务到 DeFi 协议再到社交网络，这开启了一系列新功能和商业模式。 代理可以根据需要为自己的资源付费——无论是计算资源还是数据资源——这是一个好主意。 它可以在去中心化交易所交易代币以获得不同的服务，或者利用 DeFi 协议通过机会性借贷或从其资产中赚取收益来优化其财务运营。 它可以在 DAO 中投票，甚至可以为其功能收取代币费用，并与其他专门代理进行信息交易以换取金钱。 其结果是形成了一个庞大而复杂的经济体，由专门的人工智能代理通过去中心化消息协议和链上交易信息相互通信，同时承担必要的成本。 这在传统金融体系中是不可能做到的。

考虑一下这个想法的后果：如果代理在链上行动——即使他们认为是链下的——我们最终会得到一个公开的、不可变的、经过加密签名的他们随着时间活动的记录。 区块链将确保大规模人工智能的安全部署，使我们能够执行诸如审计代理的互联网行为、区分机器制作和人造内容以及根据机器的链上活动为机器构建身份和声誉系统等事情。 它将帮助我们和他们识别和奖励好的参与者（用代币或声誉），惩罚不好的参与者（例如，通过削减），并判断哪个代理在特定任务上比另一个代理表现更好。 然后，代理将能够根据他们的链上历史记录来决定依赖谁，由于区块链数据的开源性质，他们可以轻松访问这些历史记录。

为了实现这一愿景，许多工作正整合在一起。 得益于新的共识机制和扩展解决方案，区块链基础设施正在迅速变得快速且廉价。 智能合约钱包和“钱包即服务”（WaaS）提供商将使代理能够进行交易； 与此同时，新兴的账户抽象技术可以允许人与代理之间的交互，我们可以授权代理从我们的钱包中进行支出。 我们可以使用代理的公钥作为标识符来构建注册表和信誉系统（例如，包括阻止列表或削减机制）。 我们甚至可以与 DAO 拥有的代理一起玩，尝试新的商业模式； 也许一些代理将生活在自己的第 2 层网络上，由分布式的运营商社区拥有和管理。

这些想法最初看起来有些牵强，但事后看来却非常明显。 当然，智能合约将调解代理人之间的商业活动，就像法律合约管辖人类的活动一样。 当然，代理商将使用由数字资产支持的互联网金融系统，而不是银行和信用卡。 当然，代理将通过去中心化协议使用加密身份来相互识别、通信和交易。 我没有看到其他解决办法：人工智能属于链上。
