原文作者：WILLIAM M. PEASTER

原文编译：深潮 TechFlow

关于将 NFT 从一个链转移到另一个链，实际上并没有一个正确的方法，迄今为止我们已经看到了各种不同的方法，它们都有各自的权衡考虑。

对我来说，首先想到的方法是等值桥接，就像将一个 NFT 从以太坊移动到协议层面等效于以太坊的不同链，例如 Optimism 二层扩容解决方案。

的确，经过今年早些时候的 Bedrock 升级，Optimism 几乎拥有了与以太坊完全等效的性能，因此，如果你将一个 NFT 从以太坊桥接到 Optimism，就好像将一个 NFT 从以太坊桥接到以太坊一样。

Quix 市场以前提供过这样的 NFT 桥接，但现在可能只是一个理论示例——上次我尝试时它就无法运作了。

诚然，在 NFT 生态系统中，我们尚未看到太多等值桥接。然而，更常见的做法是使用更加原始但也相当实用的销毁和铸造，进行整个收藏的迁移。

所谓销毁和铸造是指一个收藏持有者可以在一条链上销毁他们的 NFT，然后在新链上收到官方批准的重新铸造。通常将其称为桥接体验。一个例子是 y00ts，它曾两次使用这种方法从 Solana 迁移到 Polygon，然后从 Polygon 迁移到以太坊。

最近流行起来的另一种方法是由互操作性协议 LayerZero 的全链 NFTs 所推广的锁定和铸造。你可能已经在 Lil Pudgys Bridge 中看到过这种方法，它支持向 Arbitrum、BNB Chain 和 Polygon 的转移。

这种架构的工作方式是，LayerZero 提供了一个智能合约系统，可以将普通 NFT 转化为 ONFT（全链 NFT），然后你可以在 LayerZero 支持的链上部署 ONFT。

然后，为了实现跨链转移，ONFT 在源链上被锁定并在目标链上被铸造，要实现返程，ONFT 在新源链上被锁定，原始 NFT 在原始源链上被解锁。

不过，LayerZero 只是一个互操作性协议。一个明显正在同时支持 LayerZero 和其他消息协议的项目是 Holograph。

具体而言，Holograph 已经创建了一个被称为Holographic Assets的新原语，其中一个实例是 hNFTs，可以让你铸造 NFTs，它们在所有 EVM 环境中保持相同的合约地址和代币 ID，从而可以在不同的区块链之间完美转移。

Holograph 正在努力将 hNFTs 作为标准用于代币化现实世界资产，将资产层与区块链层分离。

然而，关于其他消息协议，还有一个显示出巨大潜力的是 Chainlink 的新跨链互操作性协议（CCIP）。当然，CCIP 系统旨在实现的不仅仅是支持跨链 NFTs，它还可以在一系列用例中表现出色，如跨链游戏、跨链 NFTfi 和通用 NFTs。DeFi 项目 Affine 正在专门利用该系统来支持其 Affine Pass NFTs。

因此，尽管一些努力，如 CCIP，希望提供一种使一切都能实现跨链兼容的方法，但另一方面，一些项目正在为自己特定的需求构建定制的跨链解决方案。

在这方面，我想到了链上 RPG Pirate Nation，因为它开发了一个自定义的镜像系统，可以促进L1 上交易/L2 上游玩：原始的创始人海盗 NFTs 保留在以太坊，而它们的镜像在 Arbitrum Nova 上支持无 Gas 费的游戏。

这一概念将我们重新引回到在一条链上拥有 NFTs，而无需跨链移动它们就可以在另一链上使用的情境。这种实用性得益于像 UMA 的 Optimistic Oracle 这样的服务，它能够考虑一链上的 NFTs，同时在另一链上执行相应操作。

很明显，就 NFTs 而言，不同区块链之间的界限正在逐渐模糊，各种跨链努力正逐渐崭露头角。在短期内，这一领域将面临各种挑战和障碍，但也将涌现新的可能性、新的创新和新的体验。
