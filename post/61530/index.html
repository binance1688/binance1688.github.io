作者：Kerman Kohli，Arcx money创始人；翻译：金色财经0xjs

在过去几年中，有一件事让我越来越感到困扰，那就是加密行业对“数据”的依赖日益增加。我之所以用引号，是因为其中大部分都是虚假的/不真实的。

为了演示正在发生的事情以及其运作方式，我想写一篇更长的文章围绕整个问题。当我开始写这篇文章时，我意识到整个过程是多么的产业化，以及投资者被它愚弄了多少。整个事情就像一个大笑话，展示了整个行业还有很长的路要走。

Layer 1 估值

我们的问题始于这些被过度定价、过于炒作的代币，投资者愿意为之支付数十亿美元。你只需要一份漂亮的白皮书，就能歪曲其中所有的单位经济学。我的研究始于 Dune，我在那里找到了一个计算多个空投的客户获取成本（CAC） 的仪表板。

https://dune.com/kenjisrealm/airdrop-research

这是一个不错的开始，尽管我想指出这些 CAC（从项目的角度）都是低估的数字，因为它们只是简单地计算：美元价值/ 领取空投地址。这个计算没有考虑到实际保留空投的百分比。鉴于通常只有10%-20%的地址保留了空投，可以安全地假设这些 CAC 数字比上面看到的要高5倍至10倍。

这里的第二个问题是，我们有一个暗示的层次结构，即空投的价值有多高：

  • Layer 1/Layer 2/热度很高的协议 = 几千美元的奖励

  • 中小型应用 = 几百美元的奖励

幸运的是，这两者并不是互斥的！如果你在正确的链上使用正确的应用，你将获得这两者的空投 —— 哇！去撸空投吧！

因此，理想情况下，你想要撸空投，首先专注于公链，然后在那之后尽可能多地撸其他空投。好吧，但问题是接下来会发生什么？

找到合适的空投

幸运的是，对于空投撸家，已经有一个完整的围绕撸空投而产生的行业。通常这些空投发现站点要求你执行一些非常明确定义的“行动”，你需要在链上提供这些行动的证明。无论是你的祖母还是你的机器人，都无所谓，只要确保交易在链上可见。

所有这些“探险”平台实际上都只是伪装成空投发现站点。通常情况下，如果它们能够吸引高质量的用户，这可能不是一个问题，然而当你使用这些站点时吸引的用户通常是非常唯利是图的，代表了这个行业整体上的短期欺诈。

让我们转向非常值得信赖的dApp Radar（https://dappradar.com/blog/top-10-crypto-airdrop-opportunities），看看此时可能正在进行的空投。

基于这个，我的计划最有可能是：

  • 使用 zkSync 作为我的基础链

  • 使用 Layer Zero 来桥接我的资金

  • 使用 Metamask 作为我的链钱包

这可能只是我的自然工作流程，无需额外的工作。但问题是，为了了解你在这些潜在空投中的排名，你需要做什么呢？嗯，令我惊讶的是，已经出现了一个完整的“空投模拟器”社区。其存在是为了帮助你了解你相对于其他撸空投者的位置。搜索“空投”就可以找到使用过去的空投标准来模拟项目如何分发空投的仪表板。这是我在zkSync上找到的一个例子（https://dune.com/teo/zksync-airdrop-ranking）。

令人着迷的是其中详细列出的层次。看看这张表中列出的所有列。一个任意的分数，最后的交易时间，交易次数，唯一合约，总交易的美元价值，唯一活跃天数/周/月，钱包年龄和区块时间。令人惊叹。

当你的“社区”已经为你完成了空投计算时，为什么还要费心去做呢？我截屏的这张表格有 765,000 行。

操纵系统

如果你对整个事情被详细列出感到惊讶，那么等你看到接下来的部分你会更惊讶：如果你了解所有用于标准的事物的排列组合，那么你就可以开始自动化并构建高效的系统。我花了一些时间进行研究，找到了这两个令人惊叹的工具。可能会尝试它们，只是为了报告这整个空投游戏有多么腐败。首先是nftcopilot.com，他们为你构建了一个时尚的仪表板，以自动化和设置你的撸毛工作。他们首页上有一个视频概述了整个过程。

令人惊叹的是他们在这方面的深度和细节。在产品中，你可以创建“组”，你可以自定义以下参数：

1、通过桥路由的交易数

2、要桥接的网络（以太坊、Polygon、BNB链、Arbitrum、Avalanche、Optimism、Metis、Aptos）

3、随机操作的配置，包括概率配置、休眠间隔和每个交易的最大随机交易数。

这是他们网站上的一些文字：

  • 没有女巫风险：系统通过完全随机化你的钱包所采取的行动来消除任何潜在的将其识别为女巫的可能性。每个钱包都有不同的时机、顺序和金额，创造出一种多样化和不可预测的策略，防止识别。

  • 自动费用跟踪：你不再需要在 Excel 和 Sheets 中跟踪你所有的交易并手动计算你的费用。我们将为每个钱包自动完成所有操作。你将拥有一个真正自动化的控制室来管理你的工作室。

  • 自动提现和合并：在达到目标后，资金可以自动合并到选择的链和代币，并提取到你的 CEX 钱包。不再花费数小时管理你的资金。

我对整个事情感到困扰的是他们的“使命”：

帮助赢家取胜。

我们相信，市场上的每个人都从半自动化中受益 —— 项目、个人和整个行业。

我们帮助项目提高参与度，筹集资金，并将代币分发给具有长期愿景的合适的人。

我们帮助个人节省宝贵的时间，获得知识，并在长远内分配更多资源。

我们帮助行业吸引新用户，将资源分配到能够产生价值并促进增长的人的手中。

所有这些只有在我们帮助赢家的时候才能得到保证。

为了找到他们，我们团队在接受 Copilot 之前对每个候选人进行面试，并过滤出不良行为者，以保持我们成员的最高标准，并确保这样的工具只提供给合适的人。

让我们明确一点，这远非对整个生态系统的价值增加，而是对虚假估值的证明。

如果你从更宏观的角度来看，实际上正在发生的事情是为了证明撸这些空投的成本要低于可能来自这种努力的潜在回报。我发现的另一个网站有助于更清晰地交易，列出了价格和可能的投资回报率。看看这个网站https://app.airdrop-hunter.site/。

现在你可以使用简化的数学来计算你想投入多少钱，以及你希望得到多少钱。为每个人解决一个真正的问题。在某些时候，对风险投资者来说，投资空投可能比投资真实项目更有利可图 。更快的流动性，更少的心理焦虑。

只要空投的成本低于潜在回报，撸空投就会盛行。

最终结果

那么，对于被过度炒作的项目，其上产生了组织化撸毛，结果会怎样呢？简单来说，这只是一个谁是更大女巫的竞争。如果链上的数字被夸大了，那么那些不知道如何对数据进行分析的普通参与者就会报告他们所看到的内容，并最终欺骗散户投资者，让他们相信他们正在购买的东西具有真正的吸引力。

答案

无需许可的身份。 除非我们实际检查谁在产生这种活动的指标，否则我们都在自欺欺人。
