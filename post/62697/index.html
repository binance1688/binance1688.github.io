面向普通用户：

1、用户自定义GPT

1）用户为他的特定任务创建定制版本的 ChatGPT ，且无需编写任何代码行。

2）用户可以将GPT 连接到 API 以执行任务，例如管理数据库、电子邮件、发短信等等。

3）案例：Code.org（策划编程课程）,Canva（生成设计图）,和Zapia（自动化流程）

2、本月推出GPT Store

1)让用户可共享ChatGPT的用法

2)本月晚些时候将推出GPT商店，收录最受欢迎的GPT

3)与杰出创建者分成计划

3、界面简洁化

1)GPT-4用户不用再点模型的各种下拉菜单了，以后只有一个窗口，模型可以自主调用代码解释器、联网工具、Dalle3等等

面向开发者：

1、GPT-4 API升级为 GPT-4 Turbo

1)从8K到128K上下文，提升16倍

2)开发者对模型有更多控制，引入JSON，可调用多个函数，引入可重现输出，中提供查看log prob功能

3)多模态API（文本，图像，语音）

4)开放GPT4微调（选定公司）

5)知识库更新到2023年4月

6)允许对16K版本的GPT-3.5进行细调

2、GPT4降价

1)GPT4 输入降低70%，输出降低50%

2)与GPT-4相比,Turbo整体价格降低超过2.75倍

3)降低GPT-3.5 Turbo 16K的价格，输入降低4倍，输出降低3倍

3、发布GPT Assistant API（和GPTs的区别是，开发者可以发布在自己的程序上）

1)可以帮助开发者构建在自己的应用程序中构建Agent

2)功能：Code Interpreter（代码解释器）、Retrieval（知识库）、Function calling（函数调用）。

3)改进的函散调用支持.确保JSON输出,无额外延迟可同时调用多个

4)Code interpreter 单次0.003美元，Retrieval 是0.20 美元/GB/助理/天。

4、开源语音和图像模块

1)开源WhisperV3，以及即将开放API

2)开源Consistency Decoder，是Stable Diffusion VAE的替代品

面向所有人：

1、微软合作

1)致力于通过GitHub Copilot等产品赋能开发者

2)高度重视安全性

3)微软将继续支持基础设增建设

2、GPT过往数据

1)周活跃用户1亿

2)200万开发者根据API进行开发
