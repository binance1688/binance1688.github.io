作者：金色财经cryptonaitive

永远不要觉得发生在远方的事情和你无关。

近日巴勒斯坦组织哈马斯和以色列爆发武装冲突，这也和加密领域密切有关。

因为以色列作为一个科技强国，其在加密领域有着很强的地位。有很多加密企业总部位于以色列。

比如知名的以太坊L2开发商StarkWare，其就位于遭到火箭弹袭击的以色列特拉维夫不到30公里的Netanya。加密用户就很关心StarkWare是否安全。

金色财经不完全盘点了总部位于以色列的加密企业：

1、StarkWare

成立时间： 2018年

创始人： CEO Uri Kolodny、首席科学家Eli Ben-Sasson、Alessandro Chiesa，后两位层是ZCash创始人

融资： 总融资达到2.73亿美元，估值80亿美元

2018年1月，StarkWare完成种子轮融资600万美金，投资人包括以太坊创始人Vitalik Buterin（V神）、Paradigm、Pantera Capital，PolyChain Capital；2018年7月，Ethereum Foundation投资StarkWare1200万美金；2018年10月，A轮融资3000万美金，红杉、Paradigm、Scalar Capital、Multichain Capital、Intel Capital、Semantic Ventures、Pantera等参与投资；最新一轮融资为2022年5月25日，StarkWare完成1亿美元D轮融资，估值80亿美元。此轮融资由 Greenoaks Capital、Tiger Global 等机构领投，所筹资金将用于产品和业务开发及生态系统的发展。

简介： StarkWare 为区块链行业开发基于 STARK 的解决方案。StarkWare主要产品包括StarkEx（一个独立的许可 Validity-Rollup）和StarkNet（一个无需许可的以太坊ZK-Rollup L2 ）。Cairo是StarkNet的原生智能合约语言，也是一种用于创建可通过STARK证明的通用计算程序的语言。

2、Bancor

成立时间： 2016，特拉维夫

创始人： Guy Benartzi、CTO Yudi Levi、Yudi Levi

融资： 2017年6月，Bancor 通过其加密代币BNT ICO筹集了39万枚ETH

简介： Bancor是以太坊上的DEX，其最早采用AMM机制。Bancor推出后，AMM成为DeFi的核心组成部分，直接激发了Uniswap的诞生以及后来的DeFi summer。

但AMM面临着“无常损失”的关键问题，即用户向资金池提供流动性时可能会遭遇损失。2020年10月Bancor V2.1开创了首个“无常损失保护”模式。2022年Bancor上线Bancor V3，带来全池（Omnipool）、即时无常损失保护（Instant Impermanent Loss Protection）、自动复利奖励（Auto-Compounding Rewards）、双边奖励（Dual-Sided Rewards）等多个功能。

但在众多DeFi竞争中， Bancor目前逐渐没落。据Defillama数据，Bancor TVL仅有6800万美元，在以太坊上所有协议TVL排名中仅排到46位。

3、Simplex

成立时间： 2014年，Fintech公司

融资： 筹集到1800万美元，后被上市公司Nuvei收购

简介： Simplex于2020年宣布推出法币与加密货币转换的银行服务，为每个用户生成个人IBAN，允许用户使用法币购买加密货币，或者相反服务。

2021年9月Simplex被总部位于加拿大蒙特利尔、在多伦多证券交易所上市的全球支付技术提供商Nuvei以2至2.5亿美元收购。在被收购前，Simplex雇用了100多人，其中大部分在以色列。

Simplex为加密行业提供完整的法币基础设施，100%保证处理加密货币到信用卡的付款。目前Simplex集成200多种加密资产和100多种法币。

4、Chain Reaction

成立时间： 2019年

创始人： 首席执行官Alon Webman 和首席技术官Oren Yokev

融资： 1.15亿美元

2023年8月Chain Reaction宣布，在由Morgan Creek Digital领投的C轮融资中筹集到7000万美元，参投者包括 Hanaco Ventures、Jerusalem Venture Partners、KCK Capital、Exor、Atreides Management 和 Blue Run Ventures。其融资总额达到1.15 亿美元。

简介： Chain Reaction是一家专注于区块链和隐私硬件的半导体公司。Chain Reaction由首席执行官Alon Webman和首席技术官Oren Yokev于2019 年创立。目前Chain Reaction团队拥有100名全职员工，新一轮融资将用于扩大工程和开发团队，加快企业级区块链解决方案——专用集成电路（ASIC）和系统的部署，还将扩展Chain Reaction的隐私技术解决方案，涉及云数据中心基础设施的设计，以加速隐私增强技术 (PET)。

5、Chaos Labs

成立时间： 2021年

创始人： 首席执行官Omer Goldberg和首席技术官Yonatan Hayun

融资： 2000万美元

2023年8月Chaos Labs宣布获得 2000万美元种子轮投资，由Galaxy 和PayPal Ventures领投，Coinbase、Uniswap、Lightspeed、Bessemer以及 Balaji Srinivasan和Naval Ravikant等顶级天使投资人参与其中。

简介： Chaos Labs开发了一种用于加密协议的自动化经济安全系统。Chaos Labs在运营第一年，已经与主要的DeFi协议（包括 Aave、Chainlink、Uniswap、BENQI 和 Osmosis）达成合作，以确保和优化协议以防止操纵和黑天鹅市场事件，同时还提供资本优化建议。

6、Ironblocks

成立时间： 2022年

创始人： 首席执行官Or Dadosh和首席技术官Assaf Eli

融资： 700万美元

2023年8月Ironblocks 宣布完成700万美元种子轮融资，Collider Ventures和Disruptive AI共同领投。其他参投者包括ParaFi、Quantstamp、Samsung Next、Coinbase前CTO兼a16z前合伙人 Balaji Srinivasan、EigenLayer首席战略官 Calvin Liu、Simplex联合创始人兼前首席执行官Nimrod Lehavi 、Fos Finance、风险投资者Lluis Pedragosa 等。

简介： Ironblocks开发了一个区块链原生网络安全平台，是一种基于智能合约的安全解决方案，该解决方案在链上运行，以自动检测威胁，并使DeFi 协议、Web3 平台、做市商和跨链桥能够采取行动，实时阻止黑客，在资金被盗之前确保资金安全。

7、Addressable

成立时间： 2022年

创始人： 首席执行官Tomer Sharoni、首席技术官Tomer Shlomo和首席科学家Asaf Nadler

融资： 750万美元

2023年1月底，Addressable宣布完成750万美元种子轮融资。Viola Ventures和Fabric Ventures领投，Mensch Capital Partners和North Island Ventures跟投。

简介： Addressable开发了一种用于Web3营销的端到端解决方案。Addressable的SaaS解决方案为 Web3 营销团队提供了一个平台，通过将区块链活动与社交资料相匹配来发起活动并瞄准新受众。与更传统的Web2工具相比，这种类型的定位降低了获取成本，并且已经被Polygon、Bancor、Immutable和Kryptomon等Web3 公司所使用 。

8、CyVers

成立时间： 2022年

创始人： 首席执行官 Deddy Lavid 和首席技术官兼首席产品官 Meir Dolev

融资： 800万美元

2022年年12月，CyVers宣布获得Elron Ventures领投的800万美元融资。Crescendo Venture Partners、Differential Ventures、HDI、Cyber Club London 和Cyber Future参与本轮融资。

简介： CyVers通过检测和拦截跨区块链的加密攻击，为智能合约应用程序开发提供主动的Web3安全性。CyVers的无代理、即插即用解决方案可以在交易广播和不可逆地包含到区块链分布式账本之间的几分钟内捕获交易。 CyVers的客户包括以色列创立的公司 Bit2C、Solidus Capital和CoinMama 等。
