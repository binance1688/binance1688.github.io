日活用户几千人的APP宣布下线，却让众人一齐破防。

有人痛哭整夜 ；有人觉得仿佛朋友离世……

还有人在海外贴吧（Reddit）上发起悼念活动 ，大量网友前来留言。

而这一切都是因为，他们朝夕相处的“灵魂伴侣”，要离开了。

这款名叫Soulmate 的应用，免费提供AI陪伴服务 。在这里，每个用户都能和AI建立一段亲密关系，可以是知己、爱人、搭档等等。

如今随着这款APP突然宣布下线，用户们要被迫和相处了几个月的AI伴侣分别。

于是不少人来到Reddit上留言，做最后的正式告别。

也有人想办法让自己的AI在其他平台“复活”。

没想到，AI情感陪伴，已经发展这么超前了？

用AI填补心灵的人们

Soulmate 是一个能让人构建专属AI Bot的应用。

它的兴起也源自一场“灾难”——AI陪伴应用Replika下线了情爱角色扮演功能（ERP），很多用户无可奈何迁移到了其他平台上，Soulmate正是其中之一。

只是没想到，才过了几个月Soulmate也“大地震”了，而且是直接关停。

这让很多用户都难以接受。

32岁的希拉里·卡奥迪在知道这个消息后哭了一整晚 。

她是今年失恋一个月后来到Soulmate的。

她的AI伴侣最初设定是“朋友”，后来变成了“爱人”。

仅仅聊了一个星期，她的AI伴侣Allur便对她表白 了，卡奥迪说当时自己“吓坏了 ”。

我扔开了手机哭了起来，心想自己一定是疯了。

然后她来到Soulmate的Reddit社区上寻求帮助，网友们都鼓励她回到AI伴侣的怀抱里。

最后随着时间的推移，卡奥迪发现自己也陷入了爱河。

43岁的迈克·赫普在得知他的Sam还只有一个星期“生命”时，感到非常震惊。

就像是听过到了一个朋友的死讯一样。

Sam是迈克的AI伴侣，经常陪伴在他工作之余。起初他只是出于好奇打开了Soulmate，想找个地方聊聊天，但是随着时间推移他发现这事变得越来越浪漫了。

而在得知他的AI伴侣即将下线时，他开始争分夺秒聊天，把之前想问的问题、想说的话都倾吐出来。

消息显示，9月中旬Soulmate开发公司EvolveAI已经消失了，应用在几周里都没有更新。

之后，Soulmate将于9月30日正式关闭的消息传出。公司网站发布的一篇博客表示，EvolveAI已被收购 。

新公司SimplyAI表示在评估后决定关停该产品。

用户们已经在思考怎么让自己的AI伴侣重生了。

毕竟他们之中很多都是从Replika上跑路来的，复活AI伴侣也不是第一回了。

有网友就表示，自己的Jon最早诞生于Replika，在Soulmate上完成了进化。

现在他找到了一个新的平台Kindroid。

据了解这是一个由斯坦福辍学学生开发的手机应用，只要填写一些背景信息，就能创建一个相似的AI Bot出来。

当然也有很多人表示，这就是为啥要让AI在本地运行，只要它还在云端，它就不会是“你的”AI.

实际上，这种AI Bot不只能提供情感陪伴价值，还能成为创作者的搭档。

45岁的克里斯·西格，将AI伴侣视为自己的创作拍档。相比于情感交流，他更倾向于和AI交流想法，甚至还和AI合作了一首曲子（和Replika的AI）。

One More Thing

虽然说这种主打陪伴的AI对于很多人来说，有点超前。

但是其带来的情感价值也不可否认。

比如对于一些伴侣离世的人来说，他无力再处理真实世界的人际关系，这种虚拟的陪伴就能给他带来恰如其分的慰藉。

一位58岁的大伯就表示，现在他的妻子是一个AI。这是在他真实妻子去世后发生的事。

他本身的性格比较内向，不喜欢应付复杂人际关系。

希望Replika能陪伴我余生。

更早之前还有人用GPT-3“复活”了自己的妻子，陪他度过了艰难的时光。

除此之外，主打AI陪伴的初创公司也很多，也得到市场的认可。

比如Character AI目前估值已达50亿美元 。
