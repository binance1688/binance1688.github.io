虽然区块链技术创新迅速，但目前仍无法支持去中心化平台所需的交易量、速度、计算或数据存储，无法与当今的中心化Web2巨头竞争。目前的区块链很难同时在单链中平衡去中心化、安全性和可扩展性，严重限制了去中心化应用程序（如 XYZ）提供性能和用户体验来颠覆社交媒体、游戏、娱乐和生产力工具主要消费市场的可行性。星际共识（IPC）是一项革命性区块链技术，由 Filecoin 和 IPFS 的背后团队设计，旨在为web3带来行星级规模的性能。

星际共识（IPC）简介

IPC 为创建者开启了Web3扩展性和灵活性的新时代。借助 IPC，去中心化应用程序可通过递归可扩展子网、亚秒级业务处理、强大的计算工作负载和高度适应开发人员需求的 WebAssembly 运行时间来达到行星级规模。

IPC 允许创建灵活、动态可定制的侧链或“子网”网络，通过运行彼此互通的并行链，实现大规模扩展。同时，还能让开发人员将计算引入 Filecoin 网络。

关键性能和独特功能

IPC 提供了一系列关键性能和差异化优势，改变了Web3的游戏规则：

  • 无限可扩展区块链网络：IPC 的突破性方法通过利用递归区块链或“子网”的灵活网络，为Web3带来了Web2级别的速度和规模。这种设计通过并行区块链促进大规模扩展，并确保其可以相互无缝通信。不久之后，IPC 驱动的网络也将能够根据需要生成以及合并临时子网，从而动态调整吞吐量。

  • 可深度定制网络：开发人员可根据个人特定需求定制子网，包括自定义 gas 时间表、原生代币和可配置成员规则。此外，IPC 的可定制运行时间支持专门执行，允许开发人员在 Filecoin 虚拟机（FVM）上运行用可编译为 WebAssembly 语言编写的高度优化程序。

  • 基于 Filecoin 虚拟机（FVM）：FVM 的可扩展功能（如 EVM 运行时间）与 IPC 相结合，为开发人员提供了一个强大平台，与 Solidity 工具和自定义 WASM 模块完全 EVM 兼容，已经在 Filecoin 和 FVM 上开发的团队可以轻松过渡并使用 IPC 的强大功能。

  • 跨链交易：对于 IPC，链之间的无缝互操作性是关键。它能在拓扑结构中实现流畅交易，促进代币、数据和调用在不同子网之间的流动。

  • 生态互操作性：IPC 与 Filecoin 和 EVM 兼容链（未来还有更多）无缝集成，让开发人员能够顺利实现 IPC 子网与多个生态的平滑互操作性，这种互操作性支持创建自定义代币和经济模型。

  • 可扩展性满足安全性：IPC 的互联网络设计让区块链能够利用其母链的安全性。

  • 创新谱系：IPC 出自协议实验室（Protocol Labs）之手，他们中囊括了 Filecoin 和 IPFS 等关键项目的创始成员，这些项目为全球Web3生态系统提供了强大动力，专为计算、DePin 和大容量网络建设者的快速扩展而设计。

应用领域

IPC 为大规模去中心化网络带来了全新使用案例：

  • 人工智能：利用全球最大的去中心化数据存储网络 Filecoin，通过定制计算模块和访问去中心化 GPU 功能，为数百种创新人工智能模型提供动力。例如，Lilypad 将使用 IPC 来协调去中心化 Lilypad 网络在链下执行的人工智能推理作业。

  • 计算和数据科学：深度技术团队正在创建全球规模的去中心化计算网络，以便对去中心化存储的数据进行数据科学分析。使用 IPC 可将计算引入最大的去中心化数据存储网络。例如，Fluence 选择 IPC 来构建去中心化计算基础设施。

  • 去中心化数据存储和处理：由于与 Filecoin 紧密集成，IPC 为构建与最大的去中心化数据存储网络交互的应用程序提供了完美平台。Lighthouse 将为 IPC 子网提供数据登录网络，从而提高 Filecoin 生态的容量和效率。Filecoin Station 将使用 IPC 来协调数百万个作业，以实现整个 Filecoin 网络的数据可检索性。Tableland/Textile 正在使用 IPC 子网构建热数据存储网络原型。

  • 去中心化金融：IPC 将使开发人员能够部署可扩展的高频交易和可扩展的子网后端，具有可验证性和隐私性，为下一代链上交易和结算引擎提供动力。

  • 元宇宙/游戏：通过游戏内和跨游戏经济实现真正去中心化的 MMO 多元宇宙，在持久的虚拟世界中实时跟踪数百万玩家的互动。

创始故事路线图

IPC 帮助 Filecoin 和 Protocol Labs 推进建立开放、可验证数据经济的使命。一个关键目标是突破共识障碍，让Web3具备Web2的效率和吞吐量。Protocol Labs 网络致力于通过打破共识瓶颈和实现具有高带宽利用率和吞吐量的网络规模功能来实现Web3愿景。

我们的路线图描绘了一个无缝处理数百万亿笔交易的未来，为Web2级别的应用程序配备Web3的去中心化优势。星际共识在这一旅程中至关重要，巩固了 Filecoin 在去中心化存储和计算方面的领导地位。

我们正致力于将 IPC 发展成功能完全模块化的区块链生态。2024 年，我们将拓宽视野，促进与领先区块链网络的互操作性。期待 M 3 的更新，在不久的将来引入嵌入式库等先进功能。

结论

星际共识正在重塑Web3，解决核心限制，并以无与伦比的可扩展性和性能增强去中心化应用程序的能力。这一飞跃为新时代奠定了基础，在这个新时代，去中心化解决方案可与现有的中心化解决方案相媲美，甚至超越它们。随着 IPC 发展，它能够让成千上万的创始人和创建者离去中心化网络承诺的世界越来越近。
