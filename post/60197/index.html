原文作者: Jonas@Foresight Ventures

故事缘起于上轮熊市底部时候，曾有友商成立了归零基金，将资金分散买入 50 个小市值的山寨币，预期以归零的心态实现以小搏大。最后押中了 AAVE、LINK 等几百上千倍涨幅的小概率百倍币，在牛市里整体取得几十倍的丰厚回报。此次成功的投资经历令人印象深刻，不过也提醒我们，抓到百倍币的运气成分较大，更适宜通过渔网法用概率捕获。

鉴于美联储加息周期即将结束，大盘筑底时机接近成熟，我们计划复刻归零基金的成功经验，为此精选了近 50 个有潜力的关注项目。目标是抓住能活过一轮熊市，可以涨几十倍到上百倍的山寨币。

主要筛选标准（满足其中多项）：

  • 技术面：高点跌幅曾超过 90% ，市值排名 100-1000 名左右。

  • 基本面：GITHUB 还在更新，社群还在运营。

  • 筹码面：充分换手，筹码释放完成，存在主要做市商。

  • 叙事性：有新概念，熊市期间新发币项目优先。

  • 流动性：Binance、Coinbase、OKX 等合规法币交易所。

一、Defi

去中心化衍生品或是未来 Defi 的主战场，伴随牛市投机需求的上涨和二层扩容方案的完善，将为 Defi 生态带来更多可组合性玩法。

1.VELO：OP 链上的 DEX 龙头，采用 VE33 模式流动性激励

2.GRAIL：ARB 链上的 DEX 龙头，最大打新平台

3.JOE：AVAX 和 ARB 链上龙头 DEX，做基于价格区间的贿选

4.ORCA：SOL 链上头部 DEX，SOL 链在欧美地区仍有较强共识

5.MCB：ARB 链上的合约平台聚合器 MUX

6.GNS：合成资产衍生品 DEX，包括加密、外汇、股票等类别

7.LYRA：OP 和 ARB 链上的欧式期权平台

8.RDNT：ARB 上的跨链借贷龙头，经济模型具有旁氏效果

9.GMD：GLP 的 Delta neutral 挖矿机枪池

10.PENDLE：利率衍生品赛道处于早期阶段，解放未来现金流

11.SSV：去中心化质押基础设施，获得 ETH 基金会认可

12.SD：多链流动性质押协议

13.UNIBOT：电报交易机器人龙头，用户快速增长

二、链游与元宇宙

链游打破传统游戏的商业逻辑，把收益和价值还给玩家。目前发布的游戏创新性普遍不高，可玩性一般。参考Web2游戏产业的发展规律，平台类项目更有未来。

14.MAGIC：ARB 链的链游平台

15.AGLD：开发 LOOT 链，做全链游戏

16.YGG：去中心化游戏公会，转型游戏投资基金

17.MC：从游戏公会发展到有孵化器、风投基金和交易市场

18.RLB：菠菜平台龙头，收入高且有回购机制

19.HALO：提供元宇宙中的可视化 DID 协议，Lifeform 团队孵化的项目

三、NFT

NFT 收藏品的效用在于明确了数字文化资产的属性以及所有权归属，促进数字资产的交易。未来随着资产上链的需求扩张，NFT 收藏品会持续增长，将形成创作、交易、金融服务等新的生态产业。

20.BLUR：头部 NFT 交易聚合平台，有上大所预期

21.SUDO：采用 AMM 协议的 NFT 交易平台

22.BEND：NFT 抵押借贷龙头，采取点对池方式

23.JPEG：NFT 抵押借贷平台，以 NFT 为抵押物铸造稳定币

24.OPUL：音乐 NFT 版税投资平台，融合 Defi 为音乐人提供抵押贷款

25.ORDI：BRC 20 龙头，具有先发优势

四、社交

按照互联网的发展规律，拥有庞大原生用户基础后，Web3社交图谱有望开始爆发。但是现有去中心化社交赛道大部分项目都是对传统互联网的模仿，真正现象的新范式尚未出现。

26.GAL：Web3凭证数据网络，旨在构建 DID 系统，以帮助项目方更好构建产品和社区

27.RLY：可为在线的创作者提供经济支持，如发粉丝币

五、人工智能

AI+ 区块链的结合仍在初始摸索中，或将带来新的商业模式和机会。

28.AGIX：AI 服务平台，使用以太坊创建支付和注册系统，新建 IDO 发射平台

29.FET：与 AI 深度结合的公链，包括可编写 AI 模型的智能合约

30.ALI：币安投资的 AI 生成化身项目，Mark Cuban 投资，有上大所预期

31.RNDR：去中心化渲染的 GPU 算力网络，可用于元宇宙和 AI

32.OCEAN：交易大数据协议，以及基于该协议制作的公共数据市场

六、Meme

上轮牛市人们见证了 DOGE、SHIB 等 Meme 币的上万倍涨幅的暴富故事，情绪叙事也是有价值的。

33.XEN：POP 挖矿机制，已建有生态

34.BITCOIN：哈利波特比特币是 MEME 龙头

35.TSUKA：藏传佛教相关的 MEME，与 SHIB 创始人有关，计划建设生态项目

七、支付

随着更多的机构采用加密支付，加密支付提供商将获得长远的需求。

36.ACH：去中心化的开源支付协议，为企业提供加密货币和法币混合支付解决方案，是支付赛道发展最好的项目之一

八、基础设施

去中心化的基础设施有助于确保主流大众，而不仅仅是区块链创新者，加入Web3世界。

37.DAO：Daomaker 是Web3社区孵化器，可帮助项目初始阶段进行融资

38.RAD：去中心化 Github 代码协作基础设施，Web3开发者 DAO

39.POND：致力帮助底层区块链实现扩容的第 0 层协议，借鉴 CDN 加速技术

40.CTSI：基于二层 Linux 的基础架构平台,弥合主流软件和区块链差距

41.STG：第一个基于 LayerZero 协议搭建的 Dapp，旨在帮助用户实现跨链资产转移

42.AXL：通用型跨链基础协议，基于 Cosmos 技术，以自身公链作为中继链

43.SIS：类似V2版本的 Cbridge ，DWF 对其做市

九、公链

搭建公链就像修建一条铁路，所需要的前期投入巨大，但后期收取来自使用者的费用带来可持续的收入，并且生态越繁荣，创新力量越强大。这解释了公链作为Web3核心价值载体高估值的原因。

44.CANTO：Cosmos 上的 Defi 公链，与 Polygon 合作 ZK 技术

45.KAS：挖矿算力前三的 POW 公链，采用 DAG 技术

46.CFX：国内唯一合规的公链，与国内主流 APP 合作，香港概念龙头

Web3喜欢炒新不炒旧，还有很多不错的新项目尚未发币，将于未来 1-2 年发币，因此值得我们长期关注。未来的牛市将是大家熊市期间所做工作得到回报的时候，特别属于那些坚持不懈、不断建设的人。衷心希望我们都在新一轮牛市获得一个美好的收益。
